### 降级策略

```typescript
try {
  // 尝试调用AI API
  const result = await aiService.generateSolution(request)
  return result.solutions
} catch (error) {
  // API失败，降级到本地模拟
  console.error('AI调用失败:', error)
  return getFallbackSolution(request)
}
```

**优势**:
- ✅ 即使API不可用，功能仍然可用
- ✅ 用户体验不受影响
- ✅ 便于开发和测试

## 🔒 安全特性

### API密钥保护

✅ **页面配置**:
- 存储在浏览器LocalStorage
- 不会上传到服务器
- 仅当前浏览器可用
- 用户完全控制

✅ **环境变量**:
- 存储在服务器端
- 不暴露给前端代码
- Git忽略（.env.local）
- 团队共享方便

### 最佳实践

1. **不要硬编码密钥**
   ```typescript
   // ❌ 错误做法
   const apiKey = 'sk-xxxxxxxx'
   
   // ✅ 正确做法
   const apiKey = import.meta.env.VITE_AI_API_KEY
   ```

2. **Git忽略敏感文件**
   ```gitignore
   # .gitignore
   .env.local
   .env.*.local
   src/api/ai/ai.config.ts  # 如果使用手动配置
   ```

3. **设置使用限额**
   - 在AI平台设置月度预算
   - 避免意外高额费用

4. **定期轮换密钥**
   - 每3-6个月更换一次
   - 发现泄露立即撤销
