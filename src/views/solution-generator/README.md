# 🎯 企业网站智能方案生成器

## 📖 项目简介

这是一个基于597个GSAP动画组件知识库的AI驱动智能方案生成器。通过收集用户的详细需求，自动生成最适合的企业网站组件组合方案。

## ✨ 核心功能

### 1. **五步智能引导**
- ✅ **步骤1**: 基本信息收集（网站类型、行业、业务描述）
- ✅ **步骤2**: 视觉风格选择（6种风格：科技/自然/奢华/简约/创意/复古）
- ✅ **步骤3**: 功能模块配置（8大模块可选）
- ✅ **步骤4**: 性能与预算设定（设备、预算、周期）
- ✅ **步骤5**: AI生成最佳方案（3套备选方案）

### 2. **智能推荐引擎**
基于以下维度进行智能匹配：
- 网站类型和行业特征
- 视觉风格偏好
- 选定的功能模块
- 性能要求和预算约束
- 开发时间周期

### 3. **多方案对比**
每套方案包含：
- 📊 性能评级（⭐⭐⭐⭐⭐）
- ⏱️ 预估工时
- 💡 推荐理由
- 📦 详细的模块组件清单
- 🎨 配色方案建议

### 4. **方案导出**
- 📋 复制方案代码
- 👁️ 查看详细实现
- 📥 导出JSON格式完整方案

## 📦 功能模块说明

### 1. 首页英雄区 (Hero)
首屏视觉焦点，用于品牌展示和核心价值传达

**推荐组件**:

**推荐组件**:
- 时间轴类: Timeline, CardTimeBook, CardTimeQuantumLeap
- 卡片展示: ScrollModernCards, CardParticle
- 数据统计: NumberCounter, AnimatedStats

### 3. 产品/服务 (Products)
产品展示、特性对比、服务介绍

**推荐组件**:
- 卡片列表: AccordionCards, HoverEffectCards
- 3D展示: Card3DRotate, ProductShowcase3D
- 对比表格: ComparisonTable

### 4. 技术优势 (Technology)

**推荐组件**:
- 架构图: TechArchitecture, NetworkGraph

### 5. 成功案例 (Cases)

**推荐组件**:
- 案例卡片: CaseStudyCards, TestimonialSlider

### 6. 新闻动态 (News)
最新资讯、博客文章、活动信息

**推荐组件**:
- 新闻列表: NewsList, BlogCards
- 时间线: NewsTimeline

### 7. 合作伙伴 (Partners)

**推荐组件**:
- Logo轮播: PartnerLogos, LogoMarquee

### 8. 联系我们 (Contact)
联系表单、地图位置、社交媒体

**推荐组件**:
- 联系表单: ContactForm, InquiryForm
- 地图集成: MapIntegration
- 社交链接: SocialLinks

## 🔧 技术栈

- **前端框架**: Vue 3 + TypeScript
- **动画引擎**: GSAP 3 + ScrollTrigger
- **样式方案**: SCSS
- **状态管理**: Vue Composition API
- **路由**: Vue Router

## 📝 开发指南

### 扩展新的视觉风格

在 `index.vue` 的 `visualStyles` 数组中添加：

```typescript
{
  value: 'new-style',
  name: '新风格名称',
  icon: '🆕',
  desc: '风格描述',
  tags: ['标签1', '标签2']
}
```

### 添加新的功能模块

在 `availableModules` 数组中添加：

```typescript
{
  key: 'new-module',
  name: '新模块名称',
  icon: '🔧',
  desc: '模块描述'
}
```

### 自定义组件推荐逻辑

修改 `getComponentsForModule()` 函数，根据实际需求返回对应的组件列表。

## 🎯 AI推荐算法

### 决策树流程

```
用户需求输入
    ↓
分析网站类型和行业
    ↓
匹配视觉风格组件库
    ↓
筛选适合的动画效果
    ↓
评估性能要求和兼容性
    ↓
优化组件组合方案
    ↓
生成3套备选方案
```

### 评分机制

每个方案根据以下维度评分：
- 视觉效果 (30%)
- 性能表现 (25%)
- 开发效率 (20%)
- 兼容性 (15%)
- 可维护性 (10%)

## 📊 性能分级

| 级别 | 星级 | 组件数量 | 适用场景 |
|------|------|----------|----------|
| 初级 | ⭐ | ~100个 | 所有项目，移动端优先 |
| 中级 | ⭐⭐ | ~250个 | 大多数企业网站 |
| 高级 | ⭐⭐⭐ | ~180个 | 高性能桌面端项目 |
| 专家级 | ⭐⭐⭐⭐ | ~67个 | 顶级视觉体验项目 |

## 💡 最佳实践

### 1. 根据预算选择方案
- **紧张预算**: 选择方案C，使用基础组件
- **适中预算**: 选择方案B，平衡效果和性能
- **充裕预算**: 选择方案A，追求极致视觉

### 2. 考虑目标用户设备
- **移动端为主**: 避免复杂Canvas和WebGL效果
- **桌面端为主**: 可以使用更多高级动画
- **两者兼顾**: 做好响应式和降级处理

### 3. 注意加载性能
- 图片资源使用懒加载
- 动画组件按需导入
- 合理设置ScrollTrigger触发点

### 4. 保持品牌一致性
- 统一配色方案
- 一致的动画节奏
- 协调的视觉风格

## 🐛 常见问题

### Q: 如何修改默认的配置？
A: 在 `formData` 对象中修改默认值即可。

### Q: 可以保存用户的方案吗？
A: 目前支持导出JSON格式，可以后续接入后端存储。

### Q: 如何集成到现有项目？
A: 直接访问 `/solution-generator` 路由即可使用。

### Q: 支持自定义组件库吗？
A: 可以修改 `getComponentsForModule()` 函数来适配自己的组件库。
