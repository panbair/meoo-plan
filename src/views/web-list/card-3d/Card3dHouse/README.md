# Card3dHouse - 时空裂隙·纸牌屋坍缩组件

## 效果特点

### 三维纸牌屋初始
12张卡片以精密计算的3D姿态搭建起一座微型纸牌城堡——中央四张卡片倚靠成金字塔尖，外围八张卡片以不同倾斜角度环绕支撑。每张卡片都拥有真实的`rotateX/Y/Z`与`translateZ`，背面因`backface-visibility: hidden`而不穿帮。

### 滚动驱动的连锁坍缩
向下滚动时，纸牌屋从结构最脆弱的顶部开始瓦解。卡片不是同时倒塌，而是从上至下、从内向外依次失去平衡——先微微晃动，再沿重力方向翻转跌落，在空中划出弧线，最终轻柔滑入平面网格中。

### 动态阴影与材质变化
初始状态下，每张卡片模拟温暖室内光源的单一阴影；崩塌过程中，阴影逐渐缩短、变淡。卡片自身的`filter`也从初始的微暖`sepia(0.25)`过渡到清晰无色。

### 完美双向重建
向上滚动时，平面网格中的卡片逆序飞回，重新搭建成纸牌屋，所有晃动与翻转完美倒放。

## 技术实现

### CSS 3D 纸牌屋构建
- 容器设`perspective: 1400px`、`transform-style: preserve-3d`
- 预先定义好每张卡片在纸牌屋中的3D变换参数
- `backface-visibility: hidden`确保卡片背面不显示

### GSAP Flip 双态补间
- 调用`Flip.getState`捕获所有卡片在最终平面网格中的状态
- `Flip.from`自动计算从倚靠斜卡到平放网格的完整3D路径

### GSAP Timeline + ScrollTrigger
- 一条Timeline控制整体倒塌与展开
- 通过`stagger`和`position`参数实现从上到下的顺序
- 绑定`scrub: 2`，让滚动成为时间的控制杆

## 视觉设计

- **主色调**: 温暖纸质感 (#faf5e8, #ede0c8, #e5d4b0)
- **背景**: 深色木桌面 (#1a1510, #2a2218)
- **阴影**: 长阴影模拟室内暖光源
- **滤镜**: sepia(0.25) 营造怀旧纸质感

## 动画时序

| 时间 | 动画阶段 |
|------|----------|
| 0-1.5s | 入场动画：标题 + 纸牌屋弹性飞入 |
| 滚动 0-40% | 顶部金字塔卡片轻微晃动 |
| 滚动 40-100% | 依次倒塌到平面网格 |
| 向上滚动 | 逆序重建纸牌屋 |
