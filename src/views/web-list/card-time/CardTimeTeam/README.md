# CardTimeTeam - 管理团队 · 视差悬浮的名片

## 概述

CardTimeTeam 组件实现了"管理团队"效果 - 每位成员的照片卡片在滚动时产生微妙的景深视差，鼠标悬停时卡片轻微上浮并展露出带有关键能力和联系信息的详情层。

## 设计理念

"关于我们"的团队介绍，让每位成员的照片卡片在滚动时产生微妙的景深视差，鼠标悬停时卡片轻微上浮并展露出详情层。这种设计让公司形象立刻立体且人性化。

## 特性

### 核心动画

- **滚动视差**: 不同 `data-speed` 属性驱动不同层级的 Y 轴位移
- **3D 悬停**: `rotateY` 旋转 + `scale` 放大
- **详情层滑出**: 底部信息渐显
- **头像脉冲**: 环形装饰动画

## 技术实现

### 滚动视差

```typescript
gsap.to(card, {
  y: -30 * member.speed,
  ease: 'none',
  scrollTrigger: {
    trigger: gridRef.value,
    start: 'top bottom',
    end: 'bottom top',
    scrub: 1.5
  }
})
```

### 悬停详情层

```typescript
card.addEventListener('mouseenter', () => {
  gsap.to(card, {
    y: -15,
    scale: 1.03,
    rotateY: 5,
    duration: 0.4,
    ease: 'power2.out'
  })
  gsap.to(detail, {
    opacity: 1,
    y: 0,
    duration: 0.4
  })
})
```

- **头像光环**: 脉冲动画
