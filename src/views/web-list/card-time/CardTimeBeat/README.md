# CardTimeBeat - 霓虹节拍网格 · 脉冲扩散

脉冲从中心向外扩散的霓虹节拍网格动画。

## 效果说明

- **3x3 网格布局**：9张卡片排列成网格
- **脉冲扩散**：从中心卡片开始，涟漪般向外扩散
- **霓虹光效**：边框和阴影颜色动态变化
- **粒子背景**：漂浮闪烁的粒子增强氛围

## 技术实现

### 曼哈顿距离计算

```typescript
const dist = Math.abs(row - centerRow) + Math.abs(col - centerCol)
const delay = dist * 0.12
```

距离越远，延迟越大，实现从中心向外扩散效果。

### 动画阶段

1. **准备阶段**：所有卡片缩放到 0.9，透明度 0.6
2. **中心爆发**：中心光晕和脉冲波纹扩散
3. **脉冲扩散**：根据曼哈顿距离依次触发
4. **最终稳定**：所有卡片回归 0.95 缩放

## 交互控制

- **重播脉冲**：重新触发动画
- **开启循环**：持续循环脉冲效果
