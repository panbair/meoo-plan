# CardTimeElastic

悬浮弹性阵列动画卡片 - 灵动韵律，交互感炸裂

## 效果描述

卡片从下方弹入，采用 `back.out` 缓动制造过冲回弹，每张卡片延迟错落，并在弹入后轻微沉降，形成舒适的节奏感。

## 特性

- 弹性弹入动画 (`back.out(1.7)` 缓动 + 过冲回弹)
- 卡片滚动控制动画 (ScrollTrigger)
- 入场动画 (GSAP Timeline)
- 持续弹性浮动动画
- 背景光晕与粒子动态效果
- 响应式设计

## 技术栈

- Vue 3 Composition API
- TypeScript
- GSAP + ScrollTrigger
- SCSS
