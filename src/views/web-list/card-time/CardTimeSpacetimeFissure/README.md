# CardTimeSpacetimeFissure - 时空裂隙卡片组件

## 效果特点

- **五阶段维度演化**：碎片 → 旋涡聚合 → 切片扫描 → 光柱重组 → 网格回弹
- **碎片拖影与色散**：卡片运动时产生RGB色彩分离
- **全息投影质感**：动态扫描线、颗粒噪点、光辉闪烁
- **双向时间之箭**：五阶段完美逆向播放

## 技术实现

- **手动多阶段编排**：捕获网格坐标，分段切换目标位置
- **GSAP Timeline + ScrollTrigger**：`scrub: 1.8` 厚重拖拽感
- **Canvas 全息叠加层**：粒子场、扫描光栅
- **CSS自定义属性**：全局色移 `--chroma`

## 动画阶段

| 阶段 | 进度 | 效果 |
|------|------|------|
| 碎片 | 0-0.25 | 随机散布，blur(8px) |
| 旋涡 | 0.25-0.5 | 三维聚合，旋转 |
| 切片 | 0.5-0.75 | 一维扁平，灰度 |
| 光柱 | 0.75-1.0 | 高亮亮度，悬浮 |
| 网格 | 1.0-2.0 | elastic.out 回弹 |

## 使用说明

组件会被 `index.vue` 的 glob 模式自动扫描加载。
