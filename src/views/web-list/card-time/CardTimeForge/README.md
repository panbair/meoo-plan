# CardTimeForge

光影雕刻·逐字熔铸卡片组件 - 金属锻造效果

## 效果特点

- **锥形光束扫过**：从左上角向右下移动
- **逐字熔铸显现**：字符随光束进度依次从黑暗中浮现
- **灼烧金橙边缘**：`textShadow` 模拟金属灼烧效果
- **热力波纹扩散**：光束经过处产生波纹
- **持续灼烧动画**：字符持续脉动灼烧光晕

## 技术实现

- Vue 3 Composition API + TypeScript
- GSAP + ScrollTrigger
- `gsap.call()` 触发逐字动画
- CSS `textShadow` 模拟灼烧边缘
- Timeline 精细编排多层动画

## 动画参数

| 阶段 | 时长 | 说明 |
|------|------|------|
| 光束出现 | 0.15s | 渐显 |
| 光束移动 | 1.2s | 从左上到右下 |
| 字符显现 | ~0.5s | 每字延迟 0.08s |
| 灼烧消退 | 0.4s | 稳定为暖金色 |

## 设计风格

- 深黑金属基底
- 暖金/橙色调 (`#ffd700`, `#ff8c00`, `#ffb347`)
- Cinzel 衬线字体
- 角框装饰 + 热力粒子
