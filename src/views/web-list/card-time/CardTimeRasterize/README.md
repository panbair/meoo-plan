# CardTimeRasterize

栅格化入场卡片组件 - 全息线条分割重组效果

## 效果特点

- **栅格线条分割**：每张卡片分割为 20 条水平光栅线条
- **随机偏移入场**：线条初始随机 x 位移、透明度、scaleX
- **向心汇聚**：线条从边缘向中心收敛 (`from: 'center'`)
- **内容凝聚**：卡片内容从模糊虚空中浮现 (`blur(15px)` → `blur(0px)`)
- **线条消散**：线条逐渐淡出，留下纯净全息影像

## 技术实现

- Vue 3 Composition API + TypeScript
- GSAP + ScrollTrigger
- CSS 动态样式生成线条位置
- Timeline 编排多阶段动画

## 动画参数

| 阶段 | 时长 | 说明 |
|------|------|------|
| 线条汇聚 | 0.8s | 从随机位置向中心收敛 |
| 内容凝聚 | 0.6s | 从模糊到清晰 |
| 线条消散 | 0.3s | 逐渐淡出 |
| 卡片间隔 | 0.2s | 依次入场 |
