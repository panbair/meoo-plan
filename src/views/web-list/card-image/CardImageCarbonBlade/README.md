# CardImageCarbonBlade - 碳素刀锋·光学校准影院入场

## 效果描述

图片如一张未显影的银盐相纸，被两层质感迥异的黑白遮罩严密封锁。外层为厚重的碳黑色金属闸门，表面带有极细的拉丝纹理与冷冽的铣削光泽；内层为带有微弱珍珠光泽的云母光膜，极薄极透，隐约透出底层影像的模糊轮廓。

## 技术实现

### 核心动画阶段

1. **外层碳黑闸门滑落** (0.6s - 1.8s)
   - 三段式阻尼：前20% power2.out → 中段 linear → 末段 power4.in
   - 拉丝金属纹理 + 铣削光泽
   - 模拟精密机械的阻尼感

2. **内层云母光膜溶解** (1.8s - 3.0s)
   - 透明度从1降至0
   - 珍珠光泽短暂增强后消散
   - 虹彩色散偏移效果

3. **光学校准仪式** (1.8s - 4.0s)
   - blur: 20px → 0px
   - brightness: 0.3 → 1.0
   - scale: 1.05 → 1.0
   - hue-rotate: -2deg → 0deg

4. **云母碎片消散** (3.5s - 3.8s)
   - 边缘微粒光晕0.3s内消散

5. **四角暗角显现** (3.6s - 4.0s)
   - 影院放映机窗口效果

### 技术栈

- Vue 3 Composition API
- GSAP + ScrollTrigger
- SCSS 样式
- CSS 拉丝/铣削纹理

### ScrollTrigger 参数

| 参数 | 值 | 说明 |
|------|-----|------|
| trigger | parent | 触发器元素 |
| start | 'top 100%' | 元素顶部到达视口top 100%时开始 |
| end | 'bottom 20%' | 元素顶部到达视口bottom 20%时结束 |
| scrub | 1.5 | 平滑系数 |

## 视觉效果

- 碳黑闸门：拉丝金属 + 铣削光泽
- 三段式阻尼滑落动画
- 云母光膜：珍珠光泽 + 虹彩色散
- 光学校准：blur/brightness/scale 联动
- 云母碎片光晕
- 四角影院暗角
