# CardImageSkyTower 擎天浮廊 · 引力阶梯纵深视差

## 概述

页面被构建为一座违背物理法则的垂直悬浮建筑。三层元素各自锚定于不同的引力节点，呈现截然相反的运动惯性。

## 动画特效

| 层级 | 位移 | Scale | 特效 |
|------|------|-------|------|
| **柱阵背景** | 向下 20% | 1.0→1.04 | 亮度衰减 1.0→0.75 |
| **玻璃画作** | 向上 5% | 基准 | 亮度提升 0.85→1.05，微仰角 |
| **金属铭文** | 向上 30% | 基准 | blur脉冲，字母逐字弹射 |

## 核心特性

- 三层 scrub 阶梯式反向分配
- 滚动停止后独立阻尼惯性
- 字母 stagger 微差驱动
- 冷峻精密的建筑叙事美学

## 技术实现

- Vue 3 Composition API
- GSAP ScrollTrigger + scrub: true
- SCSS 样式
- will-change 优化
