# CardImageRollingShade 卷帘启封组件

## 简介

一个具有高级精密质感的图片入场动画组件，通过 6 条横向条带的交错错位与成对归位，模拟精密卷帘从两侧向中心收拢的仪式感效果。

## 动画效果

### 核心特性

1. **精密横向切割**
   - 6 条等宽横带
   - 使用 `clip-path: inset()` 精确裁切
   - 每条带带有纵向间距拉伸

2. **交错偏移**
   - 奇数带向左偏移 25px
   - 偶数带向右偏移 25px
   - 形成锯齿状错位视觉效果

3. **成对归位顺序**
   - 第 1 与第 6 条率先启动
   - 第 2 与第 5 条间隔 0.05s 跟进
   - 第 3 与第 4 条最后收束
   - 形成向中心对称收拢节奏

4. **冷色金属边缘**
   - 伪元素边框实现极细高光线
   - 错位时清晰可见
   - 归位后随切割线消失

5. **对接辉光**
   - 上下边缘辉光效果
   - 0.08s 峰值后 0.15s 衰减
   - 精密咬合瞬间

6. **百叶暗影**
   - 四角重复线性渐变
   - 模拟百叶窗帘效果
   - 5s 周期低频呼吸

## 技术实现

| 技术 | 用途 |
|------|------|
| CSS clip-path | 水平裁切 6 条横带 |
| CSS repeating-linear-gradient | 百叶暗影 |
| GSAP timeline | 成对归位动画 |
| GSAP ScrollTrigger | 滚动驱动 |
| SCSS | 样式与动画效果 |

## 参数说明

无外部参数，直接通过 ScrollTrigger 滚动控制

## 使用方式

```vue
<template>
  <CardImageRollingShade />
</template>
```

## 动画时间轴

```
0s       - 第1、6条开始归位
0.05s    - 第2、5条开始归位
0.1s     - 第3、4条开始归位
0-0.65s  - 所有横带归位 (power3.in)
0.5s     - 对接辉光触发
0.8s     - 亮度开始过渡
0.85s    - 切割线闪烁
0.95s    - 切割线消失
1.5s+    - 百叶暗影呼吸
```

## 归位顺序

```
[1] ←←←←←←                 →→→→→→ [6]
[2]  ←←←←←←               →→→→→→  [5]
[3]   ←←←←←←             →→→→→→   [4]
```

## 兼容性

- 需要浏览器支持 CSS clip-path
- 需要浏览器支持 CSS repeating-linear-gradient
- 需要 GSAP ScrollTrigger 插件
