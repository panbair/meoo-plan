# CardImageCurtainRise 帷幕聚拢 · 合幕成光开幕

## 概述

图片被两扇厚重的剧院帷幕覆盖，帷幕初始分置于画面左右外侧。Timeline 驱动两扇帷幕从左右两侧同时向画面中心沉稳聚拢，两帷幕在画面中线交汇的瞬间迸发出金色光缝，随即化为微光粒子消散，呈现一场沉稳而隆重的商务开幕仪式。

## 核心特性

| 特性 | 实现方式 |
|------|----------|
| **双帷幕** | 左右各55%宽度，深褐色织物纹理 |
| **帷幕聚拢** | translateX -120%/+120% → 0%，duration 1.5s |
| **中线光缝** | 金色辉光脉冲，opacity 0→0.95→0 |
| **帷幕消散** | 透明度渐变 + 弹性回缩 3px |
| **亮度过渡** | brightness 0.7→1.0 + 色温柔和 |
| **四角暗影** | 帷幕边框残余 + 5s 呼吸周期 |
| **ScrollTrigger** | scrub: 1.5, start: top 100%, end: bottom 20% |

## 动画时间线

1. **滚动触发**: 帷幕从画面外侧开始聚拢
2. **帷幕合拢**: 左右帷幕向中心移动，power2.inOut 节奏
3. **光缝迸发**: 交汇瞬间金色光缝闪现 0.15s
4. **光缝消散**: 快速消失 0.25s
5. **帷幕消散**: 透明度渐变 + 弹性回缩
6. **图片揭示**: 亮度渐升，色温柔和过渡
7. **暗角显现**: 四角帷幕暗影 + 低频呼吸
8. **内容**: 底部标题淡入

## 使用方式

```vue
<CardImageCurtainRise
  image-url="https://images.unsplash.com/photo-xxx"
  title="帷幕聚拢"
  subtitle="合幕成光开幕"
/>
```

## 视觉效果

- **帷幕颜色**: 深褐色渐变 (#1a1208 → #4a3419 → #1a1208)
- **织物纹理**: 纵向条纹 + 光泽高光
- **金色元素**: 光缝、边缘高光
- **呼吸动画**: 四角暗影 5s 周期低频脉动
