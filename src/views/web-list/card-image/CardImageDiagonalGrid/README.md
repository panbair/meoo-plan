# CardImageDiagonalGrid 斜格律动 · 幕墙鳞片逐格显影

## 概述

图片被一张精密模数化的斜向幕墙严密覆盖，幕墙由等宽的平行四边形鳞片组成，沿45°对角线网格排列。鳞片沿对角线方向依次解锁滑出，呈现建筑展馆的冷峻秩序感。

## 核心特性

| 特性 | 实现方式 |
|------|----------|
| **斜向幕墙** | 12列 x 7行 = 84片平行四边形鳞片 |
| **对角线解锁** | stagger 0.08s，从左上到右下依次滑出 |
| **45°滑出** | translateX/Y 组合沿对角线向外 |
| **冷光刻线** | 每片鳞片解锁瞬间高亮 0.05s |
| **亮度过渡** | 幕墙滑完后 brightness 0.75→1.0 |
| **四角框架** | 低频呼吸 6s 周期 |
| **ScrollTrigger** | scrub: 1.5, start: top 100%, end: bottom 20% |

## 动画时间线

1. **滚动触发**: 鳞片从左上角开始沿对角线方向依次滑出
2. **冷光刻线**: 每片解锁瞬间边缘高亮 0.05s 后渐隐
3. **幕墙解体**: 84片依次滑出，总时长约 7s
4. **图片亮相**: 亮度渐升 0.75→1.0
5. **框架阴影**: 四角框架渐显 + 低频呼吸
6. **内容**: 底部标题淡入

## 使用方式

```vue
<CardImageDiagonalGrid
  image-url="https://images.unsplash.com/photo-xxx"
  title="斜格律动"
  subtitle="幕墙鳞片逐格显影"
/>
```
