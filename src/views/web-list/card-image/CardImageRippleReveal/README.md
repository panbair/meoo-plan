# CardImageRippleReveal 水波沉影入场组件

## 简介

一个具有强烈视觉冲击力的图片入场动画组件，通过 SVG 滤镜实现涟漪扭曲效果，模拟图片从水下浮至镜面的禅意仪式。

## 动画效果

### 核心特性

1. **SVG 同心涟漪滤镜**
   - 使用 `feTurbulence` 生成径向涟漪噪声
   - `feDisplacementMap` 实现扭曲位移效果
   - `scale` 从 8 渐减至 0，模拟涟漪消退

2. **石子落水脉冲**
   - 涟漪消退末段短暂升至 4 后急速归零
   - 模拟最后一圈涟漪向外扩散的物理瞬间

3. **多层涟漪扩散**
   - 3 个同心涟漪圈依次扩散
   - 中心脉冲石子落水效果
   - 视觉层次丰富

4. **透明度溶解**
   - 图片 `opacity` 从 0.4 升至 1.0
   - 水色遮罩 `opacity` 从 0.3 归零
   - 同步驱动，视觉连贯

5. **四角水光折射**
   - 四角叠加极淡水光折射
   - 低频呼吸动画
   - 强化水面刚恢复平静的刹那

## 技术实现

| 技术 | 用途 |
|------|------|
| SVG feTurbulence | 生成同心涟漪噪声纹理 |
| SVG feDisplacementMap | 位移映射扭曲 |
| GSAP ScrollTrigger | 滚动驱动动画 |
| SCSS | 样式与动画效果 |

## 参数说明

无外部参数，直接通过 ScrollTrigger 滚动控制

## 使用方式

```vue
<template>
  <CardImageRippleReveal />
</template>
```

## 动画时间轴

```
0s    - 涟漪圈开始扩散
0-2.2s - 位移 scale 从 8 消退至 0
1.9s  - 石子落水脉冲 (scale 升至 4)
2.05s - 脉冲归零
0-2s  - 透明度溶解
0.8s  - 四角水光开始消散
```

## 兼容性

- 需要浏览器支持 SVG 滤镜
- 需要浏览器支持 CSS 动画
- 需要 GSAP ScrollTrigger 插件
