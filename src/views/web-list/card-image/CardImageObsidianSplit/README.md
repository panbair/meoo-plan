# CardImageObsidianSplit 曜黑对开 · 双幕合拢商务启幕

## 概述

图片被两扇带有精密铣削纹理的曜黑金属帷幕覆盖，帷幕初始分置于画面左右两侧之外。Timeline 驱动两扇帷幕从左右两侧同时向画面中心沉稳推进，采用三段式阻尼节奏，中线碰撞瞬间迸发冷白光隙并激发拉丝纹理涟漪，帷幕随之蒸发渐隐，呈现精密冷冽的商务启幕仪式。

## 核心特性

| 特性 | 实现方式 |
|------|----------|
| **曜黑金属帷幕** | 深黑色渐变 (#0a0b0e → #22262f)，冷冽质感 |
| **竖直拉丝纹理** | repeating-linear-gradient 3px 间隔 |
| **三段式缓动** | power2.inOut 1.8s，模拟精密导轨阻尼 |
| **冷白光隙** | 极细冷白竖线，box-shadow 脉冲 |
| **纹理涟漪** | background-position-y 垂直偏移 |
| **金属蒸发** | opacity 渐变 + 弹性回缩 |
| **亮度对比** | brightness 0.65→1.0 + contrast 1.05→1.0 |
| **框架阴影** | 6s 周期低频呼吸 |
| **ScrollTrigger** | scrub: 1.5, start: top 100%, end: bottom 20% |

## 动画时间线

1. **滚动触发**: 金属帷幕从画面外侧开始聚拢
2. **帷幕推进**: 三段式阻尼节奏，前缓中匀末加速
3. **中线碰撞**: 冷白光隙闪现 0.15s → 快速收敛 0.3s
4. **拉丝涟漪**: 纹理垂直扩散 0.4s
5. **冷光脉冲**: 帷幕 box-shadow 冷白辉光
6. **帷幕蒸发**: 透明度渐变 + 弹性回缩
7. **图片揭示**: 亮度渐升，对比度微降
8. **框架阴影**: 四角阴影 + 6s 呼吸
9. **内容**: 底部标题淡入

## 视觉效果

- **帷幕颜色**: 曜黑渐变 (#0a0b0e → #22262f → #0a0b0e)
- **拉丝质感**: 3px 竖直条纹 + 环境微光
- **冷白元素**: 光隙、边缘高光、辉光脉冲
- **呼吸动画**: 四角框架阴影 6s 周期
