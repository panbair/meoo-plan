# CardImageRockCrack - 岩层震裂·浮雕爆破炸裂

## 概述

图片如同一幅被封印于黑色岩层深处的石刻浮雕，从中心向外呈层叠式依次炸裂飞散，带有炽热边缘光效和重力下坠效果。

## 核心特效

| 特效 | 描述 |
|------|------|
| **不规则碎片** | 20枚使用 clip-path: polygon() 随机顶点的不规则碎片 |
| **前震抖动** | 炸裂前高频微小震动 |
| **层叠爆破** | 从中心向外环依次炸裂，模拟冲击波传递 |
| **3D高速旋转** | rotateX/Y ±180deg, rotateZ ±540deg |
| **重力下坠** | 碎片飞行轨迹带抛物线下坠 |
| **炽热边缘** | 炸裂瞬间橙红炽光 0.1s 峰值 |
| **中心余烬** | 低频呼吸脉动的炽热光点 |

## 动画阶段

1. **前震阶段** (`top 85-70%`) - 碎片高频抖动 + 裂纹叠加
2. **爆破阶段** (`top 65-20%`) - 冲击波 + 层叠炸裂飞散
3. **余烬阶段** (`top 30% - bottom`) - 中心余烬呼吸

## 使用图片

火山岩浆纹理图片，配合橙红炽热配色。

## 技术实现

- Vue 3 Composition API
- GSAP ScrollTrigger
- clip-path: polygon() 不规则碎片
- CSS 3D transform
- power4.in 爆发缓动
