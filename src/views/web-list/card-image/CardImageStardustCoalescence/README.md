# CardImageStardustCoalescence

> 星尘归墟 · 游丝聚合入场

## 效果描述

图片被分解为 120 枚极细的微粒碎片，初始悬浮于暗空之中，如同一团被搅散的星尘遗骸。碎片各自携带着极其慵懒的布朗运动——缓慢漂移、微微旋转、半透明且笼罩在高斯模糊之中。

Timeline 驱动无形引力场从画面中心悄然启动。碎片如同受到极远处星体的温和牵引，向中心聚拢。越靠近中心越清晰凝实，最终如同雾气轻柔贴附于镜面，聚合形成完整图片。

## 核心技术

| 技术 | 说明 |
|------|------|
| Vue 3 | Composition API |
| GSAP | Timeline + ScrollTrigger |
| CSS | 模糊度联动 + 透明度渐变 |
| SCSS | 深空背景 + 星点场 |

## 动画阶段

| 阶段 | ScrollTrigger | 效果 |
|------|---------------|------|
| 星尘入场 | top 85% | 碎片弥散状态 |
| 游丝聚合 | top 85% - center | 向心聚拢 2.5s |
| 亮度过渡 | center - bottom | 亮度提升 |

## 碎片配置

120 枚碎片 (12×10)：

- 边缘碎片弥散更大 (±150px)
- 中心碎片更凝实 (±40px)
- 透明度: 0.25-0.5
- 模糊度: 4-8px
- 旋转: ±15deg

## 视觉特效

- **深空背景**: 暗紫渐变 + 星点场
- **弥散碎片**: 高模糊 + 半透明
- **渐进聚合**: 模糊→清晰, 分散→归位
- **聚合光晕**: 3层扩散环
- **微光闪烁**: 聚合瞬间边缘闪光
- **边缘游丝**: 5缕呼吸消散

## 动画曲线

`power2.inOut` 缓动：
- 前 60%: 极缓慢（遥远引力牵引）
- 后 40%: 微微加速（接近引力源）
