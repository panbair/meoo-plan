# CardImageCarbonCrystal 碳素晶格 · 斜纹光蚀逐格消隐

## 概述

图片被一层精密编织的碳纤维斜纹网格严密覆盖，网格由等宽的黑色哑光斜向条纹构成。激光校准仪沿对角线方向逐条切断碳纤维，使条纹从固态黑色渐变为完全透明，边缘泛起冷白光蚀刻线。

## 核心特性

| 特性 | 实现方式 |
|------|----------|
| **碳纤维斜纹** | 48条45度斜向条纹，碳纤维哑光质感 |
| **激光蚀刻** | 条纹渐变透明 + 边缘冷白光高光 |
| **对角线消隐** | stagger 0.07s，从左上到右下依次触发 |
| **亮度过渡** | brightness 0.65→1.0 + 色温微冷→中性 |
| **饱和度修正** | saturate 0.9→1.0 |
| **四角暗角** | 网格纹理 + 低频呼吸 6s 周期 |
| **ScrollTrigger** | scrub: 1.5, start: top 100%, end: bottom 20% |

## 动画时间线

1. **滚动触发**: 激光蚀刻从左上角沿对角线方向启动
2. **条纹消隐**: 48条条纹依次透明化，duration 0.45s
3. **边缘高光**: 每条消隐瞬间冷白辉光闪烁 0.08s
4. **图片揭示**: 亮度渐升 0.65→1.0，色温微冷→中性
5. **暗角显现**: 四角网格暗角渐显 + 低频呼吸
6. **内容**: 底部标题淡入

## 使用方式

```vue
<CardImageCarbonCrystal
  image-url="https://images.unsplash.com/photo-xxx"
  title="碳素晶格"
  subtitle="斜纹光蚀逐格消隐"
/>
```
