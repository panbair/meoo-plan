# CardImageDeepSeaCoffin 深海悬棺 · 多维空间沉浸视差

## 概述

图片如一口被沉入深海暗渊的青铜棺椁，悬浮于无尽幽蓝之中。三层独立景深（远景水体、中景棺椁、近景雪屑）以差异化速率运动，营造深海沉浸感。

## 核心特性

| 特性 | 实现方式 |
|------|----------|
| **三层景深** | 远景水体(0.02x) / 中景棺椁(0.1x) / 近景雪屑(0.3x) |
| **青铜棺椁** | 金属质感边框 + 青铜锈蚀纹理 + 上下浮沉 ±8px |
| **Y轴旋转** | rotateY -1.5deg 至 +1.5deg 缓动 |
| **光柱效果** | 从左上向右下移动，照亮锈蚀纹理 |
| **亮度对比联动** | 浮升时 +5%亮度/-3%对比，下沉时反之 |
| **深海雪屑** | 40个粒子 + 独立荧光闪烁 |
| **ScrollTrigger** | scrub: 1.5, start: top 100%, end: bottom 20% |

## 动画时间线

1. **滚动驱动**: 三层景深同步启动，差异化速度
2. **水体层**: 极缓下沉 + 透明度波动
3. **棺椁层**: 上下浮沉 + Y轴旋转 + 亮度对比联动
4. **光柱**: 从左上向右下偏转，照亮锈蚀
5. **雪屑层**: 快速上浮 + 荧光闪烁
6. **内容**: 底部标题淡入

## 使用方式

```vue
<CardImageDeepSeaCoffin
  image-url="https://images.unsplash.com/photo-xxx"
  title="深海悬棺"
  subtitle="多维空间沉浸视差"
/>
```
