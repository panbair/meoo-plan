# CardImagePulsarBurst

> 脉冲星爆 · 光子溅射炸裂

## 效果描述

图片如一颗被引力压缩至极致的脉冲星，初始完整悬浮于深空，但表面已出现能量过载的预兆——画面以极微幅度高频震颤，边缘向外渗出脉冲状的蓝紫色光晕。

Timeline 触发塌缩临界点。图片中心点瞬间坍缩，随即以数十倍强度向外爆发出环形激波。碎片沿径向以光子速度向外溅射，每枚碎片绕自身多轴高速旋转，边缘因光子摩擦产生极细的蓝紫辉光拖尾。

## 核心技术

| 技术 | 说明 |
|------|------|
| Vue 3 | Composition API |
| GSAP | Timeline + ScrollTrigger |
| clip-path | polygon() 不规则碎片 |
| CSS | 蓝紫辉光 + 深空背景 |

## 动画阶段

| 阶段 | ScrollTrigger | 效果 |
|------|---------------|------|
| 脉冲预震 | top 85-65% | 高频震颤 + 脉冲光晕 |
| 光子溅射 | top 60-10% | 激波 + 层叠炸裂 |
| 脉冲星遗迹 | top 15%- | 脉动核心 + 旋转光束 |

## 碎片分组

按同心环带分层，共 4 圈：

- Ring 1 (中心): 偏移 150px, 延迟 0.06s
- Ring 2: 偏移 250px, 延迟 0.12s
- Ring 3: 偏移 350px, 延迟 0.18s
- Ring 4 (边缘): 偏移 450px, 延迟 0.24s

## 视觉特效

- **深空背景**: 暗紫渐变 + 星点场
- **脉冲光晕**: 3层扩散环
- **蓝紫辉光**: 碎片边缘多层 box-shadow
- **光子拖尾**: 蓝紫→暗红颜色过渡
- **激波扩散**: 中心环形波纹
- **脉冲星遗迹**: 核心脉动 + 光束旋转
