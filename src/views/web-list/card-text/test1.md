## 1. 组件概述

### 1.1 基本信息

| 属性 | 值 |
|------|-----|
| **组件名称** | CardTextBlur |
| **唯一后缀** | `104` (所有类名必须使用此后缀) |
| **文件路径** | `src/views/web-list/card-text/CardTextBlur/CardTextBlur.vue` |
| **功能类型** | 滚动触发的文字模糊揭示动画 |

### 1.2 核心功能

CardTextBlur 是一个基于 GSAP ScrollTrigger 的高级文字动画组件，实现以下视觉效果：

✨ **四重动画融合**
- **模糊渐变**: `blur(25px)` → `blur(0px)` - 从模糊到清晰
- **位移动画**: Y轴 +150px, X轴 -100px - 空间层次感
- **缩放动画**: `scale(0.9)` → `scale(1)` - 呼吸感
- **透明度**: `opacity(0)` → `opacity(1)` - 淡入效果

🎨 **视觉设计**
- 深色渐变背景 (紫蓝色系)
- 径向渐变光晕装饰
- 渐变文字效果 (标题)
- 毛玻璃卡片样式 (段落)

### 1.3 应用场景

- ✅ 企业官网"关于我们"页面
- ✅ 产品展示页特性介绍
- ✅ 博客文章开篇引言
- ✅ Landing Page 核心价值主张
