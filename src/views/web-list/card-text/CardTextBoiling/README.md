# CardTextBoiling - 沸腾气泡·烧杯文字聚合

化学烧杯盛满透明液体，底部加热，文字以隐性锚点阵潜伏于液面下。气泡群密度达到临界时，瞬间凝结为文字。

## 动画特点

- **化学烧杯**: 透明液体容器，底部加热效果
- **气泡生成**: 从文字锚点密集涌出，大小不一
- **沸腾加剧**: Timeline 逐步提升气泡生成速率
- **凝结效果**: 气泡收缩凝结成清晰文字
- **液体折射**: 文字带有轻微扭曲和光影波动
- **搅拌交互**: 鼠标拖拽触发局部文字解离

## 技术实现

- Vue 3 Composition API + TypeScript
- GSAP 动画引擎 + ScrollTrigger
- DOM 气泡元素 + 动态创建
- CSS 渐变 + 动画模拟液体效果
- 随机物理运动模拟

## 使用示例

```vue
<template>
  <CardTextBoiling 
    title="沸腾气泡" 
    subtitle="化学的魔法 · 液体的诗篇"
    :bubble-intensity="1"
  />
</template>
```

## Props

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| title | string | '沸腾气泡' | 标题文字 |
| subtitle | string | '化学的魔法 · 液体的诗篇' | 副标题 |
| bubbleIntensity | number | 1 | 气泡强度 (0-2) |

## 动画流程

1. **预热**: 少量气泡开始生成
2. **沸腾**: 气泡数量和速率逐渐增加
3. **临界**: 气泡最密集状态
4. **凝结**: 文字显现，气泡消散
5. **稳定**: 文字完整展示，少量气泡
6. **交互**: Hover 放大 + 拖拽搅拌
