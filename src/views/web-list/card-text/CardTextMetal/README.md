# CardTextMetal 记忆合金组件

## 效果描述

镍钛诺合金丝从混乱状态经热激活完成形状记忆跳变，文字如精密焊接的金属浮雕显现。

## 动画阶段

| 阶段 | 效果 |
|------|------|
| 混乱 | 金属丝平铺呈杂乱状态 |
| 通电 | 电阻加热，颜色渐变 |
| 跳变 | 瞬间归直，重构字形 |
| 热浪 | 向外辐射波纹 |
| 冷却 | 哑光银白金属质感 |
| 交互 | 鼠标划过局部升温 |

## 技术实现

- SVG 路径模拟金属丝
- d 属性动画实现形状跳变
- 颜色梯度动画模拟热效应
- 同心波纹动画模拟热辐射
- GSAP quickTo 实现交互

## 使用

```vue
<CardTextMetal 
  title="记忆合金" 
  subtitle="热激活 · 形状记忆"
/>
```
