# CardTextSpikes 磁流体尖刺组件

## 效果描述

黑色金属平台上的磁流体，电流驱动磁场增强，磁流体表面生出密集的针状尖刺，沿文字轮廓排列成锋利荆棘。鼠标靠近时尖刺被吸引偏转。

## 动画阶段

| 阶段 | 效果 |
|------|------|
| 入场 | 磁流体平静铺展 |
| 激活 | 磁场逐渐增强 |
| 尖刺 | 针状尖刺沿文字轮廓生长 |
| 稳定 | 尖刺微颤如磁场涟漪 |
| 交互 | 鼠标吸引尖刺偏转 |

## 技术实现

- 尖刺网格覆盖文字区域
- Timeline 控制各区尖刺高度
- yoyo 微幅颤动模拟磁场涟漪
- quickTo 鼠标吸引效果

## 使用

```vue
<CardTextSpikes 
  title="磁流体" 
  subtitle="尖刺文字 · 磁场显形"
/>
```
