# CardTextQuantumCollapse 量子观测坍缩·文字概率云显影

## 概述

这是一个模拟量子力学中波函数坍缩效果的动画文本组件。所有文字处于未被观测的叠加态，呈现为多个半透明虚影随机偏移的概率云。观测者视线扫过文字时，概率云急速向同一位置坍缩，多重虚影逐层消除。坍缩完成瞬间，文字凝为确定形态，仅边缘保留极细的量子微光晕。

## 效果特性

### 核心视觉效果
- **概率云副本层**：每个字符由 5 层不同透明度、不同偏移的副本文本叠加
- **波函数坍缩**：所有副本同时向中心收敛并淡出
- **量子闪变**：坍缩过程中随机亮度闪烁，模拟量子不确定性
- **观测波释放**：坍缩完成瞬间释放环形观测波
- **边缘量子微光**：稳定态文字边缘微弱的呼吸光晕
- **高斯模糊**：初始状态每层带有 blur 效果

### 交互效果
- **鼠标退相干扰动**：hover 时局部短暂重回叠加态
- **副本分离再坍缩**：鼠标离开后快速分离并再次坍缩
- **弹性回归**：副本层弹性归位

### 氛围元素
- **深空网格背景**：透视网格动画
- **量子噪声层**：动态噪点纹理
- **量子波动画布**：双层波动线条
- **量子粒子**：40个随机闪烁的量子粒子
- **量子环装饰**：多层脉冲圆环

## 技术实现

### 动画技术
- **GSAP Timeline**：控制整体动画时序和滚动绑定
- **ScrollTrigger**：滚动触发，支持 scrub 平滑系数
- **Canvas 波动**：双层量子波动绘制
- **CSS 变量**：动态偏移和模糊值
- **随机闪变**：Timeline 中插入随机 opacity 噪点

### 动画时序
1. **0.2-0.7s**：副本层向中心收敛
2. **0.5-0.9s**：副本层渐变淡出
3. **0.7-0.9s**：主层出现
4. **0.75-1.1s**：8次随机闪变
5. **1.1-1.2s**：观测波释放
6. **1.3s+**：进入稳定态呼吸和鼠标交互

### 关键参数
| 参数 | 值 | 说明 |
|------|-----|------|
| 滚动触发起点 | `top 80%` | 元素顶部到达视口80%时开始 |
| 滚动触发终点 | `top 20%` | 元素顶部到达视口20%时结束 |
| 滚动平滑系数 | `1.5` | 值越大动画越平滑 |
| 副本层数 | `5` | 每字符的副本数量 |
| 初始模糊 | `6px` | 副本层初始模糊值 |

- 使用 will-change 优化动画性能
