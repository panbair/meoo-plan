# CardTextNeonSign 霓虹电弧·招牌剧场

## 概述

这是一个模拟深夜湿街复古霓虹招牌的动画文本组件。文字以黑色灯管轮廓存在，Timeline 启动后，高压变压器逐一为字母路径通电，电流伴随蓝紫电弧在管端跳跃，灯管剧烈闪烁数次后渐稳，迸发过饱和的粉橙辉光。

## 效果特性

### 核心视觉效果
- **霓虹灯管击穿效果**：每个字符依次被高压击穿点亮，伴随3次闪烁
- **电弧跳跃动画**：蓝紫电弧在字符间随机跳跃，使用 Canvas 绘制
- **多层光晕叠加**：三层 `text-shadow` 嵌套，产生真实霓虹辉光
- **变压器装饰**：左右两侧模拟高压变压器元素
- **地面倒影**：湿漉地面的霓虹文字倒影，持续微晃

### 交互效果
- **低频脉动呼吸**：稳定后每个字符做低频 opacity 呼吸
- **鼠标悬停闪烁**：hover 时触发电压不稳的快速频闪
- **电压指示器**：实时显示通电进度和电压值

### 氛围元素
- **雨夜效果**：斜向雨丝动画
- **氛围粒子**：漂浮的发光粒子
- **地面反光**：渐变地面反光效果

## 技术实现

### 动画技术
- **GSAP Timeline**：控制整体动画时序和滚动绑定
- **ScrollTrigger**：滚动触发，支持 scrub 平滑系数
- **Canvas 电弧**：动态生成随机电弧路径
- **SVG 电弧**：字符间的电弧连接效果
- **CSS 动画**：雨丝下落、呼吸脉动、粒子漂浮

### 动画时序
1. **0-0.3s**：边框入场，雨丝和边框渐显
2. **0.2-0.4s**：变压器出现
3. **0.3-0.6s**：氛围粒子入场
4. **0.4-1.2s**：主标题字符依次击穿点亮
5. **0.8-1.0s**：副标题字符依次点亮
6. **1.2-1.6s**：地面倒影渐显
7. **1.3s+**：进入稳定呼吸状态

### 关键参数
| 参数 | 值 | 说明 |
|------|-----|------|
| 滚动触发起点 | `top 80%` | 元素顶部到达视口80%时开始 |
| 滚动触发终点 | `top 20%` | 元素顶部到达视口20%时结束 |
| 滚动平滑系数 | `1.5` | 值越大动画越平滑 |
| 电弧刷新间隔 | `300ms` | 随机电弧生成频率 |
| 呼吸脉动周期 | `2-3s` | 字符呼吸效果周期 |

## 样式变量

| 变量 | 默认值 | 说明 |
|------|--------|------|
| `--char-color` | `#1a1a1a` | 初始字符颜色（未点亮时） |
| 主霓虹色 | `#ff6b9d` | 主标题点亮后的粉橙色 |
| 副霓虹色 | `#00d4ff` | 副标题点亮后的青色 |
| 背景色 | `#0a0a12` | 深邃的夜间背景 |

## 浏览器兼容性

- 支持现代浏览器（Chrome, Firefox, Safari, Edge）
- 需要支持 CSS 变量和 CSS 动画
- Canvas 和 SVG 需要支持相应的 API

## 性能考虑

- 电弧动画使用 requestAnimationFrame
- 滚动时自动暂停非可视区域动画
- 组件卸载时清理所有定时器和动画
