# CardTextPrismDispersion 棱镜色散·光谱合光剧场

## 概述

这是一个模拟光学棱镜色散与合光效果的动画文本组件。文字被预先拆解为红、绿、蓝三原色光层，各层沿水平轴向大幅错位偏移，如同透过棱镜观察的色散虚影。Timeline 驱动各色层由两侧向内滑移归位，合光完成瞬间文字凝为明亮纯白，仅边缘保留极细的彩色镶边。

## 效果特性

### 核心视觉效果
- **三原色层分离**：红、绿、蓝三层文字沿水平轴大幅偏移
- **高斯模糊与色晕**：每层带有模糊效果和强烈光晕
- **混合模式叠加**：使用 `mix-blend-mode: screen` 实现真实色光合光
- **彩虹条纹爆发**：合光瞬间迸发12条彩虹条纹
- **环形眩光**：三层 `box-shadow` 不同色相快速缩放
- **边缘彩色镶边**：最终文字边缘保留动态彩虹边缘

### 交互效果
- **鼠标局部色散**：hover 时文字局部再次产生动态色散
- **弹性回归**：鼠标离开后三色层凭借 elastic.out 弹性回归
- **实时色散指示器**：底部显示当前色散百分比

### 氛围元素
- **布朗运动粒子**：Canvas 绘制的高斯模糊彩色粒子
- **漂浮粒子**：30个随机漂浮的光谱粒子
- **旋转棱镜装饰**：右上角旋转的棱镜图标

## 技术实现

### 动画技术
- **GSAP Timeline**：控制整体动画时序和滚动绑定
- **ScrollTrigger**：滚动触发，支持 scrub 平滑系数
- **mix-blend-mode: screen**：实现色光混合叠加
- **Canvas 粒子系统**：布朗运动粒子效果
- **CSS 动画**：彩虹边缘流动、粒子漂浮

### 动画时序
1. **0-0.3s**：红蓝层入场，大幅偏移
2. **0.3-0.7s**：绿层先行归位
3. **0.5s**：彩虹条纹爆发
4. **0.6-0.8s**：红蓝层滑入归位
5. **0.9-1.0s**：环形眩光爆发
6. **1.0-1.2s**：白色层最终凝合
7. **1.2s+**：三色层淡出到边缘，进入交互状态

### 关键参数
| 参数 | 值 | 说明 |
|------|-----|------|
| 滚动触发起点 | `top 80%` | 元素顶部到达视口80%时开始 |
| 滚动触发终点 | `top 20%` | 元素顶部到达视口20%时结束 |
| 滚动平滑系数 | `1.5` | 值越大动画越平滑 |
| 红蓝层初始偏移 | `±400px` | 色散分离时的偏移量 |
| 彩虹条纹数 | `12` | 条纹数量 |

## 样式变量

| 变量 | 默认值 | 说明 |
|------|--------|------|
| 红色 | `#ff3366` | 红色层主色 |
| 绿色 | `#33ff66` | 绿色层主色 |
| 蓝色 | `#3366ff` | 蓝色层主色 |

## 浏览器兼容性

- 支持现代浏览器（Chrome, Firefox, Safari, Edge）
- 需要支持 CSS 混合模式（mix-blend-mode）
- Canvas 和 SVG 需要支持相应的 API

## 性能考虑

- 粒子动画使用 requestAnimationFrame
- 使用 will-change 优化动画性能
- 组件卸载时清理所有定时器和动画
