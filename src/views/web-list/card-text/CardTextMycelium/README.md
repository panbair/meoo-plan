# CardTextMycelium 菌丝网络组件

## 效果描述

黑暗森林腐殖土表层，菌丝从字形节点向外延伸交织成网络，荧光球体在交汇处生长，文字如蘑菇从菌床萌发。

## 动画阶段

| 阶段 | 效果 |
|------|------|
| 播种 | 菌丝节点分布于文字笔画 |
| 生长 | 丝线沿笔画路径延伸 |
| 交织 | 交汇处生长荧光球体 |
| 成字 | 文字轮廓完整显现 |
| 脉动 | 菌丝背景持续呼吸 |
| 交互 | 鼠标扰动菌丝收缩 |

## 技术实现

- SVG 路径绘制菌丝网络
- stroke-dashoffset 动画实现延伸效果
- 荧光球体 scale 弹入动画
- requestAnimationFrame 驱动孢子粒子
- GSAP quickTo 实现鼠标扰动

## 使用

```vue
<CardTextMycelium 
  title="菌丝网络" 
  subtitle="黑暗森林 · 生命的网络"
/>
```
