# CardTextSeismograph - 声波转录·地震仪文字谱记

实验室暗房，一台虚构的转筒地震仪缓缓旋转，纸面横向走纸。文字不以字形存在，而是一条连续的黑色声波振幅曲线。

## 动画特点

- **声波曲线**: 文字初始状态为压缩的声波振幅曲线
- **逐字凝结**: Timeline 驱动声波振幅放大，逐渐勾勒出文字轮廓
- **转筒视差**: 滚动时背景转筒产生水平视差效果
- **墨点飞溅**: 文字凝结瞬间边缘溅出极细的墨点飞沫
- **时间刻度**: 淡黄色细线铺底的时间刻度网格
- **Hover还原**: 鼠标划过局部文字短暂还原为波形曲线

## 技术实现

- Vue 3 Composition API + TypeScript
- GSAP 动画引擎 + ScrollTrigger
- SVG 路径动画 (d 属性渐变)
- Canvas 墨点粒子系统
- CSS 3D Transform 视差效果

## 使用示例

```vue
<template>
  <CardTextSeismograph 
    title="声波记录" 
    subtitle="振幅的轨迹 · 频率的诗篇"
    :wave-intensity="1"
  />
</template>
```

## Props

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| title | string | '声波记录' | 标题文字 |
| subtitle | string | '振幅的轨迹 · 频率的诗篇' | 副标题 |
| waveIntensity | number | 1 | 波形强度 (0-2) |

## 动画流程

1. **入场**: 声波曲线以压缩状态淡入
2. **振荡**: 波形振幅逐渐放大，产生震荡
3. **凝结**: 振幅收敛，文字轮廓显现
4. **墨溅**: 凝结瞬间触发墨点飞沫
5. **完成**: 声波淡出，文字完整呈现
6. **Hover**: 鼠标划过局部还原为波形
