# CardCircuit 电路脉冲卡片

> 酷炫的电路脉冲动画效果卡片

## 特性

- ⚡ **电路背景** - 动态脉冲的电路线路
- 🔌 **节点动画** - 发光节点的脉冲效果
- 💚 **翠绿渐变** - 科技感的翠绿色调
- 💫 **3D透视** - 增强的3D变换和模糊效果

## 技术栈

- Vue 3 Composition API
- TypeScript
- GSAP + ScrollTrigger
- SCSS

### 电路背景动画
- 7条电路线路持续闪烁
- 节点脉冲动画周期: 1.5秒
- 发光效果: 翠绿色光晕

### 关键改进
1. **fade-up-right效果** - 从右下角滑入的入场动画
2. **电路背景** - 动态脉冲的电路线路
3. **节点脉冲** - hover时卡片的电流流动效果
4. **模糊增强** - 卡片最高12px模糊
5. **3D透视** - 1200px透视距离
6. **will-change优化** - 提升动画性能
7. **cleanup清理** - 组件卸载时清理所有动画
