# CardGravity - 重力吸附效果卡片

## 效果说明

模拟重力场效果，卡片在滚动时向中心引力点聚集，带有轨道旋转和引力场增强效果。

## 动画效果

| 动画类型 | 描述 |
|---------|------|
| 标题入场 | 弹性缩放入场动画 |
| 卡片聚集 | 滚动时卡片从外围向中心移动 |
| 轨道旋转 | 每张卡片带有旋转轨道动画 |
| 引力场 | 引力场光晕随滚动增强 |
| 中心脉冲 | 中心引力点波纹持续扩散 |

## 技术实现

- **GSAP ScrollTrigger**: 滚动控制卡片移动
- **CSS Animation**: 轨道和波纹持续动画
- **动态位置计算**: 卡片根据索引计算初始位置

## 使用方式

