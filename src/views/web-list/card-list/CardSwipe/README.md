# CardSwipe 滑动刷新卡片

> 酷炫的滑动刷新动画效果卡片

## 特性

- ↔️ **滑动效果** - 背景持续滑动的线条动画
- 💫 **滑动光效** - hover时卡片的滑动光效
- 💗 **粉紫渐变** - 浪漫的粉紫色调
- 💫 **3D透视** - 增强的3D变换和模糊效果

## 技术栈

- Vue 3 Composition API
- TypeScript
- GSAP + ScrollTrigger
- SCSS

## 动画效果

### 入场动画 (fade-left)

| 元素 | 位移 | 旋转 | 缩放 | 模糊 | 缓动 |
|------|------|------|------|------|------|
| 徽章 | x: 150→0 | 180°→0° | 0→1 | 15px→0 | back.out |
| 标题 | x: 120→0 | - | 0.8→1 | 10px→0 | power3.out |
| 副标题 | x: 80→0 | - | - | - | power2.out |

### 卡片入场 (fade-left)

| 卡片 | 位移 | 旋转 | 倾斜 | 缩放 | 缓动 |
|------|------|------|------|------|------|
| 下拉刷新 | x: 200 | -10° | skewY: 5° | 0.6→1 | back.out |
| 侧滑操作 | x: 250 | 0° | - | 0.5→1 | elastic.out |
| 弹性回弹 | x: 200 | 10° | skewY: -5° | 0.6→1 | back.out |

### 滚动消失动画

| 卡片 | 滑出方向 | 位移 | 旋转 | 倾斜 | 缩放 | 模糊 |
|------|---------|------|------|------|------|------|
| 下拉刷新 | 左 | -300px | -25° | skewY: -10° | 0.3 | 0→12px |
| 侧滑操作 | 左 | -350px | 0° | - | 0.2 | 0→12px |
| 弹性回弹 | 左 | -300px | 25° | skewY: 10° | 0.3 | 0→12px |

### 背景滑动线条动画
- 8条滑动线条持续下落
- 周期: 2秒循环
- 长度: 60-160px 随机
- 颜色: 粉色渐变透明

### 关键改进
1. **fade-left效果** - 从右侧滑入的入场动画
2. **滑动线条** - 动态下落的背景线条
3. **卡片光效** - hover时卡片的滑动光效
4. **模糊增强** - 卡片最高12px模糊
5. **3D透视** - 1200px透视距离
6. **will-change优化** - 提升动画性能
7. **cleanup清理** - 组件卸载时清理所有动画
