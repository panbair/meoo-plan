# CardShatter - 破碎效果卡片

## 效果说明

滚动触发卡片破碎爆炸动画，碎片向四周分散，带有旋转和缩放效果。

## 动画效果

| 动画类型 | 描述 |
|---------|------|
| 标题入场 | 文字逐字符旋转入场，带有3D透视效果 |
| 卡片旋转 | 滚动时卡片Y轴旋转360度 |
| 碎片爆炸 | 12个碎片向四周爆炸分散 |
| 进度指示 | 底部彩色进度条跟随滚动 |
| 背景粒子 | 背景光晕持续旋转 |

## 技术实现

- **GSAP ScrollTrigger**: 滚动触发破碎动画
- **CSS 3D Transform**: 碎片3D旋转
- **动态样式变量**: 每个碎片随机爆炸方向

## 使用方式

```vue
<CardShatter />
```
