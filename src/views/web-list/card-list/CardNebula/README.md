# CardNebula 星云卡片

星云主题的酷炫卡片组件，元素从左侧滑入展示宇宙星云的梦幻效果。

## 组件特性

- **slide-right 入场动画**: 从左侧 -100% 滑入到原位
- **星云背景**: 多层渐变叠加的宇宙星云效果
- **恒星闪烁**: 多层闪烁的星星动画
- **星云飘动**: 缓慢移动的星云层
- **ScrollTrigger 滚动**: 卡片向右滑出并淡出

## 内容数据

| 序号 | 标题 | 描述 |
|------|------|------|
| 01 | 恒星摇篮 | 气体与尘埃的密集区域 |
| 02 | 行星状星云 | 恒星晚期的华美绽放 |
| 03 | 发射星云 | 氢气云中的新生光芒 |
| 04 | 暗星云 | 遮挡星光的神秘剪影 |

## 动画参数

| 动画类型 | from | to |
|---------|------|-----|
| 入场动画 | x: '-100%', visibility: 'hidden' | x: 0, visibility: 'visible' |
| ScrollTrigger | 触发滚动滑出 | x: 200, opacity: 0 |

## 技术栈

- Vue 3 Composition API
- TypeScript
- GSAP + ScrollTrigger
- SCSS
