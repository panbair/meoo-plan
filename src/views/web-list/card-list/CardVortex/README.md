# CardVortex - 能量漩涡卡片

## 效果说明

模拟能量漩涡效果，带有旋转的漩涡核心、能量波扩散和粒子轨道动画。

## 动画效果

| 动画类型 | 描述 |
|---------|------|
| 漩涡旋转 | 背景8臂漩涡持续旋转 |
| 能量核心 | 中心脉冲发光效果 |
| 能量波 | 6层能量波向外扩散 |
| 卡片旋转 | 滚动时卡片旋转入场 |
| 漩涡粒子 | 12个粒子沿轨道旋转 |
| 能量环 | 4层能量环向外扩散 |

## 技术实现

- **CSS Transform**: 漩涡旋转效果
- **GSAP ScrollTrigger**: 滚动控制动画
- **极坐标**: 粒子轨道运动
