# 创新卡片组件库 (Card Components)

## 概述

这是一个基于 Vue 3 + GSAP + ScrollTrigger 的创新卡片组件库，包含多种独特的动画效果和交互体验。

## 技术栈

| 技术 | 版本 | 用途 |
|------|------|------|
| Vue 3 | ^3.3 | 核心框架 |
| TypeScript | ^5.0 | 类型系统 |
| GSAP | ^3.12 | 动画引擎 |
| ScrollTrigger | ^3.12 | 滚动动画插件 |
| SCSS | ^1.6 | 样式预处理 |

## 通用动画模式

### 入场动画

```typescript
const initEntryAnimation = () => {
  const tween = gsap.fromTo(
    elements,
    { y: 80, opacity: 0 },
    {
      y: 0,
      opacity: 1,
      duration: 0.8,
      ease: 'back.out(1.7)',
      stagger: 0.15
    }
  )
  cleanupFns.push(() => tween.kill())
}
```

### 滚动动画

```typescript
const initScrollAnimation = () => {
  const tween = gsap.to(element, {
    y: -100,
    scrollTrigger: {
      trigger: element,
      start: 'top 80%',
      end: 'top 20%',
      scrub: 1.5
    }
  })
  cleanupFns.push(() => tween.kill())
}
```

### 动画清理

```typescript
type TweenCleanup = () => void
let cleanupFns: TweenCleanup[] = []

// 保存清理函数
cleanupFns.push(() => tween.kill())

// 组件卸载时清理
onUnmounted(() => {
  ScrollTrigger.getAll().forEach(t => t.kill())
  cleanupFns.forEach(fn => fn())
  cleanupFns = []
})
```

### 模板引用 (Vue 3)

```typescript
const refsMap = ref(new Map<number, HTMLElement>())

const setRef = (el: Element | ComponentPublicInstance | null, index: number) => {
  if (el) {
    const element = el instanceof Element ? el : (el as { $el: Element }).$el
    if (element instanceof HTMLElement) {
      refsMap.value.set(index, element)
    }
  } else {
    refsMap.value.delete(index)
  }
}
```

## 使用方式

### 自动批量注册

在 `index.vue` 中使用 `import.meta.glob` 自动加载所有组件：

```typescript
const modules = import.meta.glob('./card-list/*/[^/]*.vue', { eager: true })

const cardComponents = computed(() => {
  return Object.entries(modules)
    .map(([path, module]) => {
      const match = path.match(/\/card-list\/([^/]+)\/[^/]+\.vue$/)
      const dirName = match?.[1] || ''
      const name = dirName.replace(/Card/g, '').replace(/([A-Z])/g, ' $1').trim()
      return { name, component: (module as any)?.default }
    })
    .filter(item => item.component !== null)
})
```

### 组件宽高

所有组件统一尺寸：`100vw × 100vh`

## GSAP ScrollTrigger 配置

| 参数 | 示例值 | 说明 |
|------|--------|------|
| `trigger` | `element` | 触发器元素 |
| `start` | `'top 80%'` | 开始位置 |
| `end` | `'top 20%'` | 结束位置 |
| `scrub` | `1.5` | 平滑系数 |

## 注意事项

1. **SSR 兼容**：GSAP 需要客户端执行，确保在 `onMounted` 中初始化
2. **性能优化**：使用 `requestAnimationFrame` 确保 DOM 渲染
3. **内存管理**：组件卸载时务必清理所有动画和引用
4. **类型安全**：使用 `instanceof` 检查替代 `as any`
5. **Vue Scoped**：避免在 GSAP 中使用 CSS 选择器，使用元素引用

## 开发指南

### 添加新组件

1. 在 `card-list/` 下创建新文件夹 `CardYourComponent/`
2. 创建 `CardYourComponent.vue` 组件文件
3. 创建 `CardYourComponent/README.md` 文档
4. 自动注册，无需手动配置

### 组件规范

- 使用 `<script setup lang="ts">` 语法
- 遵循 Vue 3 Composition API 最佳实践
- 实现完整的 TypeScript 类型定义
- 提供入场动画和滚动动画
- 实现组件卸载时的动画清理
- 编写详细的 README 文档
- 宽高统一为 `100vw × 100vh`
