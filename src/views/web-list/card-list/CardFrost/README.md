# CardFrost 凛冬冰霜卡片

凛冬冰霜主题的酷炫卡片组件，元素沿Y轴从右侧翻转展示冰霜的凛冽之美。

## 组件特性

- **flip-right 入场动画**: 沿Y轴从 +100° 向右翻转展示
- **冰晶结构**: 多层几何冰晶造型
- **霜冻蔓延**: 动态生长的冰霜纹理
- **寒风粒子**: 飘散的雪花和冰晶
- **ScrollTrigger 滚动**: 卡片继续向右翻转滑出并淡出

## 内容数据

| 序号 | 标题 | 描述 |
|------|------|------|
| 01 | 冰晶凝结 | 水汽遇冷直接凝华成冰晶 |
| 02 | 霜花绽放 | 冰晶在表面形成的精美图案 |
| 03 | 冰封万物 | 低温使生物进入休眠状态 |
| 04 | 极光映雪 | 寒夜中绚丽的天幕之光 |

## 动画参数

| 动画类型 | from | to |
|---------|------|-----|
| 入场动画 | rotationY: 100, opacity: 0 | rotationY: 0, opacity: 1 |
| ScrollTrigger | 触发滚动翻转滑出 | rotationY: -90, x: 200, opacity: 0 |

## 技术栈

- Vue 3 Composition API
- TypeScript
- GSAP + ScrollTrigger
- SCSS
