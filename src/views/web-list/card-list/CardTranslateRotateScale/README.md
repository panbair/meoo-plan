# CardTranslateRotateScale 边缘滑入卡片

> 位移+旋转+缩放组合的边缘滑入动画

## 特性

- ←→ **边缘滑入** - 卡片从左/右/底部边缘滑入
- ⬅️ **边缘滑出** - 滚动时卡片向边缘滑出
- 🎨 **3D透视** - 增强的3D变换和模糊效果
- ⚡ **弹性缓动** - 使用back.out和elastic.out缓动函数

## 技术栈

- Vue 3 Composition API
- TypeScript
- GSAP + ScrollTrigger
- SCSS

## 动画效果

### 入场动画

| 卡片 | 边缘方向 | 位移 | 旋转 | 倾斜 | 缩放 | 缓动 |
|------|---------|------|------|------|------|------|
| 卡片1 | 左侧 | -500px | -30° | skewY: -20° | 0.5→1 | back.out |
| 卡片2 | 右侧 | +500px | +30° | skewY: +20° | 0.5→1 | back.out |
| 卡片3 | 底部 | +400px | +25° | skewX: -15° | 0.3→1 | elastic.out |

### 滚动消失动画

| 卡片 | 滑出方向 | 位移 | 旋转 | 倾斜 | 缩放 | 模糊 |
|------|---------|------|------|------|------|------|
| 卡片1 | 左 | -400px | -35° | skewY: -15° | 0.4 | 0→10px |
| 卡片2 | 右 | +400px | +35° | skewY: +15° | 0.4 | 0→10px |
| 卡片3 | 上 | -350px | +30° | skewX: -20° | 0.3 | 0→10px |

### 关键改进
1. **增强旋转效果** - 入场和退出都有旋转动画
2. **skewX/skewY 倾斜** - 增加3D变形效果
3. **模糊增强** - 卡片最高10px模糊
4. **3D透视** - 1500px透视距离增加深度感
5. **transform-style: preserve-3d** - 保持3D变换
6. **will-change优化** - 提升动画性能
