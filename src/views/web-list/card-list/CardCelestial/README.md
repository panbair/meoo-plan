# CardCelestial 天体融合

## 主题介绍

融合星云、等离子与波纹效果的天体主题卡片。展现宇宙能量的交汇与扩散。

## 视觉效果

### 核心元素
| 元素 | 描述 |
|------|------|
| 星云层 | 三层渐变星云，缓慢飘动 |
| 星星场 | 多层闪烁星星，不同大小和频率 |
| 能量环 | 旋转的发光能量环 |
| 波纹场 | 从中心扩散的同心波纹 |
| 粒子雨 | 垂直下落的能量粒子 |
| 脉动核 | 中心核心的呼吸脉动 |

### 色彩方案
- 主色：`#7c3aed` (紫) → `#06b6d4` (青)
- 背景：`#030712` (深空黑)
- 点缀：`#f472b6` (粉) / `#fbbf24` (金)

## 动画效果

### 入场动画：roll-up
```javascript
'roll-up': {
  from: { y: '100%', visibility: 'hidden', rotation: -15 },
  to: { y: 0, visibility: 'visible', rotation: 0 }
}
```

### ScrollTrigger 配置
| 参数 | 值 | 说明 |
|------|-----|------|
| trigger | parent | 触发器元素 |
| start | 'top 80%' | 元素顶部到达视口80%时开始 |
| end | 'top 20%' | 元素顶部到达视口20%时结束 |
| scrub | 1.5 | 平滑系数 |

### 持续动画
- 星云层缓慢漂移
- 星星闪烁
- 能量环旋转
- 波纹从中心扩散
- 粒子下落
- 核心脉动
