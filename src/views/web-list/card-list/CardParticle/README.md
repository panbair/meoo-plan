# CardParticle - 粒子网络卡片

## 效果说明

模拟粒子网络连接效果，卡片带有原子轨道动画，背景有动态粒子漂浮和连线效果。

## 动画效果

| 动画类型 | 描述 |
|---------|------|
| Canvas粒子 | 100个粒子随机漂浮并连线 |
| 标题入场 | 弹性缩放入场动画 |
| 卡片飞入 | 从屏幕各位置向中心聚集 |
| 3D旋转 | 卡片Y轴360度旋转 |
| 原子轨道 | 迷你粒子绕中心旋转 |
| 滚动粒子 | 底部粒子向上飘散 |

## 技术实现

- **Canvas API**: 实时粒子系统渲染
- **GSAP ScrollTrigger**: 滚动控制卡片聚集
- **CSS Animation**: 轨道持续旋转
