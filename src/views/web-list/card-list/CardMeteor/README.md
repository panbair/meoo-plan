# CardMeteor 流星雨卡片

> 酷炫的流星雨动画效果卡片

## 特性

- 🌠 **流星动画** - 持续循环的流星划过效果
- ⭐ **星空背景** - 闪烁的星星背景
- 🔥 **渐变光效** - 火流星、冰流星、星尘流三种效果
- 💫 **3D透视** - 增强的3D变换和模糊效果

## 技术栈

- Vue 3 Composition API
- TypeScript
- GSAP + ScrollTrigger
- SCSS

## 动画效果

### 入场动画

| 元素 | 位移 | 旋转 | 缩放 | 模糊 | 缓动 |
|------|------|------|------|------|------|
| 徽章 | - | -720°→0° | 0→1 | 20px→0 | back.out |
| 标题 | y: 60→0 | - | 0.5→1 | 15px→0 | power3.out |
| 副标题 | y: 40→0 | - | - | - | power2.out |

### 卡片入场

| 卡片 | 方向 | 位移 | 旋转 | 倾斜 | 缩放 | 缓动 |
|------|------|------|------|------|------|------|
| 火流星 | 左侧 | (-80, 200) | -15° | skewX: -10° | 0.6→1 | back.out |
| 冰流星 | 中间 | (0, 200) | 10° | - | 0.5→1 | elastic.out |
| 星尘流 | 右侧 | (80, 200) | 15° | skewX: 10° | 0.6→1 | back.out |

### 滚动消失动画

| 卡片 | 滑出方向 | 位移 | 旋转 | 倾斜 | 缩放 | 模糊 |
|------|---------|------|------|------|------|------|
| 火流星 | 左上 | (-300, -250) | -45° | skewX: -20° | 0.3 | 0→12px |
| 冰流星 | 上 | (0, -300) | 0° | - | 0.2 | 0→12px |
| 星尘流 | 右上 | (300, -250) | 45° | skewX: 20° | 0.3 | 0→12px |

### 流星动画
- 8颗流星持续循环划过
- 角度: 35°-55° 随机
- 长度: 80-200px 随机
- 轨迹: 从左向右下划过
- 发光效果: 白色发光和橙色光晕

### 关键改进
1. **流星雨效果** - 持续循环的流星动画
2. **星空背景** - CSS渐变创建的闪烁星星
3. **增强旋转** - 入场和退出都有旋转动画
4. **模糊增强** - 卡片最高12px模糊
5. **3D透视** - 1200px透视距离
6. **will-change优化** - 提升动画性能
7. **cleanup清理** - 组件卸载时清理所有动画
