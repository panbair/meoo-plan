# CardGlitch - 故障艺术卡片

## 效果说明

模拟数字故障艺术效果，带有RGB分离、扫描线、噪点等视觉效果。

## 动画效果

| 动画类型 | 描述 |
|---------|------|
| RGB分离 | 红绿蓝三层分离抖动 |
| 扫描线 | 全屏扫描线动画 |
| 标题故障 | 文字伪元素故障效果 |
| 卡片故障 | 滚动触发卡片故障动画 |
| 噪点覆盖 | SVG噪点纹理叠加 |
| 故障条 | 底部彩色故障条动画 |

## 技术实现

- **CSS伪元素**: 标题RGB分离效果
- **SVG噪点**: 程序化噪点纹理
- **GSAP**: 滚动触发动画

## 使用方式

```vue
<CardGlitch />
```
