# CardSlideEdge 边缘滑入卡片

> 从两边边缘滑入的动画卡片

## 特性

- ←→ **从边缘滑入** - 卡片从左侧或右侧边缘滑入带旋转和倾斜
- ⬅️ **滚动边缘退出** - 滚动时向边缘退出带旋转、倾斜和模糊
- 🎨 **3D透视效果** - 增强的3D变换和模糊效果
- ⚡ **弹性缓动** - 使用back.out和elastic.out缓动函数

## 技术栈

- Vue 3 Composition API
- TypeScript
- GSAP + ScrollTrigger
- SCSS

## 动画效果

### 入场动画
- 卡片1: 从左侧边缘滑入 + 逆时针旋转25° + skewX倾斜15°
- 卡片2: 从右侧边缘滑入 + 顺时针旋转25° + skewX倾斜15°
- 卡片3: 从底部边缘滑入 + 旋转15° + skewY倾斜10° + 弹性缩放

### 滚动动画
- 卡片1: 向左滑出 + 旋转-20° + skewX-12° + 模糊0-8px + 缩放0.6
- 卡片2: 向右滑出 + 旋转20° + skewX12° + 模糊0-8px + 缩放0.6
- 卡片3: 向上滑出 + 旋转15° + skewX8° + 模糊0-8px + 缩放0.5

### 关键改进
1. **增强旋转效果** - 入场和退出都有旋转动画
2. **skewX/skewY 倾斜** - 增加3D变形效果
3. **模糊增强** - 卡片最高8px模糊
4. **独立触发** - 每张卡片有专属ScrollTrigger实例
5. **3D透视** - 1200px透视距离增加深度感
