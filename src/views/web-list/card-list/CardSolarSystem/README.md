# CardSolarSystem - 太阳系卡片

## 效果说明

模拟太阳系行星轨道运行效果，行星卡片带有自转、公转和表面动画。

## 动画效果

| 动画类型 | 描述 |
|---------|------|
| 太阳脉冲 | 太阳核心持续脉动发光 |
| 日冕旋转 | 太阳外围旋转光晕 |
| 行星轨道 | 5颗行星不同大小轨道 |
| 公转动画 | 行星沿轨道运行 |
| 自转效果 | 卡片旋转动画 |
| 流星划过 | 随机流星划过夜空 |
| 星空闪烁 | 200颗星星随机闪烁 |

## 技术实现

- **CSS Animation**: 轨道和公转动画
- **GSAP ScrollTrigger**: 滚动控制行星入场
- **动态轨道**: 根据行星数据生成轨道

## 使用方式

```vue
<CardSolarSystem />
```
