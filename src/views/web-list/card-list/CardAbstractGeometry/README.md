# CardAbstractGeometry - 抽象几何

## 主题描述

抽象几何图形组合的现代艺术主题。各种几何形状在空间中旋转、缩放、组合，形成动态的艺术画面。

## 视觉风格

- **主色调**: 深黑 + 多彩几何
- **辅助色**: 红、黄、蓝、绿几何色块
- **特效**: 几何旋转、形状组合、网格背景

## 动画设计

### 入场动画：slide-up（向上滑入）

| 时间 | 动画 | 效果 |
|------|------|------|
| 0s | 背景网格 | 淡入 |
| 0.3s | 几何形状入场 | 从不同角度旋转进入 |
| 0.8s | 几何缩放组合 | 缩放到最终位置 |
| 1.2s | 标题组滑入 | Badge→标题→副标题 |
| 1.7s | **卡片错落入场** | 从下方依次出现 |
| 1.9s | 图标旋转弹入 | rotation |
| 2.0s | 标题淡入 | stagger |

### 卡片错落入场

```javascript
.fromTo('.geo-card',
  (index) => ({
    y: 80,
    opacity: 0,
    rotation: index % 2 === 0 ? -4 : 4,
    scale: 0.85
  }),
  {
    stagger: 0.12,
    ease: 'back.out(1.2)'
  }
)
```
