# CardCrystal 水晶卡片

水晶主题的酷炫卡片组件，元素沿X轴翻转展示晶体的折射之美。

## 组件特性

- **flip-up 入场动画**: 沿X轴从 -100° 翻转展示
- **水晶棱镜**: 多层折射光效的水晶结构
- **棱面光效**: 动态折射的彩虹光线
- **闪烁碎片**: 漂浮的水晶碎片动画
- **ScrollTrigger 滚动**: 卡片继续翻转滑出并淡出

## 内容数据

| 序号 | 标题 | 描述 |
|------|------|------|
| 01 | 晶核形成 | 过饱和溶液中的晶种萌发 |
| 02 | 棱面生长 | 原子排列成规整的晶格结构 |
| 03 | 光线折射 | 棱面将白光分解为七彩光谱 |
| 04 | 完美晶体 | 缺陷最少的天然矿物珍品 |

## 动画参数

| 动画类型 | from | to |
|---------|------|-----|
| 入场动画 | rotationX: -100, opacity: 0 | rotationX: 0, opacity: 1 |
| ScrollTrigger | 触发滚动翻转滑出 | rotationX: 90, y: 200, opacity: 0 |

## 技术栈

- Vue 3 Composition API
- TypeScript
- GSAP + ScrollTrigger
- SCSS
