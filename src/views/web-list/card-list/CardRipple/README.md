# CardRipple 波纹扩散卡片

> 酷炫的波纹扩散动画效果卡片

## 特性

- 🌊 **波纹效果** - 背景持续波纹扩散动画
- 💧 **水滴涟漪** - hover时卡片的波纹脉冲效果
- 🔵 **青蓝渐变** - 科技感十足的青蓝色调
- 💫 **3D透视** - 增强的3D变换和模糊效果

## 技术栈

- Vue 3 Composition API
- TypeScript
- GSAP + ScrollTrigger
- SCSS

## 动画效果

### 入场动画 (fade-down)

| 元素 | 位移 | 旋转 | 缩放 | 模糊 | 缓动 |
|------|------|------|------|------|------|
| 徽章 | y: -100→0 | -180°→0° | 0→1 | 15px→0 | back.out |
| 标题 | y: -80→0 | - | 0.7→1 | 10px→0 | power3.out |
| 副标题 | y: -50→0 | - | - | - | power2.out |

### 卡片入场

| 卡片 | 位移 | 旋转 | 倾斜 | 缩放 | 缓动 |
|------|------|------|------|------|------|
| 同心波纹 | y: -150 | -8° | skewX: -5° | 0.5→1 | back.out |
| 水滴涟漪 | y: -180 | 5° | - | 0.4→1 | elastic.out |
| 能量脉冲 | y: -150 | 8° | skewX: 5° | 0.5→1 | back.out |

### 滚动消失动画

| 卡片 | 滑出方向 | 位移 | 旋转 | 倾斜 | 缩放 | 模糊 |
|------|---------|------|------|------|------|------|
| 同心波纹 | 左下 | (-250, 200) | -30° | skewX: -15° | 0.2 | 0→12px |
| 水滴涟漪 | 下 | (0, 250) | 0° | - | 0.15 | 0→12px |
| 能量脉冲 | 右下 | (250, 200) | 30° | skewX: 15° | 0.2 | 0→12px |

### 背景波纹动画
- 5个波纹持续扩散
- 周期: 4秒循环
- 大小: 从100px扩散到800px
- 透明度: 0.8→0

### 关键改进
1. **fade-down效果** - 从上方淡入的入场动画
2. **波纹背景** - 动态扩散的背景波纹
3. **卡片脉冲** - hover时卡片的波纹动画
4. **模糊增强** - 卡片最高12px模糊
5. **3D透视** - 1200px透视距离
6. **will-change优化** - 提升动画性能
7. **cleanup清理** - 组件卸载时清理所有动画
