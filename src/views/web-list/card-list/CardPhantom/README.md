# CardPhantom 幻影卡片

幻影主题的酷炫卡片组件，元素沿X轴向下翻转展示幽灵般的视觉效果。

## 组件特性

- **flip-down 入场动画**: 沿X轴从 +100° 向下翻转展示
- **幽灵粒子**: 飘散的幻影粒子效果
- **烟雾层**: 动态流动的烟雾背景
- **幻眼神灯**: 脉动的神秘灯光
- **ScrollTrigger 滚动**: 卡片继续向下翻转滑出并淡出

## 动画参数

| 动画类型 | from | to |
|---------|------|-----|
| 入场动画 | rotationX: 100, opacity: 0 | rotationX: 0, opacity: 1 |
| ScrollTrigger | 触发滚动翻转滑出 | rotationX: -90, y: 200, opacity: 0 |

## 技术栈

- Vue 3 Composition API
- TypeScript
- GSAP + ScrollTrigger
- SCSS
