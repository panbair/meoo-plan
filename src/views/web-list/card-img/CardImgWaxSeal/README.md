# CardImgWaxSeal - 蜡封启缄 · 火漆圆章裁切展开

## 效果描述

图片如同一封以深红火漆封缄的密函，初始被一枚浑圆的蜡封印严密遮蔽。Timeline 驱动蜡层从圆心向外消融，裁切边缘带有有机的微颤。图片在蜡层消融中渐次明亮，四角残余蜡珠短暂闪现，最终画面完整呈现于亚麻纤维纹理的纸面之上。

## 核心动画

- **圆形裁切**：`clip-path: circle()` 从 8% 扩张至 100%
- **有机融边**：SVG feTurbulence 滤镜产生不规则边缘
- **亮度联动**：brightness 从 0.3 升至 1.0
- **蜡珠飞溅**：四角蜡珠显现后渐隐
- **亚麻纹理**：完成后纹理渐显

## 技术实现

- Vue 3 Composition API
- GSAP Timeline + ScrollTrigger
- SVG feTurbulence 边缘纹理
- SCSS 样式

## 视觉效果

- 深红褐色火漆封缄
- 暖金棕色调
- 亚麻纤维底纹
- 火漆家族纹章暗示
