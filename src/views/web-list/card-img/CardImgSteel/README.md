# CardImgSteel - 冷轧钢板·精密导轨滑入

## 概述

CardImgSteel 组件实现"冷轧钢板·精密导轨滑入"效果，模拟精密金属构件沿导轨滑入的动作，配合静摩擦启程、骤停微颤和冰蓝色刀痕光泽，呈现工业精密仪器的冷峻质感。

## 效果特点

### 动画机制

- **侧翻入场**：translateX 从 -120% 滑入，rotateY 从 3deg 扶正
- **静摩擦效果**：前 20% 极慢启动
- **骤停回弹**：88% 后产生 -4px 回弹
- **高频微颤**：通过 setInterval 实现定位震颤
- **阴影变化**：从左侧斜长投影渐变底部均匀
- **冰蓝刀痕**：落位后骤升→回落→低频呼吸

### 视觉风格

- **色调**：深蓝黑色背景配冰蓝高光
- **装饰**：导轨线条、金属面板
- **氛围**：工业精密、冷峻金属

## 技术实现

### ScrollTrigger 配置

| 参数 | 值 | 说明 |
|------|-----|------|
| trigger | parent | 触发器元素 |
| start | top 85% | 标准触发 |
| end | bottom 20% | 元素离开前结束 |
| scrub | 1.5 | 平滑滚动系数 |

### 核心计算

```typescript
// 静摩擦效果
if (p < 0.2) {
  moveP = Math.pow(p / 0.2, 3) * 0.2
} else if (p < 0.8) {
  moveP = 0.2 + (p - 0.2) * 0.75
}

// 回弹效果
if (p >= 0.88) {
  bounceX = -4 * Math.sin(bounceP / 0.4 * Math.PI) * (1 - bounceP)
}

// 刀痕光泽三段
if (cutP < 0.3) {
  cutOpacity = 0.3 + 0.5 * (cutP / 0.3) // 骤升
} else if (cutP < 0.5) {
  cutOpacity = 0.8 - 0.5 * ((cutP - 0.3) / 0.2) // 回落
} else {
  cutOpacity = 0.2 + 0.1 * Math.sin(Date.now() / 800) // 呼吸
}
```
