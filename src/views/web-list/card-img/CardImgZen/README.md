# CardImgZen · 凝滞空间 · 禅意和风呼吸动效

## 概述

CardImgZen 组件实现"凝滞空间·禅意和风呼吸动效"效果，图片如东方禅意静物陈列于和纸窗棂之间。通过 GSAP Timeline 驱动多层呼吸动画系统，模拟柔光脉动、尘埃漂浮、暖光流转效果，营造沉静深邃的日式冥想空间。

## 动画效果

### 入场动画

| 元素 | 动画 | 时长 | 缓动 |
|------|------|------|------|
| 外层光晕 | 缩放淡入 | 1.5s | power2.out |
| 主容器 | 缩放淡入 | 1.2s | power2.out |
| 脉冲光晕 | 缩放淡入 | 1s | power2.out |
| 标题 | 淡入 | 0.8s | power2.out |

### 循环呼吸动画

| 动画 | 范围 | 周期 | 效果 |
|------|------|------|------|
| 容器缩放 | 1.0 ~ 1.015 | 8s | 和缓呼吸 |
| 光晕缩放 | 1.0 ~ 1.08 | 8s | 柔光脉动 |
| 脉冲光晕 | scale + opacity | 4s | 呼吸灯效 |
| 位移漂移 | X: ±2px, Y: ±1.5px | 12s | 凝视漂移 |
| 暖光层 | opacity 0.2 ~ 0.4 | 6s | 光影流转 |
| 暗角涨落 | opacity 0.4 ~ 0.6 | 8s | 呼吸节奏 |

### 环境动画

| 动画 | 范围 | 周期 | 效果 |
|------|------|------|------|
| 窗棂投影 | scaleX 1.0 ~ 1.05 | 6s | 日影移动 |
| 光斑移动 | 随机轨迹 | 65s | 阳光游移 |
| 柔光层 | opacity 脉动 | 10s | 氛围光效 |

### 粒子系统

| 粒子类型 | 数量 | 动画 | 说明 |
|----------|------|------|------|
| 尘埃粒子 | 25 | 随机飘浮 | 和风尘埃 |

## 视觉层次

1. **背景层**：渐变 + 柔光层 + 和纸纹理
2. **粒子层**：漂浮尘埃粒子
3. **窗棂层**：横纵线条 + 投影 + 光斑
4. **框架层**：外层光晕 + 脉冲光晕
5. **图片层**：暖光叠加 + 和纸纹理
6. **边框层**：四角装饰 + 内层边框

## 技术实现

### 色彩系统

- 主背景：米白色 `#f5f2ed` ~ `#ebe6df`
- 框架色：`rgba(180, 170, 160, 0.15)`
- 光晕色：`rgba(255, 250, 240, 0.3)`
- 暖光色：`rgba(255, 240, 220, 0.35)`
- 文字色：深褐色 `#4a4238`

### 动画周期表

| 动画组 | 周期 | 错相策略 |
|--------|------|----------|
| 缩放呼吸 | 8s | 基准 |
| 脉冲光晕 | 4s | 独立 |
| 暖光层 | 6s | 独立 |
| 位移漂移 | 12s | 独立 |
| 光斑移动 | 65s | 复合轨迹 |
