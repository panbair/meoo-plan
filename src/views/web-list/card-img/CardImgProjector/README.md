# CardImgProjector 投影仪对焦·色差补偿缩放

## 效果概述
黑暗空间中虚拟投影仪向幕布投射一幅失焦的图片，初始画面模糊放大并伴有红蓝青三色分离的色散光斑。Timeline 驱动图片收缩与清晰化，三色分离逐渐叠合为纯净全彩。

## 技术实现

### 核心特性
| 特性 | 描述 |
|------|------|
| 色差分离 | 红、蓝、青三色层水平偏移叠加 |
| 对焦缩放 | scale 从 1.5 缩至 1.0 |
| 模糊渐锐 | blur 从 12px 降至 0 |
| 光束微尘 | 飘浮的灰尘颗粒效果 |
| 投影暗角 | 边框遮挡效果 |
| 镜头反光 | 细长光线掠过 |

### ScrollTrigger 配置
| 参数 | 值 | 说明 |
|------|-----|------|
| trigger | container | 触发器元素 |
| start | top 100% | 元素完全进入视口时开始 |
| end | bottom -100% | 元素完全离开视口时结束 |
| scrub | 1.5 | 平滑系数 |

### 动画阶段
| 阶段 | 进度 | 效果 |
|------|------|------|
| 投射 | 0-30% | 模糊放大 → 色差分离 |
| 对焦 | 30-70% | 缩放收拢 → 模糊消失 → 色差叠合 |
| 稳定 | 70-85% | 光束闪烁 → 微尘飘浮 |
| 收尾 | 85-100% | 暗角固定 → 镜头反光 |

### 色差原理
```
[初始状态]              [对焦中]              [最终]
R →   ← B              R → ← B             RGB
   ← C →                 ← C →              完全叠合
  三色分离              逐渐靠拢
```
