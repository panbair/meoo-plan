# CardImgVelvetDisplay - 丝绒展柜 · 聚焦视差

## 效果描述

深色丝绒展柜内部，前景艺术品图片悬挂于柔和顶光之下。滚动时，展柜背景丝绒纹理极缓向下滑移，图片轻微向前放大，同时伴有一次极其克制的提亮，如同展品被无形灯光缓缓聚焦。

## 核心动画

- **背景视差**：丝绒纹理 translateY 系数 0.12
- **前景视差**：图片 translateY 系数 0.06
- **聚焦放大**：scale 从 1.0 → 1.03 → 1.015
- **亮度提升**：brightness 从 0.9 → 1.0 → 0.98
- **悬浮阴影**：投影 opacity 联动加深

## 技术实现

- Vue 3 Composition API
- GSAP Timeline + ScrollTrigger
- CSS 重复渐变模拟丝绒纹理
- SCSS 样式

## 视觉效果

- 深色丝绒背景
- 丝绒纹理肌理
- 柔和顶光渐变
- 悬浮阴影
- 暖白艺术标题
