# CardImgDeepPulse - 深空脉动·全屏背景呼吸

## 特效概述

全屏背景图置于视口底层，以极缓慢的非对称节奏持续缩放与明暗交替。缩放范围控制在极小幅度内，不被肉眼刻意察觉，却让整个页面产生如同空间在安然呼吸的沉浸感。

## 动画亮点

- `scale` 在 1 至 1.04 之间以 6-8 秒周期正弦往复，缓动 `power1.inOut` 保持极度柔和
- `brightness` 与 `scale` 错相 0.5 周期联调，缩放扩大时微微增亮，收缩时微暗
- 色温同步以 ±3 度幅度极缓偏移，配合四角渐隐暗角周期性强弱变化
- 无需任何触发，页面加载即启动，动画永远匀速平缓

## 技术实现

| 技术 | 说明 |
|------|------|
| Vue 3 | 组件框架 |
| GSAP | 动画引擎 |
| ScrollTrigger | 滚动触发（视差效果） |
| SCSS | 样式预处理 |

## 使用场景

首页横幅、落地页背景、沉浸式展示等需要静谧氛围的场景。
