# CardImgAperture 虹膜光圈·光学开镜视差

## 效果概述
画面如同透过闭合的相机光圈观察，黑色叶片以机械旋转方式层层张开，图片从中心圆域渐显扩散，伴随缩放和模糊变化。

## 技术实现

### 核心特性
| 特性 | 描述 |
|------|------|
| 光圈叶片 | CSS clip-path 多边形，8片叶片旋转张开 |
| 中心渐显 | 图片从中心圆域向四周扩散显示 |
| 缩放补偿 | 图片 scale 从 1.8 缩至 1.0 |
| 景深分层 | 前景/主体/背景差异化开镜速度 |
| 镜头偏移 | 鼠标移动产生光学暗角效果 |

### ScrollTrigger 配置
| 参数 | 值 | 说明 |
|------|-----|------|
| trigger | container | 触发器元素 |
| start | top 100% | 元素完全进入视口时开始 |
| end | bottom -100% | 元素完全离开视口时结束 |
| scrub | 1.5 | 平滑系数 |

### 动画阶段
| 阶段 | 进度 | 效果 |
|------|------|------|
| 开镜 | 0-40% | 光圈叶片旋转张开，图片渐显 |
| 清晰 | 40-60% | 模糊消失，缩放完成 |
| 互动 | 60-80% | 鼠标偏移产生暗角效果 |
| 关镜 | 80-100% | 光圈闭合，图片消失 |

### 叶片动画
```
[闭合状态]              [张开过程]              [完全打开]
    ╳                      ╱╲                     ┌─┐
   ╱╲        →           ╱  ╲         →         │○│
  ╱  ╲                   ○────○                   └─┘
叶片旋转           中心圆域扩大            叶片完全张开
```
