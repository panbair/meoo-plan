# CardImgDeepSea · 深海光隙

> 深海蓝暗背景中，数道从海面穿透而下的光束斜贯画面，光柱中悬浮着亿万微型浮游生物与有机碎屑。结合真实深海摄影图片，营造沉浸式海底体验。

## 效果预览

- **主题**: 深海光隙 · 水下光束背景视差
- **图片**: 深海摄影（Unsplash）
- **配色**: 深海蓝、幽蓝、青白
- **动画**: 五层视差、光束折射、粒子漂浮、气泡上升

## 层级架构

| 层级 | 元素 | 速度系数 | 效果 |
|------|------|---------|------|
| 背景 | Unsplash深海图片 | 0.05 | 缓慢缩放+平移，鼠标跟随 |
| 远景 | 深海蓝幕 | 0.02 | 几乎静止 |
| 次远 | 光柱层 | 0.08 | 缓慢移动，角度随滚动微幅摆动 |
| 中景 | 浮游粒子 | 0.2 | 150个光点缓慢上浮 |
| 近景 | 焦散光斑 | 0.35 | 最快移动，水平摇摆 |

## 动画特效

### 视差滚动
- `ScrollTrigger` 驱动多层差异化速度垂直漂移
- 背景图片缩放 1.1→1.2
- 深海幕布几乎静止，光斑层最快且水平摇摆

### 光束折射
- 三道光柱使用 `mix-blend-mode: screen`
- GSAP 控制旋转角度随滚动微幅变化
- 模拟海面波浪折射下的光线偏移

### 浮游粒子
- Canvas 绘制 150 个微型光点
- 赋予上浮速度与随机水平布朗运动
- 持续循环：超出边界时重置到下方

### 焦散纹理
- SVG `feTurbulence` 滤镜生成动态噪点
- 配合高斯位移实现水底光纹流动
- 四块光斑以不同节奏缓慢漂移

### 鼠标交互
- 光柱层跟随鼠标产生偏移
- 焦散光斑响应更明显的径向移动
- 背景图片轻微跟随鼠标移动

### 色调呼吸
- 整体色调随滚动在深海蓝与幽绿之间渐变

## 技术栈

- Vue 3 Composition API
- TypeScript
- GSAP + ScrollTrigger
- SCSS
- Canvas 2D
- Unsplash 深海图片

## 注意事项

- 组件占满视口，外部需控制滚动区域
- 图片加载需要网络支持
- 建议在深色背景页面使用效果更佳
