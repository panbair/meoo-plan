# CardImgPearl - 柔焦轻旋 · 珍珠光泽溶解

> 珍珠缎面肖像效果，晨雾般柔和的朦胧美

## 特性

- **晨雾散去**：初始高斯模糊如晨雾轻掩，缓慢蒸发消融
- **轻柔旋转**：逆时针 2° 微偏，如沉睡中的肖像
- **上浮苏醒**：画面在清晰化时同步向上浮升 10px
- **惯性过冲**：回正时产生 0.4° 过冲，elastic.out 回弹
- **珍珠虹彩**：母贝光泽从边缘蔓延后优雅消退
- **缎面光泽**：边缘柔和光晕闪现
- **晨雾粒子**：漂浮的雾粒随动画飘散

## 动画参数

| 参数 | 值 | 说明 |
|------|------|------|
| ScrollTrigger start | `top 100%` | 元素顶部到达视口时开始 |
| ScrollTrigger end | `bottom 20%` | 元素顶部到达视口底部 20% 时结束 |
| scrub | `1.5` | 平滑系数 |
| 模糊溶解 | `10px → 0` | 高斯模糊渐变，duration: 2.0s |
| 旋转回正 | `-2° → 0.4° → 0` | 逆时针偏转 + 惯性过冲 |
| 上下浮动 | `10px → 0` | 轻微下沉后上浮 |
| 透明度 | `0.8 → 1` | 朦胧到清晰 |
| 亮度 | `0.85 → 1` | 晨雾光入效果 |
| 缩放 | `1.02 → 1` | 轻微放大后收缩 |
| 缓动 | `sine.inOut` | 全程极度柔和 |
| 过冲回弹 | `elastic.out(1.2, 0.6)` | 珍珠轻颤效果 |

## 动画阶段

1. **晨雾散去** - blur/brightness/opacity/scale 同步渐变
2. **回正旋转** - rotateZ/y 同步回正
3. **惯性过冲** - rotateZ 0.4° 后 elastic 回弹
4. **母贝虹彩** - 边缘光泽蔓延后消退
5. **缎面光晕** - 边缘柔和闪现
6. **雾粒飘散** - 晨雾粒子渐隐
7. **投影显现** - 柔和投影渐入
8. **背景渐入** - 缎面背景浮现

## 设计灵感

如同一枚镶嵌珍珠的缎面肖像，初始覆着柔和的高斯模糊，仿佛凝望被晨雾轻掩。Timeline 驱动无形晨光抚过，模糊如薄露般缓慢蒸发消融。画面在清晰化过程中同步顺时针轻柔回正，并极其克制地向上浮升。回正瞬间，旋转微微越过零点——产生一次仅 0.4° 的惯性过冲，随即如珍珠在丝绸上轻颤般悠悠弹回。最终静定于静谧矜贵的轻奢质感。
