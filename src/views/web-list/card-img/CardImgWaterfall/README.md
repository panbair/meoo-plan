# CardImgWaterfall - 流瀑音阶 · 多图阶梯坠落入场

## 效果描述

一组 9 张图片如同液态金属琴键悬浮于深色虚空，初始栖于视口上方阶梯高台。Timeline 驱动重力将其逐列唤醒，自左向右、从上至下沿 Z 字形阶梯依次坠落。触地瞬间产生双重微弹并迸发冷光粒子，形成连续倾泻的音阶式节奏。

## 核心动画

- **Z字形阶梯**：列间 0.18s 交错，列内 0.06s 间隔
- **power4 坠落**：power4.in 加速坠落
- **双重弹跳**：首次 -12px 回弹 + back.out(1.5) 锁定
- **冷光粒子**：6 枚粒子触地迸发扩散
- **阴影加深**：box-shadow 从无到深
- **亮度齐升**：brightness 0.8 → 1.0

## 技术实现

- Vue 3 Composition API
- GSAP Timeline + ScrollTrigger
- CSS Grid 3x3 布局
- CSS 粒子扩散动画
- SCSS 样式
