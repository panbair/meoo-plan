# CardImgWireless 悬浮磁感·无线充电视差

## 效果概述
深色空灵空间中，产品悬浮于环形无线充电底座上方，电磁感应光弧连接两者。滚动时产生多层次深度视差，鼠标移动产生磁吸牵引，点击触发能量脉冲。

## 技术实现

### 核心特性
| 特性 | 描述 |
|------|------|
| 三层视差 | 底座(back)、产品(body)、光弧(front) 不同 Z 轴深度 |
| 悬浮呼吸 | gsap.to 循环控制 y 轴浮动和 rotate 微摆 |
| 磁吸牵引 | 鼠标位置通过 quickTo 映射为 XY 平面微小偏移 |
| 能量脉冲 | 点击底座触发环形波纹扩散 |

### ScrollTrigger 配置
| 参数 | 值 | 说明 |
|------|-----|------|
| trigger | container | 触发器元素 |
| start | top 100% | 元素完全进入视口时开始 |
| end | bottom -100% | 元素完全离开视口时结束 |
| scrub | 1.5 | 平滑系数 |

### 动画阶段
| 阶段 | 进度 | 效果 |
|------|------|------|
| 入场 | 0-25% | 底座 → 光弧 → 产品 → 能量环 |
| 展示 | 25-75% | 三层视差滚动 + 悬浮呼吸 |
| 离场 | 75-100% | 产品下落 → 光弧断开 → 底座消失 |

### 元素分布
```
[前景层 +150px]     能量脉冲环 (高亮)
    ↓
[中景层 +50px]      产品主体 (手机/耳机)
    ↓      ↕ 电磁光弧连接
[背景层 0px]        无线充电底座 + 旋转能量环
```

### 依赖
- Vue 3.3+
- GSAP 3.12+ (ScrollTrigger, InertiaPlugin)
- SCSS
