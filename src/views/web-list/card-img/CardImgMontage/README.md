# CardImgMontage - 蒙太奇窗格组件

> 多画幅分镜渐显效果，模拟导演监视器墙的分镜布局

## 特性

- **分镜蒙太奇布局**：6 个不规则窗格，分割为不同区域，模拟多台摄影机同时捕获的效果
- **依次对焦动画**：每个窗格从失焦模糊到锐利清晰，配合亮度提升
- **时序跳跃**：第 3 和第 4 个窗格同时启动，模拟导播手动切换的不完美感
- **边缘光晕**：窗格清晰瞬间迸发极细暖白辉光
- **全局锐化过冲**：所有窗格清晰后进行微弱的锐化过冲效果
- **电影暗角**：四周叠加电影取景框暗角效果
- **导播台标记**：模拟监视器墙的编号标识

## 使用

```vue
<template>
  <CardImgMontage 
    image-url="https://images.unsplash.com/photo-1506905925346-21bda4d32df4?w=1920&q=80" 
  />
</template>

<script setup lang="ts">
import CardImgMontage from '@/views/web-list/card-img/CardImgMontage/CardImgMontage.vue'
</script>
```

## Props

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| imageUrl | `string` | `https://images.unsplash.com/photo-1506905925346-21bda4d32df4?w=1920&q=80` | 图片地址 |

## 技术栈

| 技术 | 版本 | 用途 |
|------|------|------|
| Vue 3 | ^3.3 | 核心框架 |
| TypeScript | ^5.0 | 类型系统 |
| GSAP | ^3.12 | 动画引擎 |
| ScrollTrigger | ^3.12 | 滚动动画插件 |
| SCSS | ^1.6 | 样式预处理 |

## 动画参数

| 参数 | 值 | 说明 |
|------|------|------|
| ScrollTrigger start | `top 100%` | 元素顶部到达视口时开始 |
| ScrollTrigger end | `bottom 20%` | 元素顶部到达视口底部 20% 时结束 |
| scrub | `1.5` | 平滑系数 |
| 窗格 stagger | `0.12s` | 相邻窗格启动间隔 |
| 对焦 duration | `0.5s` | 单个窗格对焦时长 |
| 模糊 | `12px → 0` | 失焦到锐利 |
| 亮度 | `0.4 → 1` | 幽暗到标准曝光 |
| 锐化过冲 | `-0.5px` | 短暂锐化后 elastic 回弹 |

## 设计灵感

模拟导演监视器墙上多台摄影机从不同角度同时捕获的分镜画面，每个窗格如同独立的摄影机视角，在导播台控制下依次"对焦"，最后全部窗格同时达到完美清晰，拼合成完整影像，四周残留极淡的电影取景框暗角。
