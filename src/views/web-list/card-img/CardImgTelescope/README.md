# CardImgTelescope - 深空锁定 · 望远镜变焦视差

## 效果描述

背景为深邃星空与遥远星云，前景悬浮一张天文摄影图片，如同望远镜目镜中的被观测天体。滚动时背景星野与前景图片以差异化速度纵向漂移，图片同步发生极其克制的轻微放大。鼠标移动时产生极微的目镜偏移。

## 核心动画

- **背景视差**：星野 translateY 系数 0.15
- **前景视差**：图片 translateY 系数 0.08
- **微量放大**：scale 从 1.0 至 1.04 再回落 1.02
- **目镜暗角**：opacity 联动 0.5 → 0.7 → 0.6
- **鼠标偏移**：quickTo 驱动 ±3px 微幅偏移

## 技术实现

- Vue 3 Composition API
- GSAP Timeline + ScrollTrigger + quickTo
- CSS 渐变模拟星云
- SCSS 样式
