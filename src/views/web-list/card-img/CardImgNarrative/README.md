# CardImgNarrative - 景深叙事 · 三层视差图文剧场

> 三层独立纵深视差，构建沉浸空间阶梯

## 特性

- **三层纵深结构**：远景纹理、中景主图、前景文案
- **差异化视差**：三层以不同速度纵向移动
- **空间阶梯**：初始状态下远景微糊、中景清晰、前景锐利
- **亮度渐变**：中景层随滚动柔和提亮
- **文案入场**：前景文案极简淡入与微位移
- **装饰点缀**：线条与圆点装饰元素

## 动画参数

| 参数 | 值 | 说明 |
|------|------|------|
| ScrollTrigger start | `top 100%` | 元素顶部到达视口时开始 |
| ScrollTrigger end | `bottom 20%` | 元素顶部到达视口底部 20% 时结束 |
| scrub | `1.5` | 平滑系数 |
| 远景速度 | `y: -80px` | 系数 0.08，极缓 |
| 中景速度 | `y: -200px` | 系数 0.2，中速 |
| 前景速度 | `y: -400px` | 系数 0.4，快速 |
| 远景缩放 | `1.0 → 1.015` | 缩放呼吸 |
| 中景亮度 | `0.85 → 1.0` | 镜头推进光线渐入 |
| 文案淡入 | `y: 15px → 0` | 极简入场 |

## 三层结构

### 远景层 (z-index: 1)
- 抽象纹理背景
- 柔和渐变光晕
- 极缓移动：`y: -80px`
- 缩放呼吸：`scale: 1.0 → 1.015`
- 轻雾模糊：`blur(1px)`

### 中景层 (z-index: 2)
- 主图画面
- 中速移动：`y: -200px`
- 亮度渐变：`0.85 → 1.0`
- 柔和投影

### 前景层 (z-index: 3)
- 标题文案
- 快速移动：`y: -400px`
- 淡入入场
- 装饰元素

## 设计灵感

图文板块被拆分为三个独立纵深图层，各自悬浮于不同的 Z 轴深度。远景微糊如水中暗流，中景清晰稳健推进，前景锐利轻盈滑过。三层以严格差异化的速度纵向移动，图文板块产生镜头在空间中缓慢推进的沉浸纵深。仅凭速度差与属性联动的精确编排，呈现冷静而高级的空间纵深叙事。
