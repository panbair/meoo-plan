# CardImgPrism · 棱镜校准

光谱合像交错入场效果组件，图片被拆解为RGB三色通道，依次聚拢融合为全彩。

## 效果预览

**动画效果**: 棱镜校准 · 光谱合像交错入场

## 核心特性

### 1. RGB 色层分离
- 红层 `x: +8px`
- 绿层 `x: 0`
- 蓝层 `x: -8px`
- 各层带模糊和饱和度

### 2. 光谱校准动画
- 色层从分离向中心聚拢
- `translateX` 归零
- `filter: blur(8px)` → `blur(0)`
- `power3.out` 缓动

### 3. 光束穿透效果
- 中心白色光束从上向下
- `scaleX` 从 0 到 1
- 快速闪现后消失

### 4. 彩虹镶边爆发
- 合光瞬间边框爆发
- 七色彩虹渐变
- `scale` 从 0.8 到 2 迅速消失

### 5. 色层融合
- RGB 层最终 opacity 归零
- 图片显现为全彩

### 6. 彩虹进度条
- 底部七色渐变进度
- 动态计数显示

## 动画时序

```
0.0s  - 头部入场
0.0s  - 进度条开始
0.2s  - 第1张：色层聚拢
0.3s  - 光束穿透
0.5s  - 彩虹镶边爆发
0.5s  - 第2张：色层聚拢
0.7s  - 第3张：色层聚拢
0.9s  - 第4张：色层聚拢
1.2s  - 色层融合消失
```

## 技术规范

- Vue 3 Composition API + TypeScript

- RGB 色层分离融合

- SCSS 样式预处理
- 完整清理逻辑
