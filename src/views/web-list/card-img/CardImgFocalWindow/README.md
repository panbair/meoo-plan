# CardImgFocalWindow - 聚焦视窗 · 焦距驱动错位视差

> 三层景深焦点传递，精密镜头的光学叙事

## 特性

- **三层景深结构**：背景肌理、中景主图、前景文案
- **焦点传递**：从前景到中景到背景的焦点切换
- **非线性速度差**：中段速度差被刻意拉大再收拢
- **扫描光带**：从下而上扫过画面的横向光带
- **色温柔和变化**：冷调与暖白之间的微妙过渡
- **对焦环装饰**：模拟镜头对焦环旋转

## 动画参数

| 参数 | 值 | 说明 |
|------|------|------|
| ScrollTrigger start | `top 100%` | 元素顶部到达视口时开始 |
| ScrollTrigger end | `bottom 20%` | 元素顶部到达视口底部 20% 时结束 |
| scrub | `1.5` | 平滑系数 |
| 前景移动 | `-250px` | 系数 0.1，慢速 |
| 中景移动 | `-420px` | 系数 0.2，中速 |
| 背景移动 | `-550px` | 系数 0.35，快速 |
| 前景模糊 | `0 → 2px` | 焦点后移 |
| 中景模糊 | `2px → 0` | 焦点锁定 |
| 背景模糊 | `8px → 0` | 逐渐苏醒 |
| 扫描光带 | `-10% → 110%` | 横向扫描 |
| 对焦环 | `rotate: 180°` | 镜头旋转 |

## 三层结构

### 背景层 (z-index: 1)
- 抽象肌理背景
- 初始严重失焦：`blur(8px)`
- 快速移动：y: -550px
- 微距纹理叠加

### 中景层 (z-index: 2)
- 高清主图
- 初始半清晰：`blur(2px)`
- 中速移动：y: -420px
- 柔和投影

### 前景层 (z-index: 3)
- 精排文案
- 初始锐利：`blur(0px)`
- 慢速移动：y: -250px
- 文字入场动画

## 动画阶段

1. **焦点传递**：前景模糊 → 中景清晰 → 背景苏醒
2. **三层位移**：速度差 0.1 / 0.2 / 0.35
3. **空间撑开**：中段速度差拉大
4. **速度收拢**：空间恢复平稳
5. **扫描光带**：横向光带推移
6. **色温柔和**：冷暖色调过渡
7. **文案入场**：淡入 + 微位移
8. **对焦环旋转**：镜头旋转效果

## 设计灵感

如同精密相机镜头在三维空间中重新选择焦点。初始时焦点锁定在近处文案，背景严重失焦。滚动时无形对焦环开始旋转，焦点从前景逐层向深处传递，三层以非线性速度差在空间中移动。伴随焦点传递，一道极淡的横向光带自下而上扫过，强化镜头镀膜反光的精密感。
