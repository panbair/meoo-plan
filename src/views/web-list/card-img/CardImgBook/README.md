# CardImgBook - 书脊滑入·精装扉页入场

## 概述

CardImgBook 组件实现"书脊滑入·精装扉页入场"效果，模拟精装书扉页从右侧滑入的优雅动作，配合初始倾斜角度、回弹效果和装订线高光呼吸，呈现精装书籍的静谧质感。

## 效果特点

### 动画机制

- **侧翻入场**：translateX 从 120% 滑入，rotateY 从 -4deg 扶正
- **非匀速滑行**：pow 曲线实现前段从容、中段提速、末段减速
- **回弹效果**：85% 后产生 -3px 微回弹
- **阴影变化**：从侧边深长到底部均匀
- **色温偏暖**：sepia 滤镜从 0.1 渐变至 0
- **装订线高光**：落位后持续呼吸

### 视觉风格

- **色调**：深棕褐色背景配暖金色高光
- **装饰**：书架书脊装饰、书页边缘纹理
- **氛围**：精装书籍、静谧书香

## 技术实现

### ScrollTrigger 配置

| 参数 | 值 | 说明 |
|------|-----|------|
| trigger | parent | 触发器元素 |
| start | top 85% | 标准触发 |
| end | bottom 20% | 元素离开前结束 |
| scrub | 1.5 | 平滑滚动系数 |

### 核心计算

```typescript
// 非匀速滑行
const moveP = Math.pow(p, 0.7)
const x = 120 * (1 - moveP)

// 回弹效果
if (p >= 0.85) {
  const bounceP = (p - 0.85) / 0.15
  if (bounceP < 0.3) {
    bounceX = -3 * Math.sin(bounceP / 0.3 * Math.PI)
  }
}

// rotateY 前段先扶正
const rotateP = Math.min(p / 0.6, 1)
const rotateY = -4 * (1 - Math.pow(rotateP, 0.8))
```
