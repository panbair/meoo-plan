# 双翼展 · 刀锋交错入场

一组图片排列成横向阵列，初始全部隐匿。Timeline驱动图片从两端向中心依次滑入，执行极为克制的刀锋式编排：首尾两张同时启动，以极短间隔向内跟进，形成向心收敛的双翼阵列。

## 动画特性

- **双向对称入场**：首尾成对向中心滑入
- **stagger 0.08秒**：精确向心节奏
- **power3.inOut + back.out**：机械落槽质感
- **scale 0.92 → 1.0**：入场弹跳
- **阴影加深**：落地重量感
- **亮度统一**：0.9 → 1.0 完成瞬间

## 技术栈

- Vue 3 Composition API
- GSAP + ScrollTrigger
- SCSS 样式
