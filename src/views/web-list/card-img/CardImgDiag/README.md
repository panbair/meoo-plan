# CardImgDiag - 对角裁切 · 单锋斜向滑入

> 精准对角切割，刀锋般滑入的视觉冲击

## 特性

- **对角裁切**：左上至右下的精准斜切分界
- **刀锋边缘**：极细半透明描边，增强切割感
- **斜向滑出**：遮罩沿对角线方向滑出画面
- **亮度微调**：从幽暗到正常曝光的克制过渡
- **干净利落**：无粒子、无弹跳、无多余修饰

## 动画参数

| 参数 | 值 | 说明 |
|------|------|------|
| ScrollTrigger start | `top 100%` | 元素顶部到达视口时开始 |
| ScrollTrigger end | `bottom 20%` | 元素底部到达视口 20% 时结束 |
| scrub | `1.5` | 平滑系数 |
| 滑出方向 | `120%, 120%` | 沿对角线向右下滑出 |
| 滑出时长 | `1.2s` | 遮罩滑出时间 |
| 缓动 | `power3.inOut` | 干净果断 |
| 亮度恢复 | `0.3s` | 遮罩滑出后 |

## 动画阶段

1. **初始遮罩** - clip-path 覆盖下半三角 (polygon)
2. **刀锋滑出** - 遮罩沿对角线向右下滑动 120%
3. **亮度恢复** - 叠加层淡出，图片恢复正常曝光
4. **内容淡入** - 标题/副标题/描述/按钮依次显现
5. **角落装饰** - L形装饰线淡入

## 核心实现

```css
/* 对角裁切遮罩 */
.clip-path: polygon(0% 0%, 100% 100%, 100% 0%);

/* 刀锋边缘 */
&::before {
  width: 3px;
  height: 200%;
  transform: translateX(50%) rotate(45deg);
  box-shadow: 0 0 15px rgba(255, 200, 150, 0.4);
}
```

## 设计灵感

一刀两断，干净利落。对角线是最具攻击性的构图方式，它打破了画面的平衡，创造了动态的张力。当那块密实的遮罩被无形的力量挑起，沿着对角线滑出——那锋利的边缘如同刀锋划过，留下一道耀眼的光痕。图片从幽暗中苏醒，以最纯粹的方式呈现，没有任何多余修饰，只有精准与锋芒。
