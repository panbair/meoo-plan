# CardImgHydraulic - 液压推入横带渐进滑裁

## 特效概述

图片被水平切割为多条等宽横带，初始全部隐于视口上方，每条横带被轻微纵向压扁，带有极细的金属光泽描边。Timeline驱动所有横带依次向下滑入，执行非线性液压节奏——前段缓慢蓄力，中段加速下坠，末段在接近终点时渐趋粘滞减速，仿佛重型液压臂推送就位。

## 动画亮点

- 使用 `clip-path` 裁切横带，初始 `translateY` 负偏移至视口外，GSAP 驱动各带 `y` 归零
- 滑入缓动采用 `power2.in` 加速段 + `power4.out` 减速段，自定义组合模拟液压阻尼
- 初始每条横带 `scaleY: 0.95`，就位时同时恢复至 1.0，增加机械压缩的物理暗示
- 横带边缘描边使用伪元素，就位时相邻两条描边同时 `opacity` 脉冲高亮并渐隐
- 全局 `filter: brightness()` 从 0.75 缓升至 1.0

## 使用场景

产品展示、创意画廊、故事叙述等需要震撼视觉冲击力的场景。
