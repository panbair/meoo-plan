# CardImgPinhole - 针孔显影·圆形裁切封面登场

## 概述

CardImgPinhole 组件实现"针孔显影·圆形裁切封面登场"效果，模拟暗房中图片从针孔大小逐渐显影的过程，配合圆形裁切扩张、亮度联动和弹性过冲，呈现极简高级的刊物封面仪式感。

## 效果特点

### 动画机制

- **圆形裁切**：`clip-path: circle()` 从 0% 扩张至 100%
- **针孔呼吸**：初始 0.5% 微小呼吸暗示内部影像
- **柔边光晕**：裁切边缘渐变光晕同步扩张
- **亮度联动**：brightness 从 0.4 随裁切半径联动至 1.0
- **弹性过冲**：96% 后半径瞬间越过 102% 再回缩至 100%
- **铜版压痕**：完成后 inset box-shadow 持续呼吸

### 视觉风格

- **色调**：暖灰褐色封面配淡金装饰
- **材质**：布纹底板肌理
- **氛围**：暗房显影、刊物封面

## 技术实现

### ScrollTrigger 配置

| 参数 | 值 | 说明 |
|------|-----|------|
| trigger | parent | 触发器元素 |
| start | top 85% | 标准触发 |
| end | bottom 20% | 元素离开前结束 |
| scrub | 1.5 | 平滑滚动系数 |

### 核心计算

```typescript
// 圆形扩张
if (p < 0.02) {
  circleRadius = 0
} else if (p < 0.96) {
  circleRadius = 0.5 + 97.5 * Math.pow((p - 0.02) / 0.94, 0.7)
} else {
  // 弹性过冲
  const overshootP = (p - 0.96) / 0.04
  circleRadius = overshootP < 0.5
    ? 98 + 4 * Math.sin(overshootP / 0.5 * Math.PI / 2)
    : 102 - 2 * Math.pow((overshootP - 0.5) / 0.5, 2)
}

// 亮度联动
brightness = 0.4 + 0.6 * Math.pow((p - 0.03) / 0.97, 0.6)
```
