# CardImgRain · 砚池墨雨 · 多点晕染径向展开

## 效果概述

素白宣纸上，图片以隐形墨色潜伏。画面中五至七个随机位置同时落入无形墨滴，各自向外不规则晕染扩散，多重墨圈边缘彼此渗透、交融，形成浓淡相生的墨层叠加。

## 设计亮点

- 多个墨滴（7个）从不同位置同时落入
- clipPath 动态扩散，多重墨圈交融
- 墨丝沿切线方向短暂延展
- 完成后水渍环痕残留
- 东方水墨艺术感

## 技术实现

- Vue 3 Composition API + TypeScript
- GSAP + ScrollTrigger 动画
- SVG clipPath 遮罩
- SCSS 样式预处理
