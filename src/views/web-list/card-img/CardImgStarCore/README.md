# CardImgStarCore - 星核共振 · 能量脉冲辐射与呼吸

> 沉睡恒星的能量脉冲释放，宇宙深空的静谧与庄严

## 特性

- **星核光点**：画面正中悬浮钴蓝色光点
- **光丝遮罩**：放射状光丝如冻结星冕严密遮蔽
- **能量脉冲**：钴蓝光环从圆心向外辐射扩散
- **二次微脉冲**：光环过冲回弹产生二次脉冲
- **光丝飘散**：脉冲解锁后向四周优雅散开
- **画面呼吸**：1%-1.5% 缩放，5s 周期脉动
- **星光镶边**：四角独立相位微微闪烁

| 技术 | 版本 | 用途 |
|------|------|------|
| Vue 3 | ^3.3 | 核心框架 |
| TypeScript | ^5.0 | 类型系统 |

| ScrollTrigger | ^3.12 | 滚动动画插件 |
| SCSS | ^1.6 | 样式预处理 |

## 动画参数

| 参数 | 值 | 说明 |
|------|------|------|
| ScrollTrigger start | `top 100%` | 元素顶部到达视口时开始 |
| ScrollTrigger end | `bottom 20%` | 元素顶部到达视口底部 20% 时结束 |
| scrub | `1.5` | 平滑系数 |
| 光丝扩张 | `3% → 100%` | 中心半径扩张，duration: 2.2s |
| 脉冲扩张 | `scale: 0 → 1.2` | 脉冲光环辐射 |
| 过冲回弹 | `scale: 1.2 → 1.0` | back.out(1.5) |
| 呼吸缩放 | `1.0 ↔ 1.015` | 5s 周期 sine.inOut |
| 亮度 | `0.4 → 0.85` | 保持微暗星夜基调 |
| 镶边闪烁 | `0.15 ↔ 0.35` | 3.5s 周期复调 |

## 动画阶段

1. **星核闪烁** - 钴蓝光芒初现
2. **光丝扩张** - radial-gradient 从中心扩散
3. **光丝松弛** - rotate + scale 飘散
4. **脉冲辐射** - 钴蓝光环向外滑行
5. **二次微脉冲** - 过冲回弹暗示
6. **亮度揭示** - 幽暗到微亮
7. **镶边显现** - 四角星光浮现
8. **持续呼吸** - 画面 + 镶边循环

## 设计灵感

图片如一枚沉睡于暗空中的恒星遗核，画面正中悬浮着一粒极微小的钴蓝色光点，周围密布着细密的光丝——它们从圆心向外辐射，如同被冻结的星冕。Timeline 驱动核心突然释放一次无声的能量脉冲，光环所到之处光丝逐层解锁，底层图像从圆心开始渐次浮现。图像完全显现后，整幅画面进入极其缓慢的呼吸状态，如同星核仍在遥远深空中持续发送着低频信号。
