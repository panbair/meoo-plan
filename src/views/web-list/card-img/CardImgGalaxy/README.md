# CardImgGalaxy - 星轨旋臂·银河螺旋背景视差

## 简介

深邃星空背景由三层银河旋臂图像构成，滚动时各层以差异化的角速度缓慢旋转，形成漩涡般的动态视差。

## 核心特性

- **三层银河图像**：外围暗尘带、中层旋臂、核心盘面
- **螺旋旋转视差**：三层旋转速度分别为 0.15 / 0.08 / 0.03 系数
- **缩放呼吸**：各层缩放相位错开，产生跳动景深
- **引力透镜效果**：鼠标移动时星点向中心弯曲汇聚
- **粒子漂流**：100个星点沿旋臂方向漂流
- **色相漂移**：深靛与冷紫之间渐变

## 视差参数

| 层级 | 元素 | 旋转系数 | 缩放效果 |
|------|------|---------|---------|
| 远层 | 外围暗尘带 | 0.15 | 向中心收敛 |
| 中层 | 旋臂 | 0.08 | 轻微膨胀 |
| 核心 | 核心盘面 | 0.03 | 缓慢膨胀 |

## 使用方式

```vue
<template>
  <CardImgGalaxy />
</template>

<script setup lang="ts">
import CardImgGalaxy from '@/views/web-list/card-img/CardImgGalaxy/CardImgGalaxy.vue'
</script>
```

## ScrollTrigger 配置

| 参数 | 值 | 说明 |
|------|-----|------|
| trigger | container | 触发器元素 |
| start | 'top 100%' | 元素完全进入视口时开始 |
| end | 'bottom 20%' | 元素离开时结束 |
| scrub | 1.5 | 平滑系数 |
