# 幻灯轮投射 · 逐帧聚焦入场

## 概述

模拟老式幻灯机轮盘投射效果，多张图片依次进行失焦→对焦的投影展示，形成精密的轮盘转动节奏。

## 效果描述

深色展台空间中，一组图片如同老式幻灯机的轮盘幻灯片，初始全部隐匿。Timeline启动后，无形投影仪依次将图片投射到预设位置。每张图片入场并非移动或弹出，而是经历光束一闪——画面极度模糊并放大，随即急速对焦至清晰、收缩至准确尺寸，伴随遮光叶片扫过的短暂暗场间隔。上一张淡出时，下一张的光束已亮起，形成轮盘转动的精密节奏。全组投射完毕后，所有图片以极微辉光呼吸，如同幻灯片展示结束后的静态陈列。

## 技术实现

### 动画原理

1. **失焦状态**：图片 scale: 1.3, filter: blur(20px), opacity: 0
2. **对焦过程**：快速收缩至 scale: 1, blur: 0
3. **光束效果**：径向渐变光斑配合入场
4. **换片暗场**：上一张淡出时下一张启动

### 核心参数

| 参数 | 值 | 说明 |
|------|------|------|
| 图片数量 | 4 张 | 轮盘排列 |
| 初始 scale | 1.3 | 放大状态 |
| 初始 blur | 20px | 严重失焦 |
| 动画时长 | 0.5s | 对焦时间 |
| 缓动函数 | power3.in | 急速对焦 |
| stagger 间隔 | 0.5s | 换片节奏 |
| 呼吸幅度 | ±0.02 | 完成后微浮 |

### 交互设计

- **投影仪装置动画**：跟随滚动旋转
- **进度指示器**：显示当前帧号
- **悬停放大**：图片悬停时轻微放大
- **微尘粒子**：漂浮的投影仪微尘
