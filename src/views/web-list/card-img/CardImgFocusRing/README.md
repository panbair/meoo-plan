# CardImgFocusRing - 裂焦环 · 手动对焦镜头入场

> 模拟精密手动镜头的对焦过程，从极度失焦到精准合焦

## 特性

- **色散效果**：R/G/B 三通道分离形成镜头色散
- **手动对焦环**：金属质感刻度环，指针从近端向无穷远旋转
- **焦点过冲**：接近合焦时的"拉回"手感模拟
- **精密指针**：零点附近的微震锁定效果
- **刻度数字**：合焦时短暂亮起
- **参数显示**：实时显示对焦距离和模糊值

## 动画参数

| 参数 | 值 | 说明 |
|------|------|------|
| ScrollTrigger start | `top 100%` | 元素顶部到达视口时开始 |
| ScrollTrigger end | `bottom 0%` | 元素完全离开视口时结束 |
| scrub | `1.5` | 平滑系数 |
| 初始缩放 | `1.35` | 极度失焦时的放大 |
| 初始模糊 | `25px` | 极重度模糊 |
| 通道偏移 | `±7px` | RGB 通道分离距离 |
| 对焦时长 | `2.4s` | 总对焦过程 |
| 过冲时间 | `0.15s` | 焦点过冲并回稳 |

## 核心光学参数

| 参数 | 初始值 | 结束值 | 说明 |
|------|--------|--------|------|
| scale | `1.35` | `1.0` | 近摄至标准焦段 |
| blur | `25px` | `0px` | 极重度模糊至清晰 |
| RGB偏移 | `±7px` | `0px` | 色散消除 |
| 指针角度 | `-15°` | `0°` | 近端到无穷远 |

## 动画阶段

1. **初始失焦** - scale 1.35, blur 25px, RGB通道分离±7px
2. **对焦环旋转** - 指针从"近"向"∞"旋转 -15°→0°
3. **缩放回归** - 1.35→1.0，模拟焦段收缩
4. **模糊衰减** - 25px→0px，模拟手动调焦过程
5. **色散消除** - RGB通道同步归零
6. **色散加速收敛** - 接近合焦时加速消除
7. **焦点过冲** - blur短暂-1.5px (锐化光晕)，scale 1.005
8. **焦点回稳** - blur 0, scale 1.0
9. **指针微震** - 零点附近2°/-1°/0.5°/0°微震锁定
10. **合焦光晕** - 短暂金色光晕闪烁
11. **刻度亮起** - 数字短暂增亮后回稳
12. **边缘呼吸** - 金属环低频呼吸维持存在感

## 设计灵感

手动对焦镜头是摄影中最具仪式感的操作。当对焦环被缓缓旋动，画面从混沌逐渐变得清晰——这是一场摄影师与镜头之间的对话。本组件模拟这种精密的光学机械：色散的RGB通道如同镜头色差的视觉语言，指针的微震如同机械锁定的触感，而那瞬间的过冲与回稳，则是精准锁定焦点时的心跳。
