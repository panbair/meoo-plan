# CardImgRiver · 暗河浮碑

深水浮升渐显效果组件，图片如沉睡于幽暗河床的远古碑文，从深处缓缓浮升破水而出。

## 效果预览

**动画效果**: 暗河浮碑 · 深水浮升渐显

## 核心特性

### 1. 深水沉浮动画
- 图片初始沉于画面之下
- `y: 100px`、`opacity: 0`、`scale: 0.85`
- `blur(20px)` + `brightness(0.3)` 制造深水模糊感
- 从下方缓缓向上浮出

### 2. 深蓝暗影遮罩
- 深蓝渐变遮罩覆盖图片
- 模拟河床深埋感
- 随浮升进度逐渐消退

### 3. 水流光纹
- 5条白色横带
- 从左向右扫过画面
- 模拟水面折射效果
- 配合浮升中段触发

### 4. 气泡粒子系统
- 20颗随机气泡
- 从底部向上漂浮
- 水平漂移 + 垂直上浮
- 破水后消失

### 5. 水膜光泽
- 完成后边缘极淡水渍
- 低频 opacity 呼吸
- 强化出水碑文质感

### 6. 深度指示器
- 底部 DEPTH 显示
- 进度条 + 数值联动
- 从 0M 到 500M

## 动画时序

```
0.0s  - 深度指示器激活
0.2s  - 图片开始浮升
0.3s  - 深蓝遮罩消退
0.5s  - 副标题显现
0.5s  - 水流光纹开始
0.7s  - 主标题显现
0.9s  - 分隔线
1.1s  - 描述文字
1.2s  - 水膜光泽显现
```
