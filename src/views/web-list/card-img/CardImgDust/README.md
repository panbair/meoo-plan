# CardImgDust · 晨风拂尘

相纸苏醒效果组件，图片如古旧相纸静卧尘光，晨风徐来，尘埃散逸，被温柔唤醒。

## 效果预览

**动画效果**: 晨风拂尘 · 相纸苏醒

## 核心特性

### 1. 相纸浮升动画
- 初始微暗、略微缩小、覆满细尘而模糊
- `y: 50px`、`opacity: 0`、`scale: 0.88`
- `blur(12px)` + `brightness(0.7)` 制造朦胧感
- 从下方缓缓向上位移

### 2. 尘埃粒子系统
- 30颗随机尘埃粒子
- 从下方向上飘散
- 水平漂移 + 垂直上浮
- 模拟灰尘被晨风吹散

### 3. 晨风路径效果
- 8条 SVG 曲线模拟晨风
- strokeDashoffset 动画
- 从左向右拂过画面

### 4. 纸角微卷动画
- 四个角依次微卷
- `rotate` 旋转 + `scale` 缩放
- 弹性展平效果
- 增加纸质触感

### 5. 光束效果
- 温暖光束从左上角照射
- opacity 渐显
- 烘托苏醒氛围

### 6. 柔白光泽
- 完成后边缘极淡柔白光泽
- 低频 opacity 呼吸
- 强化拂尘焕新感

## 动画时序

```
0.0s  - 相纸开始浮升
0.3s  - 光束效果
0.4s  - 副标题显现
0.6s  - 纸角微卷
0.6s  - 标题显现
0.8s  - 描述文字
0.9s  - 纸角展平
1.0s  - 装饰线
1.1s  - 相纸信息
1.2s  - 柔白光泽
1.3s  - 滚动提示
```

## 技术规范

- Vue 3 Composition API + TypeScript
- GSAP + ScrollTrigger 动画
- 温暖米白渐变背景
- SVG 晨风路径
- SCSS 样式预处理
- 完整清理逻辑
