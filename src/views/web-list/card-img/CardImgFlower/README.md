# CardImgFlower · 镜花水月

倒影微动视差效果组件，图片悬浮于静谧湖面，主体与倒影微动形成克制的高级感。

## 效果预览

**动画效果**: 镜花水月 · 倒影微动视差

## 核心特性

### 1. 主体与倒影分离
- 主体向上微移 `y: -6px`
- 倒影反向微沉 `y: +4px`
- 形成缝隙呼吸感

### 2. 倒影效果
- `scaleY(-1)` 垂直翻转
- `opacity: 0.3` 半透明
- `blur(2px)` 虚化处理
- 底部水深渐变遮罩

### 3. 水纹波动
- SVG feTurbulence 滤镜
- `baseFrequency: 0.01` → `0.015`
- 滚动中轻微增强
- 涟漪扩散暗示

### 4. 微缩投影
- 倒影 `scale: 1.0` → `0.96`
- `opacity: 0.3` → `0.15`
- 悬浮高度增加后变淡

### 5. 月光效果
- 右上角月光
- 渐变光束
- 湖面氛围

### 6. 持续水纹动画
- 底部涟漪层持续波动
- `scale: 1.0` → `1.02`
- 无声液态漂浮感

## 技术规范

- Vue 3 Composition API + TypeScript
- GSAP + ScrollTrigger 动画
- SVG feTurbulence 水纹
- CSS mask 渐变遮罩
- 极微动画幅度控制
- SCSS 样式预处理
- 完整清理逻辑
