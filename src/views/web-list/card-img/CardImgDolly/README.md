# CardImgDolly · 希区柯克推轨

电影级推轨变焦视差效果，前后景分层位移与缩放，呈现高级电影感的心理张力。

## 效果预览

**动画效果**: 希区柯克推轨 · 变焦扭曲视差

## 核心特性

### 1. 双图层视差
- 前景层：速度系数 0.25（快速上移）
- 远景层：速度系数 0.06（缓慢上移）
- 形成空间深度剥离感

### 2. 推轨变焦效果
- 前景 scale: 1.0 → 1.03（缓慢放大）
- 远景 scale: 1.0 → 0.97（轻微缩小）
- 幅度严格对称互逆

### 3. Dolly Zoom 空间变形
- 位移 + 缩放联动
- 前景"突前"、远景"深陷"
- 制造电影级透视扭曲

### 4. 亮度分层
- 前景轻微提亮
- 远景微暗
- 强化空间剥离感

### 5. 极简主义
- 无任何粒子或花哨元素
- 仅凭图层变换实现
- 静默而震撼的空间变形

## 动画原理

```
滚动进度
    │
    ├─── 前景层 ──┬─── translateY: +80px
    │            └─── scale: 1.0 → 1.03
    │            └─── brightness: 1.0 → 1.1
    │
    └─── 远景层 ──┬─── translateY: +20px
                 └─── scale: 1.0 → 0.97
                 └─── brightness: 1.0 → 0.9
```

## 技术规范

- Vue 3 Composition API + TypeScript
- GSAP + ScrollTrigger 动画
- 双图层 fixed 定位
- power1.inOut 缓动
- SCSS 样式预处理
- 完整清理逻辑
