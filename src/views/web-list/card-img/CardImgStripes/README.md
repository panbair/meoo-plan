# CardImgStripes - 多轨时序 · 条带滑裁入场

图片被切割为8条竖带，以波浪式时序依次滑入，模拟精密机械模块的电磁推入效果。

## 核心特性

- **8条等宽竖带**: 精准裁切
- **波浪式入场**: 60ms 时差
- **三段式缓动**: 加速 → 巡航 → 回弹
- **金属描边**: 金色高光边缘
- **光缝效果**: 拼接瞬间闪光

## 条带结构

| 条带 | 延迟 | 金属边 |
|------|------|--------|
| 1 | 0ms | 金色渐变 |
| 2 | 60ms | 金色渐变 |
| 3 | 120ms | 金色渐变 |
| 4 | 180ms | 金色渐变 |
| 5 | 240ms | 金色渐变 |
| 6 | 300ms | 金色渐变 |

| 8 | 420ms | 金色渐变 |

## 动画说明

### 入场时序
```
t=0ms    条带1 开始滑入
t=60ms   条带2 开始滑入
t=120ms  条带3 开始滑入
...
t=420ms  条带8 开始滑入
```

### 缓动曲线
- **初段**: `power3.in` 迅猛加速
- **末段**: `back.out` 微回弹
- **总时长**: 800ms 每条

### 光缝效果
- 条带就位后触发
- 0.15s 亮起
- 0.3s 渐隐

### 亮度呼吸
- 从 0.15 → 0 淡出
- 模拟光线注入效果

### 鼠标悬停
- 条带微震动
- 边框脉冲动画
