# CardImgTitanium - 钛金闸门 · 纵向裁切商务启幕

## 效果描述

图片被两块精密钛合金闸门严密封锁，上方闸门从顶框下滑、下方闸门从底框上升，中央仅余一条水平光缝微微呼吸。Timeline 驱动液压系统启动，两块闸门同步背离中线向外滑开，闸门内边缘的铣削倒角持续泛出极细的冷色高光。

## 核心动画

- **双闸门滑开**：上方向上、下方向下同步背离中线
- **中央光缝**：初始微微呼吸，扩张时逐渐消失
- **边缘高光**：冷蓝色极细光随闸门滑开渐隐
- **亮度提升**：brightness 从 0.7 升至 1.0
- **金属框投影**：完成后 inset 投影显现

## 技术实现

- Vue 3 Composition API
- GSAP Timeline + ScrollTrigger
- CSS 渐变模拟金属质感
- SCSS 样式

## 视觉效果

- 深蓝灰商务色调
- 钛金属拉丝质感
- 冷蓝色高光线条
- 精密网格背景
