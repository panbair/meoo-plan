# CardImgFloat · 浮光掠影

悬浮相片微视差效果组件，相片安静悬浮于看不见的气垫上，精致而高级。

## 效果预览

**动画效果**: 浮光掠影 · 悬浮相片微视差

## 核心特性

### 1. 悬浮微视差
- 图片 `y: -8px` 向上微移
- 投影 `y: +4px` 反向微沉
- 形成悬浮气隙变化

### 2. 微倾效果
- 图片 `rotateX: 1.5deg` 纵向微倾
- 如同被气流抬起
- 极其克制的角度

### 3. 投影变化
- `scale: 1.0` → `0.94`
- `opacity: 0.2` → `0.08`
- 悬浮高度增加后阴影变淡

### 4. 极微动画幅度
- 所有移动控制在几像素内
- 肉眼几乎无法捕捉单帧变化
- 持续滚动累积为高级悬浮感

### 5. 精致相框
- 角落金属装饰
- 内外双层边框
- 照片纹理叠加

### 6. 环境光效
- 柔和光晕背景
- 紫色调氛围光
- 增加空间深度
