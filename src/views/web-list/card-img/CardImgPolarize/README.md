# 偏振解锁 · 光学显像渐显

图片初始被交叉偏振滤镜完全遮蔽，画面一片深黑。滚动时，一束虚拟偏振光缓慢旋转其透振方向，黑暗逐渐剥落，图像从中心开始层层透出，伴随动态移动的蓝紫虹彩色散带。

## 动画特性

- **偏振遮罩**：暗色层 + 边缘虹彩光晕
- **ScrollTrigger 驱动**：滚动映射为偏振旋转角度
- **色散效果**：30°-60° 区间出现 R/G/B 通道微移
- **光学镀膜反光**：四角极淡反光呼吸
- **无粒子**：极简高级，仅靠色彩与透明度变化

## 技术栈

- Vue 3 Composition API
- GSAP + ScrollTrigger
- SCSS 样式
