# CardImgRippleWave - 环波递现·同心涟漪入场

## 概述

图片以同心圆环带排列，初始全部隐匿于模糊光晕中。滚动触发后，涟漪从中心向外层层扩散，每环图片依次从模糊中浮起、显现并伴生微光环闪，最终全阵列清澈呈现。

## 动画效果

### 核心特性

- **同心环带分布**：9张图片按距离中心的位置分为3个环带
- **涟漪扩散动画**：由内向外依次显现，每环间隔约0.2秒
- **模糊转清晰**：每张图片从 `blur(8px)` → `blur(0px)` 过渡
- **上浮回落**：显现瞬间有6px的上浮效果
- **水波光晕**：每张图片显现时生成同心光晕闪烁
- **微呼吸**：全部显现后进入低频微呼吸状态

### 视觉效果

- 深蓝色背景配径向渐变暗角
- 中心涟漪环装饰
- 蓝色光晕边缘效果
- 优雅的标题展示

## 技术实现

### 关键技术点

1. **环带计算**：基于图片位置计算与中心的欧氏距离，分配到不同环带
2. **stagger 动画**：同环图片有微小间隔依次触发
3. **光晕效果**：使用伪元素 + scale 动画实现波光
4. **呼吸动画**：GSAP yoyo + repeat 实现无限循环

### GSAP 使用

```typescript
// 按环带依次显现
ring1Cards.forEach((card, i) => {
  tl.to(card, {
    opacity: 1,
    scale: 1,
    filter: 'blur(0px)',
    duration: 0.5,
    ease: 'power2.out'
  }, i * 0.08)
})

// 微呼吸效果
gsap.to(cards, {
  y: -2,
  duration: 1.5,
  ease: 'sine.inOut',
  yoyo: true,
  repeat: -1,
  stagger: { each: 0.1, from: 'center' }
})
```

## 参数配置

| 参数 | 值 | 说明 |
|------|-----|------|
| `start` | `top 80%` | 标准触发时机 |
| `end` | `bottom 20%` | 结束时机 |
| `once` | `true` | 仅触发一次 |

## 响应式适配

- 桌面端：3列网格，180x120px 卡片
- 平板端：2列网格，140x95px 卡片
- 移动端：自适应间距，110x75px 卡片
