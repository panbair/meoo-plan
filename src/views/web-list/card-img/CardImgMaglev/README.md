# 磁浮编队 · 悬浮卡片吸附入场

一组图片以网格排列，初始悬浮于视口上方，如同失去磁力约束的金属片在空中微微飘荡。底部虚拟磁场逐渐增强，图片自上而下逐排被吸引拉回，触地瞬间产生微弱反弹，进入悬浮锁定状态。

## 动画特性

- **悬浮初始态**：y: -40vh，随机 rotateX/Y 偏移
- **磁力吸附**：power4.in 加速 + back.out 反弹
- **逐排入场**：stagger 递减（0.1s → 0.07s → 0.04s）
- **触地波纹**：box-shadow 扩散模拟磁通量线
- **低频微浮**：±1px 正弦波浮动
- **磁力线辉光**：相邻卡片间随机闪烁

## 技术栈

- Vue 3 Composition API
- GSAP + ScrollTrigger
- SCSS 样式
