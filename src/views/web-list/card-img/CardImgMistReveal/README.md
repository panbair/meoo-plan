# 迷雾驱散 · 光透雾散渐显

图片初始被数层流动的浓雾完全笼罩，仅隐约可见极模糊的色块轮廓。随着页面滚动，一束虚拟日光从画面中心渐强穿透，雾气从中心向外围被逐层驱散。

## 动画特性

- **ScrollTrigger 驱动的雾散动画**：从中心向外围梯度消散
- **光柱穿透效果**：径向渐变遮罩动态扩张
- **雾滴粒子**：边缘随机生成，蒸发消散
- **亮度渐变**：从 0.6 升至 1.0
- **色温过渡**：冷灰到自然白
- **四角残留雾丝**：低频呼吸

## 技术栈

- Vue 3 Composition API
- GSAP + ScrollTrigger
- SCSS 样式
