# CardImgSand - 涟漪沙波 · 多层扩散

> 沙粒如涟漪般从中心层层扩散，呈现水波荡漾的视觉盛宴

## 特性

- **多层涟漪**：4 层不同质感的沙质涟漪依次扩散
- **冲击波核心**：带脉冲光环和星芒闪烁的中心能量点
- **粒子散射**：50 颗金色粒子沿切线方向飞散
- **水珠弹跳**：12 颗晶莹水珠在涟漪上弹跳
- **湿润边缘**：低频呼吸的金色边缘光晕
- **沙砾沉淀**：底部自然堆积的沙砾纹理
- **景深模糊**：涟漪层带有自然的虚实层次

## 动画参数

| 参数 | 值 | 说明 |
|------|------|------|
| ScrollTrigger start | `top 100%` | 元素顶部到达视口时开始 |
| ScrollTrigger end | `bottom 10%` | 元素底部到达视口底部 10% 时结束 |
| scrub | `1` | 平滑系数 |
| 涟漪层数 | `4` | 不同质感的多层涟漪 |
| 层延迟 | `0.15s` | 相邻层的启动间隔 |
| 扩散时长 | `1.8s` | 单层扩散时间 |
| 粒子数量 | `50` | 沿切线散射的金色粒子 |
| 水珠数量 | `12` | 涟漪上的弹跳水珠 |
| 亮度变化 | `0.25 → 1.0` | 涟漪揭示感 |
| 饱和度变化 | `0.5 → 1.0` | 色彩恢复 |

## 动画阶段

1. **核心激活** - 中心能量点绽放，带脉冲光环和星芒
2. **涟漪扩散** - 4 层涟漪依次从中心向外扩散
3. **亮度揭示** - 图片从幽暗逐渐变亮
4. **图片放大** - 微妙的景深效果 (0.95 → 1.0)
5. **粒子散射** - 50 颗粒子沿切线飞散
6. **水珠弹跳** - 12 颗水珠在涟漪上弹跳
7. **涟漪渐隐** - 各层依次淡出消失
8. **边缘光晕** - 湿润边缘金色光泽显现
9. **沙砾沉淀** - 底部沙砾纹理浮现

## 设计灵感

一滴水落入平静的沙漠，表面沙粒被水波激起层层涟漪。每层涟漪都带有独特的沙质纹理，从中心的细腻到边缘的粗粝，如同真实的水波在沙面上扩散。中心能量核心持续脉动，粒子如星火般四散，最终留下边缘那一圈湿润的光泽——这是沙漠深处的"滴水成波"仪式。
