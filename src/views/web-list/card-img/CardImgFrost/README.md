# CardImgFrost · 冰霜融显 · 窗影滚动渐现与凝结还原

## 概述

CardImgFrost 组件实现"冰霜融显·窗影滚动渐现与凝结还原"效果。图片如凝结在冬日窗玻璃后的旧影，被一层不规则霜花全覆盖，画面严重模糊。用户滚动时，霜层从画面中央向四角逐渐消融，图片同步放大并恢复清晰；反向滚动时，霜花重新从边缘向中心凝结，画面朦胧退隐。

## 动画效果

### 滚动融霜动画（核心）

| 元素 | 初始状态 | 最终状态 | scrub |
|------|----------|----------|-------|
| 主容器 | opacity: 0, scale: 0.9, y: 25 | 正常显示 | 1.5 |
| 图片 | blur: 20px, scale: 0.9 | blur: 0, scale: 1 | 1.5 |
| 霜花层 | opacity: 1 | opacity: 0 | 1.5 |
| 霜花遮罩 | opacity: 1 | opacity: 0 | 1.5 |
| 冰晶碎片 | opacity: 1 | opacity: 0 | 1.5 |
| 光晕 | scale: 0.8, opacity: 0 | 正常显示 | 1.5 |
| 脉冲冰晕 | scale: 0.6, opacity: 0 | 正常显示 | 1.5 |
| 光环 | scale: 0.7, opacity: 0 | 正常显示 | 1.5 |
| 暗角 | opacity: 0.9 | opacity: 0.45 | 1.5 |
| 标题 | opacity: 0, blur: 6px, y: 20 | 正常显示 | 1.5 |
| 边框角 | scale: 0.5, opacity: 0 | 正常显示 | 1.5 |

### ScrollTrigger 参数

| 参数 | 值 | 说明 |
|------|-----|------|
| trigger | parent | 触发器元素 |
| start | 'top 100%' | 元素顶部到达视口top 100%时开始 |
| end | 'top 20%' | 元素顶部到达视口top 20%时结束 |
| scrub | 1.5 | 平滑系数 |

### 循环呼吸动画（独立于滚动）

| 动画 | 范围 | 周期 | 效果 |
|------|------|------|------|
| 脉冲冰晕 | scale + opacity | 3.5s | 呼吸灯效 |
| 外层光环 | rotation 360° | 50s | 顺时针旋转 |
| 内层光环 | rotation -360° | 40s | 逆时针旋转 |
| 光晕呼吸 | scale + opacity | 5s | 柔和脉动 |
| 暗角呼吸 | opacity | 6s | 氛围变化 |
| 极光动画 | backgroundPosition | 20s | 极光流动 |
| 冷光闪烁 | opacity | 4s | 光影变化 |

### 粒子系统

| 粒子类型 | 数量 | 动画 | 说明 |
|----------|------|------|------|
| 冰晶粒子 | 40 | 闪烁脉动 | 漂浮冰晶 |
| 冰晶碎片 | 12 | 微光闪烁 | 霜花碎片 |
| 霜花斑块 | 8 | 脉动呼吸 | 霜层纹理 |

## 视觉层次

1. **背景层**：深蓝渐变 + 极光层 + 冰晶粒子
2. **霜层**：纹理层 + 冰晶碎片
3. **光环层**：双层旋转光环
4. **框架层**：冰蓝光晕 + 脉冲冰晕
5. **图片层**：冷光叠加 + 边缘冰霜
6. **霜花层**：8块霜花斑块
7. **暗角层**：动态暗角
8. **边框层**：四角冰蓝光框

## 技术实现

### 色彩系统

- 主背景：深蓝 `#0a1520` ~ `#102535`
- 冰蓝光晕：`rgba(100, 200, 255, 0.2)`
- 光环色：`rgba(150, 220, 255, 0.15)`
- 粒子色：`rgba(200, 240, 255, 0.9)`
- 文字色：冰白色 `#d8f0ff`

### SVG 滤镜

```svg
<filter id="frost-blur">
  <feGaussianBlur stdDeviation="12" />
</filter>
```

### 霜花纹理

使用多层渐变叠加模拟霜花纹理效果：
- 霜花遮罩层
- 8块动态霜花斑块
- 12个冰晶碎片

## 动画清理

组件卸载时自动清理所有动画：

```typescript
onUnmounted(() => {
  cleanupFns.forEach((fn) => fn.revert())
  ScrollTrigger.getAll().forEach((st) => st.kill())
})
```
