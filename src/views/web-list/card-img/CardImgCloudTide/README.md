# CardImgCloudTide - 云隙潮汐·丁达尔光呼吸

## 特效概述

全屏背景为壮阔的云层与穿透云隙的丁达尔光束。图片以极长周期执行非对称缩放与光强涨落，模拟云层缓慢翻滚、天光时明时暗的自然呼吸。

## 动画亮点

- 缩放极为克制：`scale` 在 1 至 1.03 之间以 7 秒周期往复
- 光柱层透明度与缩放错相联动，缩放峰值时光柱最盛
- 四角柔光暗角随缩放周期淡入淡出，强化空间纵深
- 全局色温以 ±5 度幅度与缩放周期错开漂移
- 所有动画无限平缓循环，不设触发

## 技术实现

| 技术 | 说明 |
|------|------|
| Vue 3 | 组件框架 |
| GSAP | 动画引擎 |
| ScrollTrigger | 滚动视差 |
| SCSS | 样式预处理 |

## 使用场景

首页横幅、落地页背景、自然景观展示等需要静谧氛围的场景。
