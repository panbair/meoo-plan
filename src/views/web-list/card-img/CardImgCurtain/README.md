# CardImgCurtain - 光幕裁切 · 商务光影启幕

## 效果描述

图片被上下两块冷白光幕严密遮蔽，光幕表面泛着微光颗粒。Timeline 驱动光幕沿垂直轴向上下同步滑开，亮度渐弱、透明度渐增。金色引导线先于遮罩滑出，强调路径。图片在光幕后清晰亮相，四周残留极淡的冷光镶边。

## 核心动画

- **双光幕滑开**：上下背离中线同步位移
- **金色引导线**：先于遮罩 0.1s 滑出
- **微粒消散**：光幕表面微光颗粒渐隐
- **亮度联动**：brightness 从 0.6 升至 1.0
- **冷光镶边**：完成后 inset 光晕呼吸

## 技术实现

- Vue 3 Composition API
- GSAP Timeline + ScrollTrigger
- CSS 渐变模拟光幕质感
- SCSS 样式

## 视觉效果

- 深蓝黑商务色调
- 冷白微光光幕
- 金色引导线高光
- 冷蓝镶边呼吸
