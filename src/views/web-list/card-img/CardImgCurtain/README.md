# CardImgCurtain 双开启幕·商务仪式感遮罩展开

## 效果概述
图片被两扇对开的遮罩严密覆盖，初始仅中央可见极细垂直线缝。Timeline 驱动遮罩如精密商务幕布向左右同步滑开，揭开过程伴随克制的缓动曲线，初段微滞蓄力，中段匀速舒展，末段轻微回弹。

## 技术实现

### 核心特性
| 特性 | 描述 |
|------|------|
| 双门遮罩 | 左右两扇 CSS 遮罩层 |
| 仪式感缓动 | power3.inOut + back.out 回弹 |
| 金属反光 | box-shadow 细光线扫过 |
| 亮度渐升 | brightness 从 0.85 升至 1.0 |
| 投影细节 | 遮罩带向外渐变投影 |
| 舞台灯光 | 开启后整体亮度提升 |

### ScrollTrigger 配置
| 参数 | 值 | 说明 |
|------|-----|------|
| trigger | container | 触发器元素 |
| start | top 100% | 元素完全进入视口时开始 |
| end | bottom -100% | 元素完全离开视口时结束 |
| scrub | 1.5 | 平滑系数 |

### 动画阶段
| 阶段 | 进度 | 效果 |
|------|------|------|
| 蓄力 | 0-10% | 遮罩微震，中央线缝显现 |
| 开启 | 10-70% | 左右遮罩同步滑开 |
| 回弹 | 70-80% | 末段轻微回弹 |
| 舞台 | 80-100% | 亮度提升，遮罩隐于边缘 |
