# CardImgCrystal - 水晶沉坠·珠宝盒轻奢入场

## 概述

CardImgCrystal 组件实现"水晶沉坠·珠宝盒轻奢入场"效果，图片如同珍贵的水晶胸针从丝绒礼盒中优雅沉降，配合双层回弹、压痕光晕和冷调透光镶边，呈现珠宝陈列的矜贵质感。

## 效果特点

### 动画机制

- **主属性驱动**：使用 `power3.inOut` + `power4.in` 混合曲线实现前段徐缓、后段加速的晶质垂坠感
- **双层回弹**：触底瞬间产生首次弹起（-6px）和二次回落，配合 scaleY 垂直压弹
- **压痕光晕**：底座伪元素圆形渐变随触底弹放，强化丝绒承接感
- **水晶微尘**：菱形粒子在触底时扬起并缓慢飘落
- **冷调透光镶边**：落地后边缘 inset box-shadow 持续呼吸

### 视觉风格

- **色调**：深蓝紫色背景配冷调银白高光
- **装饰**：四角金属边框、菱形装饰元素
- **光效**：背景径向渐变提亮、网格纹理

## 技术实现

### ScrollTrigger 配置

| 参数 | 值 | 说明 |
|------|-----|------|
| trigger | parent | 触发器元素 |
| start | top 85% | 标准触发 |
| end | bottom 20% | 元素离开前结束 |
| scrub | 1.5 | 平滑滚动系数 |
