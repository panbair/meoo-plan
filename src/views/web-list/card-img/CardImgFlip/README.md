# CardImgFlip · 扇形翻页

画廊序列翻页效果组件，图片如画廊展品依次立起，呈现波浪式翻页序列。

## 效果预览

**动画效果**: 扇形翻页 · 画廊序列展卷

## 核心特性

### 1. 3D 翻页动画
- 图片绕左边缘为轴向外翻倒
- `rotateY: 85deg` → `rotateY: 0`
- `perspective: 1500px` 构建立体空间
- `back.out(1.2)` 产生过冲回弹

### 2. 波浪式序列
- 4张图片依次翻起
- `stagger: 0.2` 间隔
- 相邻图片轻微交叠
- 上一张尚未完全立稳时下一张已开始

### 3. 高光线效果
- 翻转时侧边高光线
- `box-shadow` 侧边照射
- 随角度变化游移
- 强化立体翻页质感

### 4. 阴影系统
- 卡片左侧阴影
- 翻页时阴影位移
- 全部立起后阴影扩散

### 5. 整体前移
- 全部立起后 `translateZ: 20px`
- 展品从墙面"站出"
- `box-shadow` 同步扩散

### 6. 翻页计数器
- 金色数字显示当前翻页
- 动态更新
- 展示动画进度

```
│     ┌───┐  ┌───┐  ┌───┐  ┌───┐              │

│                                                │

│           GALLERY FLIP SEQUENCE               │

```

## 动画时序

```
0.0s  - 头部入场
0.2s  - 第1张图片翻起
0.4s  - 第2张图片翻起
0.6s  - 第3张图片翻起
0.8s  - 第4张图片翻起
1.0s  - 高光线效果
1.2s  - 阴影扩散
1.4s  - 整体前移站出
0.5s  - 底部标题入场
0.6s  - 标题显现
0.8s  - 计数器显现
```

## 技术规范

- CSS 3D Transform

- SCSS 样式预处理
- 完整清理逻辑
