# CardImgTidalDream · 潮汐入梦 · 深海水光交织动效

## 概述

CardImgTidalDream 组件实现"潮汐入梦·深海水光交织动效"效果，图片如深海宝藏在神秘海域中漂浮。通过 GSAP Timeline 驱动多层动画系统，模拟海底生物发光、水下焦散光纹、漂浮粒子和脉冲光晕效果，营造沉浸式深海梦境体验。

## 动画效果

### 入场动画

| 元素 | 动画 | 时长 | 缓动 |
|------|------|------|------|
| 外层光环 | 缩放淡入 + 旋转 | 1.2s | power2.out |
| 主图容器 | 缩放淡入（back弹跳） | 1s | back.out(1.7) |
| 脉冲光晕 | 缩放淡入 | 0.8s | power2.out |
| 标题 | 淡入上移 | 0.8s | power2.out |

### 循环呼吸动画

| 动画 | 范围 | 周期 | 效果 |
|------|------|------|------|
| 容器缩放 | 1.0 ~ 1.03 | 6s | 水压呼吸 |
| 光环缩放 | 0.95 ~ 1.05 | 6s | 反向呼吸 |
| 位移漂移 | X: ±4px, Y: ±3px | 8s | 悬浮漂移 |
| 亮度联动 | saturate + brightness | 4.8s | 水下光感 |
| 暗角涨落 | opacity 0.6 ~ 0.8 | 4.8s | 水压变化 |
| 脉冲光晕 | scale + opacity | 3s | 呼吸灯效 |

### 粒子系统

| 粒子类型 | 数量 | 动画 | 说明 |
|----------|------|------|------|
| 气泡粒子 | 30 | 上升漂浮 | 模拟深海气泡 |
| 发光生物 | 15 | 脉冲闪烁 | 生物发光效果 |
| 漂浮粒子 | 40 | 随机飘动 | 浮游生物 |
| 水波涟漪 | 5 | 扩散消失 | 水面波纹 |

### 光效层次

1. **焦散层**：模拟水下光线折射纹理
2. **光环层**：双层反向旋转光环
3. **脉冲层**：呼吸式发光光晕
4. **散射层**：水下光线散射效果
5. **边框层**：四角发光角框装饰

## 技术实现

### 动画周期表

| 动画组 | 周期 | 错相策略 |
|--------|------|----------|
| 缩放呼吸 | 6s | 基准 |
| 亮度联动 | 4.8s | 基准 × 0.8 |
| 脉冲光晕 | 3s | 独立 |
| 位移漂移 | 8s | 独立 |
| 焦散动画 | 15s | 背景层 |

### 色彩系统

- 主色调：深海蓝 `#0a2845` ~ `#0d3055`
- 发光色：青蓝色 `rgba(80, 180, 255)`
- 粒子色：冰蓝色 `rgba(100, 200, 255)`
- 暗角色：`rgba(0, 15, 40, 0.75)`
