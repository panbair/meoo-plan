# CardImgBokeh - 散景幻镜 · 光斑错位视差

> 三层视差散景效果，营造梦幻高级的镜头感

## 特性

- **三层视差**：远景光斑(0.05x) / 中景图片(0.2x) / 前景文案(0.45x)
- **散景光斑层**：25 个柔和光斑，远景漂浮呼吸
- **高清图片层**：中景推进，模糊聚焦过程
- **精致文案层**：前景锐利，高级对比
- **光斑色相流转**：冷紫至暖黄，营造光影变幻
- **粒子散射**：40 颗发光粒子伴随

## 视差参数

| 层级 | 速度系数 | 效果 |
|------|----------|------|
| 远景光斑 | `0.05` | Y轴移动 + Scale呼吸(1.0-1.04) + Opacity起伏(0.6-0.8) |
| 中景图片 | `0.2` | Y轴移动 + Opacity渐显(0.75→1) + Blur聚焦(2px→0) |
| 前景文案 | `0.45` | 快速掠过，保持锐利清晰 |

## 动画参数

| 参数 | 值 | 说明 |
|------|------|------|
| ScrollTrigger start | `top 100%` | 元素顶部到达视口时开始 |
| ScrollTrigger end | `bottom 20%` | 元素底部到达视口 20% 时结束 |
| scrub | `1.5 - 2` | 平滑系数 |
| 光斑数量 | `25` | 远景散景点 |
| 粒子数量 | `40` | 发光粒子 |
| 色相范围 | `260° - 60°` | 冷紫到暖黄 |

## 动画阶段

1. **远景光斑漂浮** - 25 个光斑缓慢向下飘移，呼吸变换
2. **光斑色相流转** - 冷紫至暖黄渐变
3. **中景图片推进** - 图片稳健推进，透明度渐显
4. **图片模糊聚焦** - blur(2px) → blur(0) 模拟对焦
5. **前景文案掠过** - 标题/副标题/描述/按钮依次显现
6. **粒子散射** - 40 颗发光粒子随滚动飞散

## 设计灵感

散景（Bokeh）是摄影中的虚化光斑效果。当镜头聚焦于主体时，前景和背景的光源会化作柔和的光斑，营造梦幻的氛围。这组件模拟这种镜头语言：远景光斑如星辰漂浮，中景图片逐渐清晰，前景文案锐利掠过——三层速度差形成纵深视差，仿佛置身于一场梦幻的光影之旅。
