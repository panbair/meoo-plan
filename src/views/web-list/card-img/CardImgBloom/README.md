# CardImgBloom 花苞绽放·植物舒张缩放

## 效果概述
图片初始如同未绽的花苞，画面以中心为原点向外折叠，呈现紧缩的放射状褶皱。多层半透明花瓣交叠覆盖，Timeline 驱动花瓣从外向内逐层旋转舒张，图像同步渐显变清晰，最终花瓣完全展开平铺为无形底衬。

## 技术实现

### 核心特性
| 特性 | 描述 |
|------|------|
| 花瓣遮罩 | 12 片旋转 clip-path 多边形 |
| 层层舒张 | 花瓣由外向内依次展开 |
| 图像渐显 | 每片花瓣内图像同步清晰 |
| 全局缩放 | scale 从 0.4 升至 1.0 |
| 花粉粒子 | 飘浮的细小花粒 |
| 随风起伏 | 鼠标驱动花瓣微动 |

### ScrollTrigger 配置
| 参数 | 值 | 说明 |
|------|-----|------|
| trigger | container | 触发器元素 |
| start | top 100% | 元素完全进入视口时开始 |
| end | bottom -100% | 元素完全离开视口时结束 |
| scrub | 1.5 | 平滑系数 |

### 动画阶段
| 阶段 | 进度 | 效果 |
|------|------|------|
| 萌芽 | 0-20% | 花苞显现，中心微光 |
| 绽放 | 20-70% | 花瓣逐层旋开，图像渐显 |
| 盛开 | 70-85% | 完全展开，花粉飘落 |
| 凋零 | 85-100% | 花瓣收拢，图像淡出 |

### 花瓣结构
```
[外层]    花瓣1-4  (最先展开)
   ↓
[中层]    花瓣5-8  (其次展开)
   ↓
[内层]    花瓣9-12 (最后展开)
   ↓
[花蕊]    中心焦点
```
