# 涟漪绽放 · 中心扩散入场

## 概述

卡片网格从画面中心点向外呈涟漪状依次扩散绽放入场，模拟水滴落入湖面激起涟漪扩散的视觉体验。

## 效果描述

一组图片排列成网格，初始所有图片汇集于画面中心，彼此重叠为一点，略微暗淡模糊。Timeline驱动后，仿佛中心奇点向外释放涟漪，图片按与中心的距离由近及远依次向外弹射，沿径向方向绽放至各自预设位置。每张图片在抵达瞬间产生极短暂的弹性微震，随即稳定。数组从中心向外层层铺开，形成精密优雅的绽放仪式，全程伴随阵列整体亮度的轻缓爬升。

## 技术实现

### 动画原理

1. **初始状态**：所有卡片 scale: 0，opacity: 0.6，定位在容器中心
2. **涟漪扩散**：按卡片距中心的欧氏距离排序，距离越近越先弹出
3. **弹性动画**：power2.out + back.out 产生微震效果
4. **亮度爬升**：全局 brightness 从 0.85 升至 1.0

### 核心参数

| 参数 | 值 | 说明 |
|------|------|------|
| 卡片数量 | 6 张 | 2×3 网格布局 |
| 初始 scale | 0 | 聚集于中心 |
| 最终 scale | 1 | 扩散到位 |
| 动画时长 | 0.55s | 单张卡片 |
| 缓动函数 | power2.out | 主扩散 |
| 回弹 | back.out(1.2) | 微震效果 |
| 亮度变化 | 0.85 → 1.0 | 全局爬升 |

### 交互设计

- **鼠标涟漪**：鼠标移动时产生涟漪扩散效果
- **悬停增强**：卡片悬停时轻微放大并显示信息
- **涟漪波纹**：每张卡片扩散时产生视觉波纹
