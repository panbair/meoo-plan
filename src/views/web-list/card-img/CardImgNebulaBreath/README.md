# CardImgNebulaBreath - 星云吐纳·恒星星尘呼吸

## 特效概述

全屏背景为深邃星云与新生恒星的广域深空摄影。图片以极缓的非对称节奏缩放——吸气阶段漫长而沉静，呼气阶段略短而轻柔。

## 动画亮点

- `scale` 在 1 至 1.025 之间以 8 秒不对称周期往复，吸气 5 秒呼气 3 秒
- 星点亮斑使用独立 GSAP 动画，每颗星以略微偏离主周期的节奏独立呼吸
- `brightness` 与 `scale` 错相 0.3 周期联动
- 星云暗带使用 `background-position` 以极微幅度水平漂移
- 四角暗角 `opacity` 在 0.3 至 0.5 间与缩放同步呼吸

## 技术实现

| 技术 | 说明 |
|------|------|
| Vue 3 | 组件框架 |
| GSAP | 动画引擎 |
| ScrollTrigger | 滚动视差 |
| SCSS | 样式预处理 |

## 使用场景

天文展示、宇宙主题首页、沉浸式背景等需要宇宙尺度静谧律动的场景。
