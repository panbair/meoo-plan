# CardImgVortex - 时空漩涡视差深渊图片组件

## 概述

这是一个震撼的时空漩涡视差深渊图片组件，利用螺旋路径与多层级深度视差效果，展现图片从黑洞漩涡中重生排列的诡异美感。

## 效果特点

### 多层视差
- **层1 (最深层)**: 背景图片，模糊收缩状态开始，逐渐清晰展开
- **层2 (中层)**: 扭曲图片，带有色调偏移，螺旋展开
- **层3 (前层)**: 发光装饰图片，最后展开并带有发光效果
- **文字层**: 随滚动位置动态浮现

### 滚动扭曲
- 向下滚动元素加速扩散，形成漩涡效果
- 向上滚动平滑回收
- 跟随滚动实时联动

### 渐入渐出
- 元素随滚动位置动态改变透明度
- 模糊度从高到低过渡
- 缩放从中心收缩到展开

### 无缝双向
- 全流程 scrub 顺滑动画
- 无卡顿、无跳帧
- 适配所有滚动速度

## 技术实现

- **GSAP ScrollTrigger**: 滚动驱动、精准定位触发区间、scrub 双向联动
- **GSAP 时间线**: 统一管理多层级动画，stagger 序列控制层次感
- **CSS 3D 透视**: perspective 配合增强真实视差立体感
- **Vue 3 + TS**: 响应式获取 DOM，生命周期安全注册/销毁动画

## 动画阶段

| 阶段 | 时间范围 | 效果 |
|------|----------|------|
| 粒子聚合 | 0 - 0.3 | 背景粒子螺旋聚集 |
| 漩涡粒子 | 0 - 0.4 | 漩涡粒子旋转扩散 |
| 层1展开 | 0.1 - 0.6 | 最深层图片展开 |
| 层2扭曲 | 0.2 - 0.7 | 中层扭曲图片展开 |
| 层3发光 | 0.3 - 0.8 | 前层发光图片展开 |
| 文字浮现 | 0.4 - 0.9 | 文字渐入显示 |

## ScrollTrigger 参数

| 参数 | 值 | 说明 |
|------|-----|------|
| trigger | vortex | 触发器元素 |
| start | 'top 100%' | 元素顶部进入视口时开始 |
| end | 'bottom -100%' | 元素底部离开视口时结束 |
| scrub | 2 | 平滑系数，值越大动画越平滑 |

## 清理机制

组件卸载时自动清理：
- 所有 GSAP Tween
- 所有 ScrollTrigger 实例
- 元素内联样式

## 响应式适配

- 桌面端 (≥1024px): 正常尺寸显示
- 平板端 (768-1024px): 适当缩小尺寸
- 移动端 (<768px): 最小尺寸，优化触摸体验
