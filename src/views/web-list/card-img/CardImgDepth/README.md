# CardImgDepth - 深度漫游·多维背景视差漫游

## 概述

这是一个多维视差效果的图片展示组件，通过前、中、远三层独立图像的差异化速度运动，创造出强烈的空间穿透感和深度漫游体验。

## 视觉特性

| 层级 | 速度系数 | 描述 |
|------|----------|------|
| **前景层** | 0.3 | 最快速度，景深虚化最强 |
| **中景层** | 0.15 | 中等速度，主体图像 |
| **远景层** | 0.05 | 最慢速度，极淡虚化 |

## 动画效果

- **垂直漂移**：三层以差异化速度纵向移动
- **缩放呼吸**：各层伴随微小的缩放变化
- **鼠标视差**：鼠标移动产生视角偏转
- **光粒子带**：细粒子向上流动
- **云影尘埃**：远景层动态装饰

## 使用方法

```vue
<template>
  <CardImgDepth />
</template>

<script setup lang="ts">
import CardImgDepth from '@/views/web-list/card-img/CardImgDepth/CardImgDepth.vue'
```
