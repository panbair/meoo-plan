# CardImgFilm - 胶片潜影 · 负片滚轴视差

模拟老式胶片放映机的负片效果，三层视差 + 齿孔 + 刮痕 + 漏光。

## 核心特性

- **三层视差**: 颗粒层 / 负片层 / 刮痕层
- **负片效果**: invert + hue-rotate 反转色调
- **齿孔动画**: 模拟走片滚动
- **胶片摆动**: 横向微弱晃动
- **漏光交互**: 鼠标触发暖色光斑

## 层级结构

| 层级 | 速度系数 | 内容 |
|------|----------|------|
| 远层 | 0.04 | 暗室颗粒 + 光晕 |
| 中层 | 0.18 | 负片主画面 + 齿孔 |
| 近层 | 0.5 | 刮痕 + 灰尘纤维 |

## 动画说明

### 滚动视差
- 三层以不同速度纵向漂移
- 近层刮痕快速飞掠
- 远层颗粒几乎静止

### 齿孔动画
- 左右两侧齿孔列
- 持续向上滚动
- 模拟放映机走片

### 胶片摆动
- 微弱的横向摆动
- 正弦波驱动
- 模拟放映机不稳

### 帧计数器
- 001 - 036 循环
- 随滚动进度更新

### 鼠标交互
- 光斑跟随光标
- 局部色调反转
- 漏光效果叠加
- 离开后缓慢恢复
