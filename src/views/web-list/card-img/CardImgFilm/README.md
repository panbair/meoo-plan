# CardImgFilm · 胶片格

## 效果概述

图片被等分为五条水平横格，如同被胶片刻刀精准分割的连续单帧。每格画面初始状态完全不可见，各自独立地处于极度模糊与幽暗之中。Timeline 驱动横格按从上至下的顺序依次亮起，每格画面经历一场极快的微型显影仪式——从失焦混沌中急速收敛至清晰。前后格之间的显影时机严格错开，形成流畅的纵向光波递进。

## 核心特性

### 五格分镜结构
| 序号 | 延迟 | 动画时长 | 效果 |
|------|------|----------|------|
| 格1 | 0s | 0.6s | 急速对焦 + 过冲锐化 |
| 格2 | 0.18s | 0.6s | 依次递进 |
| 格3 | 0.36s | 0.6s | 依次递进 |
| 格4 | 0.54s | 0.6s | 依次递进 |
| 格5 | 0.72s | 0.6s | 依次递进 |

### 显影动画细节
- **初始状态**：`blur(15px)`、`brightness(0.3)`、`opacity: 0`
- **渐显过程**：`blur(15px) → 0`、`brightness(0.3) → 1`、`opacity: 0 → 1`
- **过冲锐化**：`blur(-0.8px)` 后回升至 0，模拟对焦瞬闪
- **边缘发光**：每格点亮时的透光缝隙效果

### 视觉元素
- **分隔线**：格间极细黑框，亮起时 box-shadow 脉冲
- **齿孔装饰**：左右两侧胶片齿孔
- **细微尘噪点**：低频呼吸的噪点动画
- **暗角**：径向渐变模拟放映机效果
