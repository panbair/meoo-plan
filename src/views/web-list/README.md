## 详细描述

长页面中的图文模块在用户向下滚动、即将离开聚焦区域时，并非机械淡出，而是如同被清晨第一缕阳光照射的薄霜——从底部开始，缓慢向上升腾，边缘逐渐融解为极淡的金色微粒光晕，整体轮廓在上升中柔软消散。升腾过程中，模块并非同步模糊，而是自下而上分层消融：底部先变得稀薄并透明，中部紧随，顶部最后散逸，形成一种物质从固态直接升华为气态的优雅过程。反向滚动时，散逸的光晕重新从上方凝聚沉降，微粒归位，轮廓恢复清晰，模块完整重现。进退之间呈现自然界物质相变的静谧诗意。

## 动画亮点

- 使用 `ScrollTrigger` 并 `scrub: true` 绑定退场进度：向下滚动触发升华，向上滚动触发凝聚。
- GSAP 驱动 `y` 从 0 至 -60px、`opacity` 从 1 至 0，`duration` 映射为滚动距离，匀速丝滑。
- 模块拆分为三层纵向分区（伪元素或子容器），自下而上以 0.05s stagger 依次淡出与上浮，制造分层升华的物理感。
- 模块边缘叠加极淡的金色光晕（`box-shadow` 暖色弥散），在退场中 `opacity` 从 0 渐升至 0.2 再归零，模拟升华时的晨光折射。
- 细微光粒从模块底部随机生成，使用 `Physics2D` 赋予极轻上升速度与水平漂移，数量随退场递增后递减，模拟升华微粒。
- 反向滚动时所有变化原路逆行，光粒反向沉降消失，模块从气态重新凝聚为固态。




## 元素要完全进入屏幕后开始，在元素完全离开后结束

### ScrollTrigger 参数说明

| 参数 | 值              | 说明 |
|------|----------------|------|
| `trigger` | `parent`       | 触发器元素 |
| `start` | `'top 100%'`   | 元素顶部到达视口top 100%时开始 |
| `end` | `'bottom 20%'` | 元素顶部到达视口bottom 20%时结束 |
| `scrub` | `1.5`          | 平滑系数，值越大动画越平滑 |


### ScrollTrigger的start，end参数根据实际情况来定义。
黄金法则：
重要内容：start: 'top 75%' （提前展示）
一般内容：start: 'top 80-85%' （标准触发）
补充内容：start: 'top 90%' （接近才触发）
scrub值：1-2秒 平衡流畅度和响应性
