# 动画效果：网格碎片溶解飘散组件（滚动触发）
1、核心原理
自动将图片切割为网格碎片，滚动触发后，碎片从模糊透明状态，逐步清晰、归位，同时伴随轻微飘散动画，完成分裂入场+聚合显示。
2、动画亮点
① 溶解分裂自然不生硬：碎片初始状态为高模糊+全透明，动画过程中模糊度逐步降低、透明度逐步提升，实现“溶解”质感，而非生硬的显隐切换；② 飘散位移有层次：每个碎片的x、y偏移量和旋转角度均为随机值，模拟自然飘散效果，避免整齐划一的呆板感；③ 滚动触发适配性强：滚动至组件80%位置触发动画，滚出20%位置自动还原，适配长页面多组件布局，不影响页面整体滚动逻辑；④ 组件化可复用：样式隔离，配置项可灵活修改（切割行列、模糊度、飘散幅度），无需修改核心代码，可嵌入任意父容器；⑤ 性能优化：碎片自动生成，无需手动编写，GSAP动画采用stagger错开时序，避免卡顿，适配各类设备。
3、GSAP实现思路
① 组件初始化：先通过JS自动切割图片为网格碎片，获取图片路径、组件尺寸，计算每个碎片的宽高和定位，动态创建碎片元素并设置背景定位，确保碎片拼接后与原图一致；② 滚动触发配置：使用ScrollTrigger监听组件容器，设置start（top 80%）和end（top 20%），开启scrub实现动画与滚动进度的柔和绑定，toggleActions设为“play none none reverse”，实现滚动入场播放、滚出还原；③ 碎片动画逻辑：使用gsap.from()定义碎片初始状态（opacity:0、高模糊、随机x/y偏移、随机旋转），目标状态为正常显示（opacity:1、blur:0、x/y:0、rotate:0）；④ 时序控制：通过stagger设置碎片动画错开时间（0.03s），模拟自然飘散的先后顺序；⑤ 缓动曲线：选用expo.out缓动，让碎片归位时先快后慢，贴合溶解飘散的柔和质感，避免生硬卡顿。


### ScrollTrigger的start，end参数根据实际情况来定义。
黄金法则：
重要内容：start: 'top 75%' （提前展示）
一般内容：start: 'top 80-85%' （标准触发）
补充内容：start: 'top 90%' （接近才触发）
scrub值：1-2秒 平衡流畅度和响应性
