# 动画效果：# 沉淀离去 · 优雅沉降退场

## 详细描述

长页面中的图文模块在用户向下滚动、即将进入下一章节时，并非被简单卷走，而是如同被无形之手轻托底部，缓慢而庄重地向画面深处沉降。模块在向下沉降过程中，伴随极其微弱的尺寸收缩与透明度递减，仿佛一座珍贵的展品被温柔地降入地下宝库。与此同时，模块周围泛起一层极淡的暗角光晕，由外向内缓缓收拢，如同聚光灯在告别时轻轻调暗。整个退场过程与滚动进度严密同步，沉降速度均匀而从容，没有任何突兀或生硬，营造一种沉静而隆重的谢幕感。反向滚动时，模块从深景中优雅上升，暗角重新舒展，完整恢复原貌，进退皆由观者指尖掌控。

## 动画亮点

- 使用 `ScrollTrigger` 并 `scrub: true` 将退场进度完全绑定于滚动位置：当模块接近视口底部时触发退场，向下滚动则沉降淡出，向上滚动则上升淡入。
- GSAP 同步驱动 `y` 从 0 至 50px、`opacity` 从 1 至 0、`scale` 从 1 至 0.94，三属性匀速线性同步，呈现稳重从容的沉降节奏。
- 模块周围叠加一层极淡的暗角遮罩（伪元素径向渐变），其 `opacity` 从 0 缓升至 0.25，随沉降进度同步收拢。
- 附加极微弱的 `filter: blur()` 从 0 至 1.2px，模拟物体远离焦平面时的光学柔化，强化沉降的空间纵深感。
- 无粒子、无旋转、无弹跳，仅凭沉降位移、尺寸收缩、透明度递减与暗角收拢的四维联动，呈现克制而隆重的退场仪式。

### ScrollTrigger 参数说明

| 参数 | 值              | 说明 |
|------|----------------|------|
| `trigger` | `parent`       | 触发器元素 |
| `start` | `'top 100%'`   | 元素顶部到达视口top 100%时开始 |
| `end` | `'bottom 20%'` | 元素顶部到达视口bottom 20%时结束 |
| `scrub` | `1.5`          | 平滑系数，值越大动画越平滑 |


### ScrollTrigger的start，end参数根据实际情况来定义。
黄金法则：
重要内容：start: 'top 75%' （提前展示）
一般内容：start: 'top 80-85%' （标准触发）
补充内容：start: 'top 90%' （接近才触发）
scrub值：1-2秒 平衡流畅度和响应性
