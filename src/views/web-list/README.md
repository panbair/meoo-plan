## 详细描述

图片初始呈现为数百枚不规则碎片在深色空间中的弥散悬浮状态。每枚碎片携带慵懒的布朗运动、极低透明度与均匀的高斯模糊，彼此间距疏密无序，仿佛刚刚经历了一场无声湮灭后正在缓慢重组。碎片边缘泛着极细微的冷色光晕，暗示它们曾属于某个完整图像。

用户向下滚动时，一场精确的逆向工程被触发。碎片并非直接飞向原位，而是先经历一次快速的预聚合——所有碎片向最近的切割线靠拢，沿横纵网格排列成松散的带条，旋转与透明度在半途趋于规整。紧接着，六条横向切割带整体执行精密滑移：奇数带与偶数带沿水平方向反向错开，再以中心为轴向内同步合拢。每对带条在即将接触时加速吸附，对接瞬间迸发出极细的金属辉光。所有带条完全拼合后，画面从幽暗过渡至标准曝光，仅余网格暗影以低频呼吸，仿佛一场空间分裂与重组的精密演示刚刚落幕。

## 动画亮点

- 图片被预切割为数百枚碎片及六条横向带区。初始碎片使用 `canvas` 或大量粒子实现，`translateX/Y` 在 ±80px 至 ±150px 随机分布，`opacity` 0.15—0.4，`filter: blur(4px—8px)`，`rotateZ` 随机 ±10deg，模拟湮灭后的弥散状态。
- `ScrollTrigger` + `scrub: true` 绑定滚动进度。第一段进度驱动所有碎片向最近带区靠拢，`opacity` 升至 0.7，`blur` 降至 2px，`rotateZ` 收敛至 ±2deg 内，形成松散带条。
- 第二段进度驱动六条横带执行交错滑移：奇数带 `translateX` 从 -30px 归零，偶数带从 +30px 归零，同时 `translateY` 从拉伸状态归位，`ease: power3.inOut`。带间以 0.05s `stagger` 从外向内依次合拢。
- 对接瞬间带条边缘伪元素 `box-shadow` 脉冲辉光并迅速消失，整体 `brightness` 从 0.7 升至 1.0。最终网格暗影以 5s 周期低频呼吸。



### ScrollTrigger的start，end参数根据实际情况来定义。
黄金法则：
重要内容：start: 'top 75%' （提前展示）
一般内容：start: 'top 80-85%' （标准触发）
补充内容：start: 'top 90%' （接近才触发）
scrub值：1-2秒 平衡流畅度和响应性
