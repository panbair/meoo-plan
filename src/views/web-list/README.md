# 动画效果：记忆合金 · 热激活文字

## 详细描述
镍钛诺合金丝平铺于深灰隔热台面，每段文字被弯曲成一团杂乱的金属丝。Timeline 接通电流，丝体由电阻加热从暗银转为暗红，温度攀升至临界点后瞬间执行形状记忆跳变——所有弯曲急速归直，沿文字笔画精准重构。跳变时金属丝短暂亮至橙黄，向外辐射热浪波纹，随后冷却为哑光银白。完成后文字如同精密焊接的金属浮雕，台面投射出清晰热影。鼠标划过时局部升温再降，丝体产生微弱暂态形变。

## 动画亮点
- 初始金属丝为 SVG 路径，`d` 属性存储混乱曲线；Timeline 分段驱动 `d` 渐变为字形规范路径，模拟预热滑动，接着用 `power4.in` 瞬间完成跳变。
- 颜色同步 `fromTo`：暗银→暗红→橙黄→银白，时长与温度变化匹配。
- 热浪波纹用多层同心描边，`scale` 从 1 到 3，`opacity` 瞬增后缓降，`stagger` 分层。
- 台面投影使用 `box-shadow` 动态随丝体温度变化偏移与模糊度。
- 鼠标 hover 触发 `quickTo` 局部再次微加热，丝体颜色短暂偏暖并伴随微弱路径回弹，离开后 `InertiaPlugin` 控制冷却复位。




### ScrollTrigger 参数说明

| 参数 | 值 | 说明 |
|------|-----|------|
| `trigger` | `parent` | 触发器元素 |
| `start` | `'top 80%'` | 元素顶部到达视口80%时开始 |
| `end` | `'top 20%'` | 元素顶部到达视口20%时结束 |
| `scrub` | `1.5` | 平滑系数，值越大动画越平滑 |
