- **React 代码示例**:
  ```tsx
  // sections/CaseStudySection.tsx
  import React, { useRef, useEffect } from 'react';
  import gsap from 'gsap';
  import { ScrollTrigger } from 'gsap/ScrollTrigger';
  import CardTimeLuxeDrop from '../components/CardTimeLuxeDrop';

  gsap.registerPlugin(ScrollTrigger);

  const CaseStudySection: React.FC = () => {
    const sectionRef = useRef<HTMLDivElement>(null);

    useEffect(() => {
      const ctx = gsap.context(() => {
        // 组件内部 timeline 使用 paused: true，需要在此处通过 ScrollTrigger 触发播放
        const tl = gsap.timeline({ paused: true });
        // ... 假设我们获取到组件内部的 timeline 引用，此处仅为示例
        // 实际开发中，可能需要通过 ref 或回调方式获取组件内部的 timeline
        ScrollTrigger.create({
          trigger: sectionRef.current,
          start: 'top 80%',
          onEnter: () => tl.play(),
        });
      }, sectionRef);
      return () => ctx.revert();
    }, []);

    return (
      <section ref={sectionRef} id="cases" className="relative min-h-screen bg-gradient-to-b from-indigo-900 via-blue-800 to-slate-900 py-20 px-8">
        <CardTimeLuxeDrop />
      </section>
    );
  };

  export default CaseStudySection;
  ```
- **GSAP 动画策略**:
  - **CardTimeLuxeDrop**: 组件内部 `timeline` 使用了 `paused: true`，并设置了 `gsap.set` 初始状态。这意味着动画不会自动播放，需要外部触发。**Section层需要通过 `ScrollTrigger.create` 的 `onEnter` 回调来调用 `timeline.play()`**。这是典型的“paused: true + ScrollTrigger 联动机制”。**注意**：此处的 `ScrollTrigger.create` 是用于触发组件内部暂停的timeline，而不是创建一个空的ScrollTrigger。

```tsx
// components/Navbar.tsx
import React, { useState, useEffect, useRef } from 'react';
import gsap from 'gsap';
import { ScrollToPlugin } from 'gsap/ScrollToPlugin';

gsap.registerPlugin(ScrollToPlugin);

const Navbar: React.FC = () => {
  const [isScrolled, setIsScrolled] = useState(false);
  const [activeLink, setActiveLink] = useState('');
  const navRef = useRef<HTMLDivElement>(null);

  const navItems = [
    { label: '首页', href: '#hero' },
    { label: '产品', href: '#product' },
    { label: '案例', href: '#cases' },
    { label: '客户', href: '#clients' },
  ];

  useEffect(() => {
    const handleScroll = () => {
      setIsScrolled(window.scrollY > 50);
      // 高亮当前可见的 section
      const sections = navItems.map(item => document.querySelector(item.href));
      let current = '';
      sections.forEach((section) => {
        if (section) {
          const rect = section.getBoundingClientRect();
          if (rect.top <= 100) {
            current = section.id;
          }
        }
      });
      setActiveLink(`#${current}`);
    };

    window.addEventListener('scroll', handleScroll);
    return () => window.removeEventListener('scroll', handleScroll);
  }, []);

  const scrollToSection = (href: string) => {
    gsap.to(window, { duration: 1, scrollTo: href, ease: 'power2.inOut' });
  };

  return (
    <nav
      ref={navRef}
      className={`fixed top-0 left-0 w-full z-50 transition-all duration-300 ${
        isScrolled ? 'bg-slate-900/80 backdrop-blur-md shadow-lg' : 'bg-transparent'
      }`}
    >
      <div className="max-w-7xl mx-auto px-4 sm:px-6 lg:px-8">
        <div className="flex items-center justify-between h-16">
          <div className="text-xl font-bold text-transparent bg-clip-text bg-gradient-to-r from-blue-400 to-cyan-300">
            智域·AI
          </div>
          <div className="flex space-x-4">
            {navItems.map((item) => (
              <button
                key={item.href}
                onClick={() => scrollToSection(item.href)}
                className={`px-3 py-2 rounded-md text-sm font-medium transition-colors ${
                  activeLink === item.href
                    ? 'text-blue-400 bg-blue-900/30'
                    : 'text-blue-200 hover:text-white hover:bg-blue-800/20'
                }`}
              >
                {item.label}
              </button>
            ))}
          </div>
        </div>
      </div>
    </nav>
  );
};

export default Navbar;
```

### 5.2 各模块动画时序表
| 模块 | 触发位置 | ScrollTrigger模式 | 动画类型 | 持续时间 |
| :--- | :--- | :--- | :--- | :--- |
| **首屏Hero** | `top 100%` ~ `bottom 20%` | **scrub: 1.5** (CardImageCarbonCrystal) | 碳纤维条纹消隐 | 随滚动 |
| **首屏Hero** | 组件内部 `ScrollTrigger.create` | **无** (CardTextCorona) | 恒星粒子凝聚、日冕爆发 | 序列动画 + 持续粒子 |
| **产品详情** | `top 80%` ~ `center center` | **scrub: 1.5** (CardImageCrossCrack) | 十字切割合拢 | 随滚动 |
| **产品详情** | `top 80%` ~ `top 20%` | **scrub: 1.5** (CardTextFluid3D) | 三维涡流流体 | 随滚动 |
| **客户案例** | `top 80%` | **paused: true + onEnter** (CardTimeLuxeDrop) | 弹性坠落回弹 | 序列动画 (约2s) |
| **我的客户** | `top 85%` | **toggleActions** (标题) + **scrub: 1.5** (卡片) | 标题渐入，卡片滑入 | 标题1s，卡片随滚动 |

### 5.3 性能优化建议
- **GPU加速**: 在动画元素上使用 `will-change: transform, opacity` 或通过GSAP的 `force3D: true` 来启用GPU加速。
- **减少动画偏好适配**: 使用 `prefers-reduced-motion` 媒体查询，为偏好减少动画的用户提供静态版本或简化动画。
- **Canvas离屏渲染**: `CardTextCorona` 和 `CardTextFluid3D` 中的Canvas粒子系统，可使用离屏Canvas (`OffscreenCanvas`) 进行渲染，避免阻塞主线程。
- **IntersectionObserver懒加载**: 对于非首屏的模块，可以使用 `IntersectionObserver` 延迟初始化GSAP动画和Canvas，仅在进入视口时才创建动画实例。

### 6.2 全局样式
- **渐变背景**: 各Section使用 `bg-gradient-to-br from-slate-900 via-blue-900 to-indigo-900` 等深色渐变，营造科技感。
- **毛玻璃效果**: 导航栏和部分卡片使用 `bg-slate-900/80 backdrop-blur-md` 实现毛玻璃效果。
- **文本发光**: 标题使用 `text-transparent bg-clip-text bg-gradient-to-r from-blue-400 to-cyan-300` 实现渐变发光文字。

### 6.3 响应式断点策略
- **`sm` (640px)**: 手机端，所有模块采用单列布局，字体和间距适当缩小。
- **`md` (768px)**: 平板端，产品详情模块开始采用双列布局。
- **`lg` (1024px)**: 桌面端，展示完整的多列布局和丰富的动画效果。
- **`xl` (1280px)**: 大屏桌面端，内容区域宽度固定为 `max-w-7xl`，保持舒适阅读体验。
