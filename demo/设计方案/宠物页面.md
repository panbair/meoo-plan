# 萌宠网站设计方案

## 项目概述

**项目名称**：萌宠星球  
**文件路径**：`page-demo/demo7/pets.html`  
**核心功能**：萌宠展示、宠物服务、养宠知识、故事分享、联系领养  
**技术栈**：HTML + CSS + JavaScript + GSAP + ScrollTrigger

## 四、动画规范

### 4.1 横向滚动面板

```javascript
// 横向滚动初始化
const initHorizontalScroll = () => {
  const sections = gsap.utils.toArray('.scroll-panel');
  const track = document.querySelector('.horizontal-scroll-track');

  // 主横向滚动动画
  const horizontalAnimation = gsap.to(sections, {
    xPercent: -100 * (sections.length - 1),
    ease: 'none',
    scrollTrigger: {
      trigger: '.horizontal-scroll-section',
      pin: true,
      scrub: 1,
      anticipatePin: 1,
      snap: 1 / (sections.length - 1),
      start: 'top top',
      end: () => '+=' + (track.offsetWidth - window.innerWidth)
    }
  });
};
```

### 4.2 面板内卡片动画

```javascript
// 面板入场动画（参考面板1）
gsap.fromTo('.gallery-item',
  { y: 100, scale: 0.8, rotationY: 20 },
  {
    y: 0,
    scale: 1,
    rotationY: 0,
    stagger: 0.15,           // 卡片间隔
    ease: 'back.out(1.7)',    // 弹性缓动
    scrollTrigger: {
      trigger: section,
      start: 'left 80%',
      end: 'left 20%',
      containerAnimation: horizontalAnimation,
      scrub: 1,
      toggleActions: 'play none none reverse'
    }
  }
);

// 面板2/面板3动画参数对比
面板2 (pet-card):    { y: 100, scale: 0.8, rotationY: 20 }  →  ease: back.out(1.7)
面板3 (service-card): { x: 150, scale: 0.7 }                →  ease: power3.out
面板4 (knowledge-card): { y: -100, scale: 0.8, rotationX: 30 } →  ease: back.out(1.7)
```

### 4.3 动画参数速查表

| 动画类型 | 初始状态 | 结束状态 | 缓动函数 | 适用场景 |
|----------|----------|----------|----------|----------|
| 弹入 | `y:100, scale:0.8, rotationY:20` | `y:0, scale:1, rotationY:0` | `back.out(1.7)` | 卡片入场 |
| 滑入 | `x:150, scale:0.7` | `x:0, scale:1` | `power3.out` | 服务卡片 |
| 飞入 | `y:-100, scale:0.8, rotationX:30` | `y:0, scale:1, rotationX:0` | `back.out(1.7)` | 知识卡片 |
| 淡入 | `opacity:0` | `opacity:1` | `power2.out` | 文字/图片 |

### 4.4 Hero区域动画

```javascript
// 标题动画（模糊+位移）
gsap.fromTo('.hero-title',
  { y: 60, opacity: 0, filter: 'blur(10px)' },
  { y: 0, opacity: 1, filter: 'blur(0px)', duration: 1, ease: 'power3.out' }
);

// 副标题
gsap.fromTo('.hero-subtitle',
  { y: 50, opacity: 0 },
  { y: 0, opacity: 0.95, duration: 1, delay: 0.5, ease: 'power3.out' }
);

// 按钮（弹入）
gsap.fromTo('.hero-cta',
  { scale: 0, opacity: 0 },
  { scale: 1, opacity: 1, duration: 0.8, delay: 1, ease: 'back.out(1.7)' }
);

// 标签（依次入场）
gsap.fromTo('.hero-tag',
  { y: 30, opacity: 0 },
  { y: 0, opacity: 1, stagger: 0.1, duration: 0.6, delay: 1.3, ease: 'power3.out' }
);
```

## 七、数据统计模板

```html
<div class="stat-card">
  <div class="stat-icon">🏠</div>
  <div class="stat-number" data-target="2800">0</div>
  <div class="stat-label">成功领养</div>
  <div class="stat-desc">副说明文字</div>
</div>
```

```javascript
// 数字递增动画
const statNumbers = document.querySelectorAll('.stat-number');
statNumbers.forEach(num => {
  const target = parseInt(num.getAttribute('data-target'));
  
  gsap.to(num, {
    textContent: target,
    duration: 2,
    ease: 'power1.out',
    snap: { textContent: 1 },
    scrollTrigger: {
      trigger: '.stats-section',
      start: 'top 70%',
      end: 'top 30%',
      scrub: 1
    },
    onUpdate: function() {
      num.textContent = Math.round(this.targets()[0].textContent);
    }
  });
});
```

## 九、快速复制模板

### 9.1 新增面板流程

1. **HTML**: 在 `.horizontal-scroll-track` 内添加面板
2. **CSS**: 定义 `.panel-xxx` 背景渐变
3. **JS**: 在 `sections.forEach` 中添加动画配置

### 9.2 新增卡片流程

1. **HTML**: 按照组件标准结构编写
2. **CSS**: 应用卡片基础样式
3. **JS**: 确保选择器名称在动画代码中匹配

### 9.3 常用渐变背景

| 面板 | 背景渐变 | 卡片风格 |
|------|----------|----------|
| 萌宠相册 | `#ffecd2 → #fcb69f` | 圆形图片 |
| 宠物展示 | `#a8edea → #fed6e3` | 圆形emoji |
| 服务项目 | `#f093fb → #f5576c` | 方形图标 |
