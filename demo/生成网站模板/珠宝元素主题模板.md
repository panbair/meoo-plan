## 📋 项目要求

### 主题定位
创建一个以「珠宝元素」为主题的高端珠宝企业官网，融合奢华美学与优雅交互体验，打造尊贵、精致、永恒的珠宝品牌形象。

### 页面结构
1. **导航栏** - 固定顶部，Logo + 导航链接，毛玻璃效果，滚动样式变化
2. **首页 Hero** - 珠宝产品展示（钻戒/项链/耳环），奢华视觉冲击力，3D 悬浮效果
3. **产品系列** - 5-6 个核心系列卡片，磁力交互 + 金银光效
4. **工艺参数** - 数据统计展示，MotionPath 路径动画
5. **珠宝图库** - 6 张高清图片，MorphGrid 变形网格 + ImageComparison 对比
6. **客户体验** - 4 个体验场景，MorphingCards 变形翻转
7. **产品规格** - ModernCards 现代卡片展示
8. **客户评价** - HorizontalScroll 横向滚动
9. **联系我们** - 联系信息 + 在线表单 + MinimalistCards 极简设计

3. ✅ 使用 `toggleActions: 'play none none reverse'` 实现反向播放
4. ✅ 离开视窗时自动反向播放动画
5. ✅ 使用 GPU 加速属性（transform, will-change, backface-visibility）
6. ✅ 使用 3D transform 增强深度感
7. ✅ 磁力交互跟随鼠标移动（优雅流畅）
8. ✅ 珠宝光效营造奢华感（金色/钻石闪烁）

### 注意避免内容不可见
- 使用了 gsap.set 设置初始状态，
- 然后使用 gsap.to 进行动画，
- 元素在动画触发前会先显示为初始状态（虽然 opacity: 0，但位置已经在正确的位置），
- 当滚动到触发区域时，动画会平滑地执行，内容就会显示出来。

**标准 ScrollTrigger 配置（滚动控制）：**
```javascript
scrollTrigger: {
  trigger: element,
  start: 'top 85%',
  end: 'top 25%',
  scrub: 1,           // 滚动控制进度
  toggleActions: 'play none none reverse'
}
```

```javascript
// 必须引入 ScrollToPlugin
gsap.registerPlugin(ScrollTrigger, ScrollToPlugin, MotionPathPlugin);

// 平滑滚动导航
document.querySelectorAll('.nav-link, .hero-button').forEach(link => {
  link.addEventListener('click', (e) => {
    e.preventDefault();
    const targetId = link.getAttribute('href');
    const target = document.querySelector(targetId);
    if (target) {
      const offset = 80; // 导航栏高度
      const targetPosition = target.getBoundingClientRect().top + window.pageYOffset - offset;
      gsap.to(window, {
        duration: 1,
        scrollTo: targetPosition,
        ease: 'power3.inOut'
      });
    }
  });
});
```

```javascript
gsap.to('.jewelry-3d', {

  const jewelry = document.querySelector('.jewelry-3d');
  const rect = jewelry.getBoundingClientRect();
  const x = ((event.clientX - rect.left) / rect.width - 0.5) * 15;
```

### 4. 磁力卡片交互
- 鼠标靠近时卡片跟随移动（x/y 偏移，最大 20px）
- 3D 旋转效果（rotateX/rotateY 根据鼠标位置，最大 10deg）
- 金色光效增强（gold-aura 元素，opacity + scale 变化）
- 悬停时放大（scale: 1.03）+ 发光（box-shadow 金色光晕）
- 磁力距离检测（鼠标进入卡片范围 80px 内触发）
- 平滑过渡（duration: 0.4s，ease: power2.out）
- 离开时复位动画

**磁力交互代码：**
```javascript
// 磁力效果
const handleMagneticMove = (event, item) => {
  const rect = item.getBoundingClientRect();
  const centerX = rect.left + rect.width / 2;
  const centerY = rect.top + rect.height / 2;
  const moveX = (event.clientX - centerX) / rect.width * 20;
  const moveY = (event.clientY - centerY) / rect.height * 20;

  gsap.to(item, {
    x: moveX,
    y: moveY,
    rotateX: moveY * -0.3,
    rotateY: moveX * 0.3,
    scale: 1.03,
    duration: 0.4,
    ease: 'power2.out'
  });

  // 金色光效
  gsap.to(item.querySelector('.gold-glow'), {
    opacity: 0.5,
    scale: 1.2,
    duration: 0.4
  });
};
```

### 5. 金色粒子背景系统
- 30-40 个金色调粒子（根据屏幕大小调整数量）
- 持续向上漂浮动画（y 方向运动，x 方向随机偏移）
- 随机大小（1-4px）、速度（4-6s）、延迟（0-2s）
- 固定定位，不影响滚动（position: fixed）
- 部分粒子钻石闪烁效果（30% 概率，box-shadow: 0 0 15px）
- 粒子透明度渐变（opacity 从 0.7 到 0.1）
- 使用 requestAnimationFrame 优化性能
- 粒子超出屏幕后重置位置（循环复用）

**粒子系统代码：**
```javascript
// 创建金色粒子
function createParticles() {
  const container = document.getElementById('particles');
  const particleCount = 35;

  for (let i = 0; i < particleCount; i++) {
    const particle = document.createElement('div');
    particle.className = 'particle';
    particle.style.left = Math.random() * 100 + '%';
    particle.style.top = Math.random() * 100 + '%';
    particle.style.width = Math.random() * 3 + 1 + 'px';
    particle.style.height = particle.style.width;
    particle.style.opacity = Math.random() * 0.5 + 0.2;

    // 30% 粒子钻石闪烁
    if (Math.random() > 0.7) {
      particle.style.boxShadow = '0 0 15px #D4AF37, 0 0 30px #F5DEB3';
    }

    container.appendChild(particle);

    gsap.to(particle, {
      y: '-=80',
      x: `+=${Math.random() * 80 - 40}`,
      opacity: 0.1,
      duration: Math.random() * 2 + 4,
      repeat: -1,
      delay: Math.random() * 2,
      ease: 'none'
    });
  }
}
```

**避免常见错误：**
1. ❌ 不要过度使用动画，影响性能（控制同时运行的动画数量）
2. ❌ 不要让动画遮挡内容（动画元素设置 z-index: -1 或 pointer-events: none）
3. ❌ 不要忘记响应式适配（使用 media queries 和 clamp()）
4. ❌ 不要使用过大的图片影响加载（压缩图片，使用 WebP 格式）
5. ❌ 不要忽略移动端体验（触摸事件、手势交互、字体大小）
6. ❌ 不要过度使用金色光效（控制发光范围和强度）
7. ❌ 不要让 3D 效果过于剧烈（限制旋转角度和位移距离）
8. ❌ 不要忽略无障碍访问（aria-label、alt 文本、键盘导航）
9. ❌ 不要使用固定的硬编码数值（使用 CSS 变量和相对单位）
10. ❌ 不要忘记清理动画实例（页面卸载时 kill 动画）

**动画节奏控制：**
- Hero 区域：慢速优雅（2.5-3.5 秒）
- 系列卡片：中速交互（0.6-1.2 秒）
- 数据展示：中等节奏（1.5-2.5 秒）
- 滚动动画：跟随滚动（scrub: 1）
- 微交互：快速响应（0.3-0.5 秒）
