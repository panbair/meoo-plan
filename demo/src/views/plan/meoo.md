# 动漫视界 - 详细开发方案文档

## 项目概述

**动漫视界**是一个专业的动漫文化企业展示平台，致力于为动漫爱好者打造一个充满创意与激情的二次元世界。平台整合了作品展示、IP孵化、资讯中心、周边商城、二次元社区和关于我们等六大模块，构建一个全方位的动漫文化生态系统。

**设计理念**：融合东方美学与现代科技，用动态视觉效果和沉浸式体验，让每一位访客都能感受到动漫的魅力与震撼。

## 动画效果汇总

### GSAP 动画

**ScrollTrigger 滚动触发动画：**
- 作品卡片3D入场
- 角色卡片翻开入场
- 资讯卡片分镜入场
- 商品卡片礼盒入场
- 团队成员旋转入场
- 联系信息滑入

**Timeline 时间线序列动画：**
- Hero区域元素依次入场
- 背景→粒子→标题→副标题→按钮→统计

**缓动函数：**
- `power2.out` - 平滑减速
- `power3.out` - 强减速
- `elastic.out(1, 0.5)` - 弹性效果
- `back.out(1.7)` - 回弹效果

### Anime.js 动画

**故障文字效果：**
- translateX 抖动
- skewX 倾斜
- opacity 闪烁

**发光扩散效果：**
- scale 从0到1.5
- opacity 从0.5到0

**弹性缩放：**
- scale 1 → 0.9 → 1.1 → 1

### CSS 动画

**粒子连线动画：**
- Canvas绘制粒子
- 粒子间距离小于100px时连线
- 粒子随机移动

**流光扫过效果（shimmer）：**
```css
@keyframes shimmer {
  0% { transform: translateX(-100%); }
  100% { transform: translateX(100%); }
}
```

**霓虹边框呼吸发光（border-glow）：**
```css
@keyframes border-glow {
  0%, 100% { box-shadow: 0 0 5px rgba(255, 107, 157, 0.5); }
  50% { box-shadow: 0 0 20px rgba(255, 107, 157, 0.8); }
}
```

**滚动指示器弹跳：**
- animate-bounce
- 内部圆点脉冲 animate-pulse

## 响应式设计

| 设备 | 断点 | 布局 |
|:-----|:----:|:-----|
| 手机 | < 640px | 单列 |
| 平板 | 640-1024px | 双列 |
| 桌面 | 1024-1280px | 多列网格 |
| 大屏 | > 1280px | 宽屏布局 |

**响应式规则：**
- 导航栏：桌面横向排列，移动端汉堡菜单

## 注意事项

1. **GSAP 插件注册**：在组件内使用 `gsap.registerPlugin()` 注册 ScrollTrigger 和 ScrollToPlugin
2. **ScrollTrigger 清理**：在 useEffect cleanup 中调用 `ctx.revert()` 清理动画
3. **图片优化**：使用 Unsplash 图片，添加合适的尺寸参数
4. **动画性能**：使用 `will-change` 和 `transform-gpu` 启用硬件加速
5. **移动端优化**：减少粒子数量，简化复杂动画
6. **表单验证**：客户端验证姓名、邮箱格式，主题必选
