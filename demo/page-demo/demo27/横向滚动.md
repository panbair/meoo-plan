# NexusTech 企业官网横向滚动实现方案

## 一、核心技术概述

本页面使用 **GSAP ScrollTrigger** 实现横向滚动效果，是企业官网的经典布局方案。

### 核心依赖
- GSAP 3.x (GreenSock Animation Platform)
- ScrollTrigger 插件
- Font Awesome 6.x 图标库
- Google Fonts (Inter)

## 四、JS 核心实现

### 1. GSAP 初始化
```javascript
gsap.registerPlugin(ScrollTrigger, ScrollToPlugin);
```

### 2. 横向滚动主动画
```javascript
function initAnimations() {
  const wrapper = document.getElementById('horizontalWrapper');
  const container = document.getElementById('horizontalContainer');
  const sections = gsap.utils.toArray('.panel-27');
  const panelCount = sections.length;
  const viewportWidth = window.innerWidth;
  
  // 计算所有面板的总宽度
  const totalWidth = sections.reduce((sum, panel) => sum + panel.offsetWidth, 0);
  const scrollDistance = totalWidth - viewportWidth;
  
  // 横向滚动主动画
  const mainScroll = gsap.to(container, {
    x: () => -scrollDistance,  // 滚动距离 = 总宽度 - 视口宽度
    ease: 'none',               // 线性滚动
    scrollTrigger: {
      trigger: wrapper,
      pin: true,                // 固定 wrapper
      scrub: 0.8,               // 平滑滚动绑定
      snap: {                   // 吸附到面板
        snapTo: 1 / (panelCount - 1),
        duration: { min: 0.2, max: 0.4 },
        ease: 'power1.inOut'
      },
      end: () => '+=' + scrollDistance,
      invalidateOnRefresh: true
    }
  });
}
```

### 3. 面板入场动画
```javascript
function animatePanel(trigger, ...elements) {
  const horizontalST = ScrollTrigger.getAll().find(st => st.vars?.pin === true && st.animation);
  const containerAnim = horizontalST ? horizontalST.animation : null;
  
  elements.forEach((el) => {
    gsap.utils.toArray(el).forEach((item, i) => {
      // 1. 设置初始状态
      gsap.set(item, { y: 80, opacity: 0, scale: 0.9, visibility: 'hidden' });
      
      // 2. 设置入场动画
      gsap.to(item, {
        y: 0,
        opacity: 1,
        scale: 1,
        visibility: 'visible',
        duration: 0.8,
        delay: i * 0.1,
        ease: 'power3.out',
        scrollTrigger: {
          trigger: trigger,
          containerAnimation: containerAnim,  // 关键：关联横向滚动
          start: 'left 85%',                  // 面板左侧到达85%视口时触发
          end: 'left 60%',
          toggleActions: 'play none none reverse'
        }
      });
    });
  });
}

// 调用
animatePanel('#about', '.about-text-27', '.about-feature-27', '.about-card-27');
animatePanel('#services', '.services-header-27', '.service-card-27');
// ...
```

### 4. 数字递增动画
```javascript
function animateNumbers() {
  const horizontalST = ScrollTrigger.getAll().find(st => st.vars?.pin === true && st.animation);
  const containerAnim = horizontalST ? horizontalST.animation : null;
  
  document.querySelectorAll('[data-target]').forEach(num => {
    const target = parseInt(num.dataset.target);
    
    gsap.to(num, {
      textContent: target,
      duration: 2,
      ease: 'power2.out',
      scrollTrigger: {
        trigger: num,
        containerAnimation: containerAnim,
        start: 'left 80%',
        toggleActions: 'play none none none'
      },
      snap: { textContent: 1 },
      onUpdate: function() {
        num.textContent = Math.round(this.targets()[0].textContent).toLocaleString();
      }
    });
  });
}
```

### 5. 导航跳转
```javascript
const sectionIds = ['hero', 'about', 'services', 'products', 'team', 'stats', 'contact'];

document.querySelectorAll('.nav-dot-27').forEach((dot, i) => {
  dot.addEventListener('click', () => {
    const horizontalST = ScrollTrigger.getAll().find(st => st.vars?.pin === true);
    if (horizontalST) {
      const progress = i / (sectionIds.length - 1);
      const targetScroll = horizontalST.start + (horizontalST.end - horizontalST.start) * progress;
      
      gsap.to(window, {
        duration: 1.2,
        scrollTo: { y: targetScroll, autoKill: false },
        ease: 'power3.inOut'
      });
    }
  });
});
```

### 6. 导航点高亮
```javascript
const navDots = gsap.utils.toArray('.nav-dot-27');
const sectionCount = sectionIds.length;

ScrollTrigger.create({
  trigger: '#horizontalWrapper',
  end: () => '+=' + (document.getElementById('horizontalContainer').offsetWidth - window.innerWidth),
  onUpdate: (self) => {
    const progress = self.progress;
    const currentIndex = Math.round(progress * (sectionCount - 1));
    
    navDots.forEach((dot, i) => {
      dot.classList.toggle('active', i === currentIndex);
    });
  }
});
```

## 五、关键参数说明

### ScrollTrigger 配置
| 参数 | 值 | 说明 |
|------|-----|------|
| `pin` | `true` | 固定容器，页面不随滚动滚动 |
| `scrub` | `0.8` | 滚动与动画的平滑绑定，越大越跟手 |
| `snap` | 对象 | 吸附到固定位置 |
| `invalidateOnRefresh` | `true` | 窗口大小变化时重新计算 |

### containerAnimation
这是横向滚动的**核心参数**。所有子元素的 ScrollTrigger 都需要设置 `containerAnimation` 指向主横向动画，这样子元素的位置判断才基于横向容器而不是页面。

### 动画时序参数
| 参数 | 值 | 说明 |
|------|-----|------|
| `start: 'left 85%'` | 面板左侧到达视口85%位置时触发 | 预判入场 |
| `end: 'left 60%'` | 面板左侧到达视口60%位置时动画完成 | 动画范围 |
| `scrub: 3` | 背景光效延迟3秒跟随滚动 | 缓动效果 |

## 六、初始化时序

```
1. 页面加载 → CSS 隐藏所有元素 (opacity:0)
         ↓
2. DOMContentLoaded → 注册 GSAP 插件
         ↓
3. 加载动画结束 (2秒) → 显示横向容器
         ↓
4. initAnimations() 执行 → 创建 ScrollTrigger
         ↓
5. 0.3秒后 → Hero 区域入场动画
         ↓
6. 滚动开始 → 横向滚动 + 面板入场动画
```

## 八、常见问题与解决

### 1. 滚动不了
**原因**: `.horizontal-container-27` 缺少 `width: fit-content`

**解决**: 确保容器宽度等于所有面板宽度之和

### 2. 首屏闪烁
**原因**: 元素在动画前可见

**解决**: CSS 设置初始 `opacity: 0; visibility: hidden`，JS 中用 `gsap.set()` 设置初始状态

### 3. 子元素动画不触发
**原因**: 缺少 `containerAnimation` 配置

**解决**: 获取主横向 ScrollTrigger 的 animation，传递给子元素

### 4. 导航跳转位置不准
**原因**: 直接用总宽度计算

**解决**: 基于 ScrollTrigger 的 start/end 属性计算目标位置
